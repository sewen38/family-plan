# 架构图有效性验收清单

每张核心架构图必须回答：

- [ ] 解决什么税务/资金/身份问题？
- [ ] 主体有哪些：中国经营主体、境外公司、个人、信托/保险/投资账户？
- [ ] 资金如何流动？
- [ ] 利润如何归集？
- [ ] 税务居民如何判断？
- [ ] DTA / 预提税如何处理？
- [ ] CFC / CRS / FATCA / FBAR / PFIC 是否适用？
- [ ] 37号文 / ODI / FDI / 外汇是否触发？
- [ ] 哪些动作合法降低税负或隔离风险？
- [ ] 哪些动作禁止？
- [ ] 手机端是否可读？
- [ ] 是否无黑块遮字、无出框、无低对比？
