# 新加坡、日本、泰国、马来西亚 最新移民政策全景（截至2026年6月）

> **数据来源说明**：以各国政府官网（ICA、MOM、EDB、入管局、BOI、MOHA等）及权威律所/机构（KPMG、Baker McKenzie、PwC等）为准，标注来源年份。

## 一、新加坡（Singapore）

### 1.1 投资移民：GIP全球投资者计划（Global Investor Programme）

> 来源：EDB官网 GIP Factsheet（2023修订版，现行）、ASEAN Briefing（2024）

| 维度 | 详情 |
|---|---|
| **管理机构** | 新加坡经济发展局（EDB） |
| **申请人资格** | ①**资深企业主**：≥3年创业/经商经验，持股≥30%，公司近1年营业额≥2亿新元且近3年均≥2亿新元；②**下一代企业主**：直系亲属持股≥30%，本人为管理层（C-suite/董事会），公司营业额≥5亿新元；③**快速增长企业创始人**：估值≥5亿美元的科技公司创始人/大股东；④**家族理财办公室负责人**：≥5年投资管理经验，管理资产≥2亿新元 |
| **方案A（投资企业）** | 投资**1,000万新元**（约740万美元）到新加坡新设/现有企业 |
| **方案B（投资GIP基金）** | 投资**2,500万新元**（约1,850万美元）到GIP审批基金 |
| **方案C（设立家族理财办公室）** | 在管资产≥**2亿新元**的单一家族办公室，其中≥**5,000万新元**部署到新加坡 |
| **获得身份** | 获批后即获**PR（永久居民）**身份，初始有效期5年（需REP） |
| **转PR条件** | 获批即为PR；5年后续签REP需证明投资持续有效 |
| **入籍条件** | PR满2年后可申请公民，实际审批考虑经济贡献、融入度等 |
| **福利** | PR享有购房（可购组屋）、教育补贴、医疗补贴（Medisave/MediShield Life）、CPF公积金 |

### 1.2 技术移民：EP就业准证（Employment Pass）

> 来源：MOM官网（2025年1月更新）、Piloto Asia（2026）

| 维度 | 详情 |
|---|---|
| **薪资门槛（2025.1.1生效）** | 非金融业：**S$5,600/月**；金融业：**S$6,200/月**；年龄越大要求越高，45岁以上非金融业最高**S$10,700/月**、金融业**S$11,800/月** |
| **COMPASS打分体系** | 2023.9.1起实行，需获≥**40分**。6个评估维度（各0-20分）：①薪资（C1）②学历（C2）③员工多样性（C3）④本地就业支持（C4）⑤技能加分（C5）⑥战略经济优先加分（C6） |
| **签证有效期** | 首次最多**2年**，续签后最多**3年** |
| **续签条件** | 2026.1.1起续签适用新薪资门槛；仍需通过COMPASS评分 |
| **转PR路径** | EP持有6个月后可申请PR，通过ICA审批，无固定年限要求（来源：ICA官网） |
| **入籍条件** | PR满**2年**、21岁以上可申请公民（来源：ICA官网） |
| **福利** | EP本身仅为工作准证，无PR福利；获批PR后方享住房、教育、医疗、CPF等 |
| **附加：5年经验科技EP** | 2025年新设，薪资≥S$10,700（36岁以下）至S$14,270（45岁以上），有效期5年（来源：MOM） |

### 1.3 科技准证：Tech.Pass

> 来源：MOM官网、EDB（2021年起推行）

| 维度 | 详情 |
|---|---|
| **申请条件** | 满足以下**任意2项**：①过去1年月薪≥**S$20,000**（或等值外币）②在估值/市值≥5亿美元或融资≥3,000万美元的科技企业有≥**5年**领导经验③有≥**5年**领先科技产品开发经验且产品月活≥10万或年收入≥1亿美元 |

| **特殊权利** | 可同时为多家公司工作、创业、担任顾问、投资、任教 |
| **转PR路径** | 2年后可申请PR |
| **来源年份** | 2021年推出，2025年持续有效 |

### 1.4 家族办公室：13O/13U税务豁免

> 来源：MAS/PwC税务公告（2024.10）、Baker McKenzie（2025.1更新）

| 维度 | 13O（在岸基金） | 13U（增强层基金） |
|---|---|---|
| **最低AUM** | 申请时S$2,000万（2年宽限期内达标） | 申请时S$5,000万 |
| **投资专业人士(IP)** | ≥**3名**IP（其中至少1名非家族成员） | ≥**3名**IP（其中至少1名非家族成员） |
| **本地商业支出(LBS)** | 分级制：AUM<S$5,000万→S$20万；S$5,000万-S$1亿→S$50万；≥S$1亿→S$100万 | 同上分级LBS |
| **资本部署要求** | ≥10% AUM或S$1,000万（取低值）部署到新加坡 | 同左 |
| **税务豁免** | 指定投资收入免所得税 | 指定投资收入免所得税 |
| **EP获取** | 家族办公室可为自己和投资团队申请EP（通过MAS审批框架） | 同左 |
| **生效时间** | 2025.1.1起新修订生效，旧有获准者宽限至2028财年 | 同左 |

### 1.5 新加坡短期签证+移民路径

| 路径 | 详情 |
|---|---|
| **短期签证** | 免签/电子签（一般为30-90天，不可工作） |
| **EP→PR→公民** | 最常见普适路径（见1.2） |
| **GIP→PR→公民** | 高净值投资者捷径（见1.1） |
| **家属路径** | EP持有者配偶/子女可申请DP（家属准证）/LTVP；配偶结婚满2年可通过配偶担保申请PR |
| **留学生路径** | 在新通过PSLE/O/A水准考试的外国学生可申请PR |

## 二、日本（Japan）

### 2.1 投资移民：经营管理签证

> 来源：日本入管局（2025年10月16日生效修订）、KPMG Flash Alert（2025-195）、Baker McKenzie（2026.2）

| 维度 | 旧制度（2025.10.15前） | 新制度（2025.10.16起） |
|---|---|---|
| **最低资本金** | **500万日元** | **3,000万日元**（约20万美元） |
| **雇佣要求** | 至少2名全职员工**或**满足资本金 | 必须雇佣≥**1名全职员工**（日本人/永住者/定住者等） |
| **日语能力（新增）** | 无要求 | 申请人**或**一名全职员工需达**CEFR B2/日语N2**或同等级别 |
| **管理经验/学历（新增）** | 无要求 | **硕士及以上**（管理/技术/商业相关领域）**或**≥**3年经营管理经验** |
| **办公场所（收紧）** | 家庭办公室可接受 | **独立物理办公空间**，家庭办公室原则上不可用 |
| **事业计划书（新增）** | 无要求 | 需由**持牌专家**（中小企业诊断士、公认会计师等）认证 |
| **签证有效期** | 通常1年、3年、5年 | 同左 |
| **续签条件** | 企业正常运营、纳税 | 3年过渡期内（至2028.10.16）综合评估；之后必须完全符合新规，严格审查税务/社保缴纳 |
| **转永居** | 通常居住**10年**（HSP加速见2.2） | 不变 |
| **入籍** | 居住**5年**（拟改至10年） | 拟从5年延长至**10年**（2026年4月起或之后实施，来源：Asahi/Japan Today 2025.12） |

### 2.2 技术移民：高度人才签证（Highly Skilled Professional）

> 来源：日本入管局、visajapan.jp（2026版）

| 维度 | 详情 |
|---|---|
| **类型** | 三类：①学术研究(i)(a) ②专业技术(i)(b) ③经营管理(i)(c) |
| **打分体系** | 按学历（博士30分/硕士20分/学士10分）、工作经验、年收入（400万日元起/最高40分）、年龄、日语能力、研究成果等综合打分 |
| **70分路径** | 获70分→获5年签证→3年后可申请**永居** |
| **80分路径** | 获80分→获5年签证→**1年后**即可申请**永居** |
| **特权** | 可从事多种在留活动（如研究+经营）、配偶工作无限制、可带父母（满足条件）、家政人员许可 |
| **签证有效期** | **5年** |
| **入籍条件** | 永居后再居住一段时间可申请归化；现行全流程5年（HSP加速），拟改10年 |
| **来源** | 入管局2017年修订，2025年持续执行 |

### 2.3 雇主移民：技术·人文知识·国际业务签证

> 来源：入管局

| 维度 | 详情 |
|---|---|
| **申请条件** | 日本企业雇佣，从事技术/人文知识/国际业务工作；需有大学学历或相关工作经验 |
| **签证有效期** | 1年、3年、5年 |
| **续签** | 雇佣持续且合规 |
| **转永居** | 居住**10年**（连续）且持有3年或5年签证 |
| **入籍** | 居住**5年**（拟改10年），需放弃原国籍（日本不承认双重国籍） |
| **福利** | 加入日本社保（健康保险、厚生年金）、子女可入读公立学校 |

### 2.4 日本短期签证+移民路径

| 路径 | 详情 |
|---|---|
| **短期签证** | 中国公民需签证，旅游签15-90天 |
| **留学→技术签证→永居** | 最普适路径 |
| **经营管理签证→永居/归化** | 自营经商路径 |
| **高度人才→永居→归化** | 最快1年永居 |

## 三、泰国（Thailand）

### 3.1 投资签证：泰国精英签证（Thailand Privilege Visa，原Elite Visa）

> 来源：Thailand Privilege Card Co., Ltd. 官方（2026）、thailandelite.net（2026）

| 层级 | 有效期 | 费用（泰铢） | 约合美元 |
|---|---|---|---|
| **青铜（Bronze）** | **5年** | ฿650,000 | ≈$19,000 |
| **黄金（Gold）** | **5年** | ฿900,000 | ≈$26,500 |
| **铂金（Platinum）** | **10年** | ฿1,500,000 | ≈$44,000 |
| **钻石（Diamond）** | **15年** | ฿2,500,000 | ≈$73,500 |
| **黑卡（Reserve）** | **20年** | ฿5,000,000 | ≈$147,000（仅邀请） |

| 维度 | 详情 |
|---|---|
| **实质** | 长期**旅游签证**，非移民签证 |
| **单次停留** | 每次入境**1年**（需每年离境一次或每年报到） |
| **特权** | VIP机场快速通道、专属助理、90天报到简化（延至1年）、24/7客服、年度积分（可用于SPA/接送/体检等） |
| **转永居条件** | **不可直接转PR**——精英签证年份不计入PR资格年限（来源：Forbes & Partners, 2025） |
| **入籍条件** | 不可通过精英签证入籍 |
| **福利** | 长期居住权，不可工作、不可享受公共医疗/教育 |
| **工作权利** | 不允许；如需工作需单独申请工作许可 |
| **来源** | 2023年品牌改名Thailand Privilege Visa |

### 3.2 长期居留签证：LTR（Long-Term Resident Visa）

> 来源：泰国BOI官网 ltr.boi.go.th（2025-2026）、泰国驻洛杉矶领事馆

| 类别 | 条件 |
|---|---|
| **富裕全球公民** | 全球资产≥**100万美元**+泰国投资≥**50万美元**（国债/房产/企业）+个人年收入≥**$80,000** |
| **富裕退休人士** | 年龄≥**50岁**+被动收入≥**$80,000/年**（如低于但≥$40,000则需额外投资$250,000） |
| **泰国远程工作者** | 年收入≥**$80,000**+受雇于海外上市公司或年营收≥$5,000万的私企（≥3年历史）+硕士或知识产权/5年经验 |
| **高技能专业人士** | 年收入≥**$80,000**+在泰国目标产业就业（低于但≥$40,000需硕士）+雇主为泰国企业/大学/政府 |

| 维度 | 详情 |
|---|---|
| **签证有效期** | 首次**5年**+可续**5年**（共**10年**） |
| **单次停留** | 无出境要求，90天报到改为**1年报到** |
| **税务优惠** | ①海外收入免税（适用于前三类）②高技能专业人士17%固定所得税率 |
| **工作许可** | 提供**数字工作许可**（Digital Work Permit），免除外籍:泰籍4:1比例 |
| **健康保险** | 需覆盖≥$50,000，或以银行存款$100,000替代 |
| **转永居** | LTR为**非移民签证**，持有满3年后可申请泰国PR（来源：Forbes & Partners, 2025） |
| **来源** | 2022年9月推出，2025年设统一服务中心TIESC |

### 3.3 泰国工作签证+工作许可（Non-Immigrant B Visa + Work Permit）

> 来源：泰国就业厅、BOI

| 维度 | 详情 |
|---|---|
| **签证类型** | Non-Immigrant B（商务/工作签证） |
| **申请条件** | 泰国注册公司雇佣，注册资本≥**200万泰铢**/每外籍员工 |
| **外国:泰籍比例** | 一般**1:4**（每1外国人需4名泰籍员工），BOI企业可豁免 |
| **薪资门槛** | 视国籍和职业而定（如西欧/北美/日本等国通常≥5万泰铢/月） |
| **签证有效期** | 初始**90天单次入境**，后续可延至**1年**（年年续签） |
| **工作许可** | 需单独申请Work Permit，有效期与签证同步 |
| **转永居** | 持有Non-Immigrant签证+工作许可连续**3年**后可申请PR（泰国移民局每年仅开放一次申请窗口，来源：PKF Thailand 2025） |
| **入籍** | PR满**5年**（即总共约8年）后可申请公民；另一路径为与泰国人结婚（PR满3年） |
| **福利** | 可入泰国社保体系、购车购房 |

### 3.4 泰国短期签证+移民路径

| 路径 | 详情 |
|---|---|
| **短期签证** | 中国公民落地签15天/电子签30-60天 |
| **Non-B→PR→公民** | 最常见工作移民路径（约8-10年） |
| **LTR→PR** | 高净值/高技能快速长期居留路径 |
| **精英签证** | 纯生活方式签证，无移民路径 |

## 四、马来西亚（Malaysia）

### 4.1 投资移民：MM2H第二家园（2024修订版）

> 来源：马来西亚旅游部/移民局（2024年6月修订）、Alestria Property（2026年5月更新）

#### MM2H半岛（Mainland MM2H）2026现行四级分类

| 层级 | 定期存款（USD） | 签证有效期 | 最低年龄 | 最低居住要求 | 参与费（马币） |
|---|---|---|---|---|---|
| **SEZ（经济特区）- 21-49岁** | $65,000 | **10年**可续 | 21岁 | 90天/年 | RM 1,000 |
| **SEZ - 50岁以上** | $32,000 | **10年**可续 | 50岁 | 无 | RM 1,000 |
| **银卡（Silver）** | $150,000 | **5年**可续 | 25岁 | 90天/年 | RM 1,000 |

| **白金（Platinum）** | $1,000,000 | **20年**可续 | 25岁 | 90天/年 | RM 200,000 |

| 维度 | 详情 |

| **转永居条件** | MM2H**不直接通往PR**（来源：马来西亚移民局） |

| **福利** | 无公共医疗/教育福利（需自购私立保险）；可99年租赁产权/永久产权购房（视州） |
| **Sarawak S-MM2H** | 独立管理：个人月收入RM10,000/夫妻RM15,000或流动资产RM100,000/夫妻RM200,000；定存RM500,000；10年可续；30天/年居住要求；允许有限兼职 |
| **来源年份** | 2024年6月重大修订，2026年实施确认 |

### 4.2 技术/雇主移民：就业准证（Employment Pass）

### 4.3 人才居留证：RP-T（Residence Pass-Talent）

> 来源：TalentCorp Malaysia 官网 rpt.talentcorp.com.my

| 维度 | 详情 |
|---|---|
| **申请条件** | ①已持EP在马来西亚工作≥**3年** ②总工作经验≥**5年** ③月薪≥**RM15,000**（不含奖金）④马来西亚税号 ⑤专业领域符合需求 |
| **有效期** | 最长**10年**（以护照有效期为准），可续签 |
| **特权** | 可自由换雇主（无需重新申请）、配偶可工作、子女可读书 |
| **转永居** | RP-T不自动转PR，但被视为最接近PR的长期居留身份 |
| **入籍条件** | 马来西亚入籍极难——需居住≥**10年**（12年中至少10年）、精通马来语、放弃原国籍（通常不批准）（来源：马来西亚联邦宪法） |

### 4.4 马来西亚短期签证+移民路径

| 路径 | 详情 |
|---|---|
| **短期签证** | 中国公民免签30天（2023年12月起） |
| **MM2H** | 纯长期居留生活方式签证，无明确PR/入籍路径 |
| **EP→RP-T** | 最现实的外籍专业人士长期居留路径 |
| **马来西亚PVIP（Premium Visa Programme）** | 2022年新设，定存RM100万+年收入RM48万，20年居留，但2024年后接受度有限 |
