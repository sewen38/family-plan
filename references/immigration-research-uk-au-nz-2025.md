## 1.2 创新创始人签证（Innovator Founder Visa）

| 维度 | 详情 |
|------|------|
| **申请条件** | (1) 商业想法必须新（不能加入已在运营的企业）、创新（与市场上任何现有业务不同）、可行且有增长潜力、可扩展（包括创造就业和进入全国/国际市场）；(2) 获得英国认可的背书机构（Endorsing Body）的背书；(3) 英语能力：B2水平（听说读写）；(4) 足够个人储蓄支持在英生活 [gov.uk, 2025] |
| **投资额** | **无最低投资金额要求**，但须向背书机构证明有足够资金及来源 [gov.uk, 2025] |
| **签证有效期** | **首次3年**，可续签，无续签次数上限，每次续签3年 [gov.uk, 2025] |
| **单次停留时长** | 签证有效期内可自由出入境；但若离境超2年ILR可能失效 |
| **续签条件** | 需在签证到期前重新获得背书机构的背书，证明商业进展 [gov.uk, 2025] |
| **转永居(ILR)条件** | 在英国居住**3年**后可申请ILR；须通过「Life in the UK」考试（18-64岁）；需证明商业满足背书要求 [gov.uk, 2025] |
| **入籍条件** | ILR后满12个月可申请入籍；过去5年内离境不超过450天，最后一离境年不超过90天；通过Life in the UK测试和英语B1 [gov.uk, 2025] |
| **费用** | 境外申请 £1,357/人；境内续签/转换 £1,693/人；背书费 £1,000（付给背书机构）；每次与背书机构会面 £500（至少2次）；IHS医疗附加费每年£1,035/人 [gov.uk, 2025] |
| **福利与限制** | |
| — 工作 | 可为自己的企业工作（包括担任董事或自雇）；可从事企业外的工作，但须至少为Level 3资格级别的工作 [gov.uk, 2025] |
| — 教育 | 可学习 |
| — 医疗 | 支付IHS后可使用NHS |
| — 养老 | 不可申领公共福利（public funds）和国家养老金（State Pension） |
| — 就业自由 | 不能做职业运动员（如体育教练）[gov.uk, 2025] |
| — 家人 | 可携带配偶和子女作为"家属"（dependant）[gov.uk, 2025] |

## 1.3 技术工人签证（Skilled Worker Visa）

| 维度 | 详情 |
|------|------|
| **申请条件** | (1) 英国雇主已获Home Office批准为担保方；(2) 持有雇主发出的「担保证书」(CoS)；(3) 从事符合资格的紧缺职业清单上的工作；(4) 达到最低薪资门槛；(5) 英语B1水平（听说读写）[gov.uk, 2025] |
| **薪资要求（2025年7月起）** | **标准门槛 £41,700/年**（£17.13/小时）或职业的"going rate"全额，取较高者；非STEM博士 £37,500（going rate的90%）；STEM博士/新人 £33,400（going rate的80%/70%）；ISL清单角色 £33,400 [LinkedIn/Home Office, 2025]；NHS医护有国家薪资表独立标准 [NHS Employers, 2025] |
| **签证有效期** | **最长5年**，可无限次续签 [gov.uk, 2025] |
| **单次停留时长** | 有效期内可自由出入境 |
| **续签条件** | 续签时须满足当时的最新薪资门槛；保护已有签证持有者的过渡安排存在（2026年12月前续签可适用£31,300较低门槛）[NHS Employers, 2025] |
| **转永居(ILR)条件** | 在英国居住**5年**后可申请ILR；须仍在担保工作中且薪资达到going rate全额；无需额外证明英语（已在签证阶段满足）[gov.uk, 2025] |
| **入籍条件** | ILR后满12个月可申请入籍；居住要求同上 [gov.uk, 2025] |
| **费用** | 申请费 £719-£1,500（根据签证时长和地点）；IHS £1,035/年/人 [gov.uk, 2025] |
| **福利与限制** | |
| — 工作 | 仅限担保职位；可做额外工作（特定条件）；不能换雇主/job须更新签证 [gov.uk, 2025] |
| — 教育 | 可学习 |
| — 医疗 | 支付IHS后可使用NHS |
| — 养老 | **不可领取公共福利或国家养老金** [gov.uk, 2025] |
| — 就业自由 | 换雇主需申请更新签证 |
| — 家人 | 配偶和子女可作为家属，家属可工作（除运动员/教练外）[gov.uk, 2025] |

| 维度 | 详情 |
|------|------|
| **申请条件** | 年满18岁；在以下领域为领导者或潜在领导者：**学术/研究、艺术与文化、数字技术**。两条申请途径：(1) 获得认可背书机构背书；(2) 赢得指定的"著名奖项"（Prestigious Prize）可直接申请。必须在科学、工程、人文、社科、医学、数字技术或艺术文化领域 [gov.uk, 2025] |

| — 医疗 | 支付IHS后NHS |

## 1.5 短期签证转永居路径（英国汇总）

| 短期签证 | 可否转ILR | 路径 |
|----------|----------|------|
| Student Visa | 不可直接转ILR | 需转Graduate→Skilled Worker/Innovator等 |
| Graduate Visa | 不可直接转ILR（不计入5年ILR居住） | 转Skilled Worker(再5年) |
| Youth Mobility Scheme | 不可直接转ILR | 转Skilled Worker等 |
| Health and Care Worker Visa | 可，5年 | 同Skilled Worker，但费用更低、免IHS [gov.uk, 2025] |
| Scale-up Worker Visa | 可，5年 | 前6个月需担保，之后可自由工作 |
| Family Visa (配偶) | 可，5年 | 配偶签证5年路径 |

## 1.6 英国ILR后福利总览

| 福利类别 | ILR持有者 | 入籍后 |
|----------|----------|--------|
| **医疗(NHS)** | 全面免费（已缴IHS） | 全面免费 |
| **教育** | 本地学费，可申请学生贷款 | 同左 |
| **公共福利** | 符合条件可申请（Universal Credit、住房补贴等） | 同左 |

## 2.1 投资移民（BIIP 188类别）— 已关停

| 维度 | 详情 |
|------|------|
| **状态** | **BIIP（商业创新与投资计划，含188签证所有子类别）于2024年7月31日永久关闭**，不再接受新申请 [homeaffairs.gov.au, 2024] |
| **关停范围** | 包括：商业创新流、投资者流、重大投资者流(SIV, 500万澳元)、高端投资者流(PIV, 1500万澳元)、企业家流 [homeaffairs.gov.au, 2024] |
| **已提交申请** | 2024年7月31日前提交的申请将继续审理 [homeaffairs.gov.au, 2024] |
| **2024-25财年配额** | BIIP仅分配1,000个名额（含主副申请人），主要用于处理积压 [homeaffairs.gov.au, 2025] |

### 替代路径 — 国家创新签证（National Innovation Visa, Subclass 858, NIV）

| 维度 | 详情 |
|------|------|
| **推出时间** | **2024年12月6日**，取代原全球人才签证(GTI Subclass 858)和BIIP [One Planet Migration Law, 2024] |
| **申请条件** | (1) 拥有国际公认的**杰出和卓越成就**记录；(2) 在专业领域内有持续的知名度；(3) 有能力对澳大利亚创新、生产力和竞争力做出重大贡献；(4) 须获得在相关领域有全国声誉的澳大利亚个人或组织的提名（Form 1000）；(5) 须提交EOI并获得邀请（60天内递交签证申请）[homeaffairs.gov.au / Align Law, 2025] |
| **投资额** | **无固定投资金额要求**（非投资类签证），重点考察个人成就与贡献潜力 |
| **年龄** | 无严格年龄上限；18岁以下或55岁以上需证明对澳大利亚有特殊利益 [homeaffairs.gov.au, 2025] |
| **英语** | 仅需功能性英语（Functional English），未达标可交额外费用 [homeaffairs.gov.au, 2025] |
| **签证类型** | **永久居留签证**（一步到位PR）[homeaffairs.gov.au, 2025] |
| **优先领域(Tier 1)** | 关键科技(Critical Technologies)、健康产业(Health Industries)、可再生能源与低排放技术(Renewables & Low Emission Technologies) [homeaffairs.gov.au, 2025] |
| **优先领域(Tier 2)** | 农业食品与农业科技、国防与太空、教育、金融服务与金融科技、基础设施与交通 [homeaffairs.gov.au, 2025] |
| **优先级排序** | ①获国际顶级奖项者（任何领域）→ ②联邦/州/领地政府提名 → ③Tier 1领域杰出成就者 → ④Tier 2领域 → ⑤其他合格申请人 [homeaffairs.gov.au, 2025] |
| **福利** | 永久居留，可在澳无限期生活、工作、学习；可加入Medicare；可担保家庭成员；可申请入籍（满足居住要求后）[homeaffairs.gov.au, 2025] |

## 2.4 澳大利亚PR福利总览

| 福利类别 | 临时签证（482/491等） | PR（189/190/186/858等） | 入籍 |
|----------|----------------------|------------------------|------|
| **医疗(Medicare)** | 部分签证（如491）可申请 | 全面免费公立医疗 | 同左 |
| **教育** | 国际生学费 | 本地学费（部分州PR可享免费公立中小学教育）；大学可申请CSP/HECS-HELP贷款 | 同左 |
| **Centrelink福利** | 有限 | 居住满2-4年后可申请大部分福利（失业金、育儿补贴等）[Australian Skilled Migration, 2025] | 无等待期 |
| **养老(Age Pension)** | 不可 | 居住满10年（其中5年连续）后可申请 | 同左 |
| **首次置业** | 不可 | 可申请First Home Owner Grant(FHOG)、印花税减免 [Australian Skilled Migration, 2025] |
| **就业** | 受签证条件限制 | 无限制（除军队/部分公共服务） | 可参军、参公职 |
| **选举权** | 无 | 无 | 有 |

### Skilled Migrant Category Resident Visa — 6分制

- 年龄：55岁及以下。
- 必须有新西兰认证雇主 skilled job offer 或当前在新西兰 skilled job。
- 必须达到6分。分数来自一个核心技能类别（学历/职业注册/收入）+ 可叠加新西兰技能工作经验。
- 工作要求：ANZSCO 1-3通常至少NZD $35/小时；ANZSCO 4-5通常至少1.5倍中位数（NZD $52.50/小时，按2025口径）。
- 英语：需能说和理解英语，常见要求雅思6.5或同等，具体以INZ为准。
- 居民签证后2年可申请Permanent Resident Visa（PRV），入籍通常需5年居住且满足天数。

### Green List Straight to Residence — Tier 1直通居留

- 年龄55岁及以下。
- 有认证雇主offer或已在新西兰从事Green List Tier 1职业。
- 满足该职业具体要求（注册/学历/薪资等）。
- 薪资至少为该职业规定工资，或未规定时不低于2025中位数工资NZD $35/小时。
- 获批后为Resident Visa，2年后可转PRV。

### Green List Work to Residence — Tier 2工作转居留

- 年龄55岁及以下。
- 已持工作签证并在新西兰Green List Tier 2岗位工作满24个月。
- 当前有/从事认证雇主Tier 2岗位，满足职业要求和薪资要求。
- 获批后为Resident Visa，2年后可转PRV。

### Accredited Employer Work Visa (AEWV)

- 临时工作签证。
- 必须有认证雇主提供全职工作offer，雇主须完成job check。
- 通常需2年相关经验或NZQCF Level 4以上相关学历（本科及以上可更宽）。
- 不可包含配偶和子女在同一申请中，家属需另行申请。
- AEWV本身不是居留，但可用于积累SMC/Green List工作经验。
- 需警惕付费买工作、虚假offer和雇主合规风险。

### 家庭/配偶路径

- 新西兰公民/居民配偶路径通常要求真实稳定关系、共同居住证据等。
- 可从配偶工签/访签逐步转居留，具体周期取决于关系时长和证据。

### 新西兰评估问卷项目覆盖标准

评估问卷必须覆盖：
1. Active Investor Plus Growth / Balanced；
2. SMC 6分制；
3. Green List Straight to Residence；
4. Green List Work to Residence；
5. AEWV；
6. 配偶/家庭路径；
7. Resident Visa → PRV → Citizenship 时间线。

## 3.7 新西兰补充：绿名单细分、双倍高薪、派遣/ICT（2026-06-06）

### Green List 绿名单细分

新西兰 Green List 分为 Tier 1 和 Tier 2：

| 类别 | 居留路径 | 核心逻辑 |
|------|----------|----------|
| Tier 1 | Straight to Residence | 符合Tier 1职业、认证雇主offer、薪资/注册/学历要求后，可直接申请Resident Visa |
| Tier 2 | Work to Residence | 先持工作签证在新西兰对应Tier 2岗位工作满24个月，再申请Resident Visa |

常见绿名单行业方向包括：
- 医疗健康：医生、护士、专科医疗、心理/社工等；
- 工程与建筑：土木、电气、机械、结构、测量、项目管理等；
- ICT/数字技术：软件工程、ICT安全、系统架构、数据/网络等（具体以Green List角色为准）；
- 教育：中小学教师、幼教等部分岗位；
- 建筑/基础设施技工：部分技工、建筑相关岗位；
- 农业/初级产业及其他紧缺岗位。

评估时必须先问：职业是否明确在Green List，属于Tier 1还是Tier 2；如果客户不确定，只能标注“待职业代码/岗位职责核实”。

### 双倍高薪 / 高收入路径

在SMC 6分制中，收入可作为核心技能分来源之一：

| 收入水平 | 评估意义 |
|----------|----------|
| ≥中位数工资（2025口径NZD $35/小时） | 可满足部分 skilled job 基础工资要求 |
| ≥1.5倍中位数（约NZD $52.50/小时） | 对ANZSCO 4-5岗位/部分SMC收入路径有意义 |
| ≥2倍中位数（约NZD $70/小时） | 高薪技能岗位，通常可作为SMC高收入路径强信号 |
| ≥3倍中位数（约NZD $105/小时） | 最高收入强信号，SMC收入路径可更强，但仍需认证雇主skilled job |

注意：高薪不等于自动居留。仍需认证雇主、真实岗位、岗位职责、合同、薪资可验证、健康品行和英语等要求。

### 派遣 / ICT / 跨国公司内部调动优化

新西兰没有一个等同于美国L-1的“直接ICT移民签证”。跨国公司派遣通常需要根据目的选择路径：

| 场景 | 可能路径 | 定位 |
|------|----------|------|
| 海外公司向新西兰分公司/关联公司派遣员工长期工作 | AEWV认证雇主工签 | 需要新西兰雇主认证、job check、岗位/薪资/经验匹配 |
| 短期特定项目、安装调试、技术交付、管理任务 | Specific Purpose Work Visa / 其他短期工作签证 | 适合明确项目、期限和目的，不是居留路径 |
| 高管/专家派驻且薪资高、岗位符合技能要求 | AEWV → SMC/Green List路径 | 可作为先登陆工作，再积累居留条件 |
| 海外企业计划在新西兰设立运营实体 | 先做公司/雇主认证/岗位设计，再AEWV或投资路径 | 需要商业实质和合规雇佣安排 |

评估时必须问：是否有海外母公司、新西兰分/子公司或客户项目、是否为高管/专才、派遣时长、薪资、岗位是否可由新西兰认证雇主承担。派遣/ICT本身不应被包装为直接居留；它是“先工签/项目落地，再转SMC/Green List/雇主路径”的优化工具。
