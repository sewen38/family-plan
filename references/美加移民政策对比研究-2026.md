## 一、【投资移民】

### 1A. 美国 EB-5 投资移民

| 维度 | 详情 |
|------|------|
| **法律依据** | 2022年《EB-5改革与诚信法案》(RIA)，2022年5月13日生效 |
| **申请条件** | ①年满18岁 ②投资合法来源资金 ③创造至少10个全职就业 ④投资于新商业企业(NCE) |
| **投资金额（2025-2026）** | **TEA（目标就业区/基建项目）：$800,000**（约584万人民币）；**非TEA：$1,050,000**（约767万人民币）。RIA设定该金额有效期至2027年9月30日，之后按通胀调整。 |
| **签证有效期** | 获批后获2年**有条件永久居留权**（Conditional LPR） |
| **转无条件永居** | 2年期满前90天内提交 **I-829** 申请，证明：①投资已持续维持 ②创造了10个就业。获批后获10年无条件绿卡。 |
| **入籍条件** | 自**初始有条件居留权获批起第5年**可申请入籍（无论I-829是否已批准）。需满足：5年内至少30个月在美实际居住、良好道德品质、通过英语及公民考试。 |
| **排期情况（2026.6）** | 《2026年6月签证公告》：预留签证类别（乡村/高失业区/基建）全部"Current"无排期；非预留中国内地Final Action Date有排期（约2015-2016年）。印度非预留2026年1月大幅前进。 |
| **来源** | USCIS Policy Manual Vol.6 Part G；IIUSA June 2026 Visa Bulletin；RIA 2022；Cohen Tucker Law (2026) |

### 1B. 美国 E-2 条约国投资者签证

| 维度 | 详情 |
|------|------|
| **适用条件** | ①申请人须为**E-2条约国公民**（80+国家，包括加拿大、英国、日本、德国、韩国、土耳其、格林纳达等，**中国大陆不在其中**）②对美国企业进行"实质性投资"（substantial investment）③持股≥50%并积极参与经营管理 |
| **投资额** | 法律无固定最低金额，但须"实质性"且"不可撤销"。通常建议至少 $100,000+，投入资金与业务类型成比例 |
| **签证有效期** | 通常签发 **2-5年**（取决于对等协议），每次入境通常给2年停留期 |
| **续签** | **无限次续签**，只要企业持续运营并符合要求 |
| **能否转绿卡** | **E-2本身为非移民签证，不能直接转绿卡**。但可通过以下路径间接转：①EB-5投资 ②EB-1C跨国高管 ③EB-2/EB-3雇主担保 ④亲属移民。需注意：E-2不是"双重意图"签证，进入美国时须保持非移民意图，但可后续转换 |
| **关键限制** | 中国籍申请人须先获得E-2条约国国籍（如格林纳达CBI），并在该国居住一定时间后方可申请 |
| **来源** | USCIS；U.S. Department of State Foreign Affairs Manual (9 FAM 402.9)；NNU Immigration (2026) |

### 1C. 加拿大 SUV 创业签证 (Start-Up Visa)

## 二、【技术移民】

### 2A. 美国 H-1B 工作签证

| 维度 | 详情 |
|------|------|
| **基本条件** | ①学士及以上学位 ②职位须为"专业职业"(specialty occupation) ③美国雇主担保 ④工资须达prevailing wage |
| **配额** | 年度常规配额：**65,000** + 硕士及以上豁免配额：**20,000** = 总计85,000。非营利研究机构、大学等免配额 |
| **抽签制度** | FY2027（2026年3月开放电子注册）：注册费$215/人。2026年3月注册，抽签后中签者提交I-129 |
| **签证有效期** | 初次3年，可续至最多**6年**（如进入绿卡流程PERM/I-140可超6年延期） |
| **转绿卡路径** | H-1B是"双重意图"签证，标准路径：**PERM劳工证→I-140移民申请→Priority Date排期→I-485调整身份→绿卡**。全程约3-8年（视国籍和类别）、印度籍排期极长（10年以上） |
| **重要变化** | 2025年"一人一签"制度（beneficiary-centric）生效，减少重复注册作弊 |
| **来源** | USCIS (uscis.gov FY2026 H-1B Cap Alert)；Gozel Law (2026) |

### 2B. 美国 EB-1A 杰出人才 & EB-2 NIW 国家利益豁免

| 维度 | EB-1A 杰出人才 | EB-2 NIW 国家利益豁免 |

| **标准** | 满足10项中3项（如获奖/论文/高薪/评审/媒体报道等），或一次性重大成就 | ①高级学位或杰出能力 ②Dhanasar三要素：事业具有实质性价值和国家重要性、申请人有能力推进、免除劳工证对美国有利 |

### 2C. 加拿大 Express Entry 快速通道

| 维度 | 详情 |
|------|------|
| **子类别** | ①**联邦技术工人(FSWP)** ②**加拿大经验类(CEC)** ③**联邦技工(FSTP)** |
| **FSWP基本条件** | ①过去10年内至少1年连续全职(或等同)技术工作经验(NOC TEER 0-3) ②语言CLB 7（雅思6.0各科）③学历认证(ECA) ④六项评分≥67/100（年龄、学历、语言、经验、工作offer、适应力） |

| **类别抽签** | 2026年继续实行category-based draws：法语、医护、STEM、技工、运输、农业六大优先类别 |

| **2026新动向** | ①联邦继续优先"加拿大境内申请人" ②2026年计划将33,000临时工签持有者转PR ③为美国H-1B持有者开辟快速PR通道 |
| **来源** | IRCC (Canada.ca)；Moving2Canada Draw Tracker；Green & Spiegel Draw Database |

## 三、【雇主移民】

### 3A. 美国 EB-3 雇主担保移民

| 维度 | 详情 |
|------|------|
| **子类别** | EB-3(A) 专业人士(Skilled Workers)：需本科+2年经验/培训；EB-3(B) 专业人士(Professionals)：需美国本科或同等学历；EB-3(C) 非技术工人(Other Workers)：<2年培训 |
| **申请条件** | ①美国雇主担保永久全职职位 ②**PERM劳工证**（证明无美国工人可用）③学历/经验符合职位要求 |
| **排期（2026.6）** | 中国EB-3有排期，Final Action Date约2021年；EB-3 Other Workers更慢。印度EB-3约2013-2014年。 |
| **全程时间估算** | PERM（8-14月）→ I-140（可加急15天）→ 等排期 → I-485（6-18月）。全程约**2-5年**（中国出生），印度/菲律宾更长 |
| **转绿卡** | EB-3直接为绿卡类别，I-485获批即为永久居民 |
| **来源** | USCIS；DOL (PERM)；Boundless June 2026 Visa Bulletin；Alma EB-3 Guide (2026) |

### 3B. 加拿大雇主担保移民

| 项目类别 | 详情 |
|------|------|

| **来源** | IRCC；CIC News "New Pathways 2026" |

## 四、【短期签证+移民路径】

### 4A. 美国 B1/B2 短期签证

| 维度 | 详情 |
|------|------|
| **签证有效期** | 通常**10年**多次入境（中国公民），实际以使馆签发为准 |
| **单次停留时长** | CBP官员在入境时决定，通常**最多6个月（180天）**，I-94上会载明具体日期 |
| **能否工作/学习** | **不可**工作或就读学位课程（短期娱乐性学习除外） |
| **能否转换身份** | **可以申请Change of Status(COS)**转为F-1学生/H-1B工作/L-1等签证，但须满足：①入境时即有意申请COS则属于"misrepresentation"（欺诈），须证明入境时有非移民意图且后来情况变化 ②通常需等待90天以上再申请（90天规则）③B1/B2转身份审理期间不能开始新活动（如上学/工作） |
| **延期** | 可申请I-539延期最多6个月 |
| **⚠️ 2025变化** | 2025年9月2日起，B1/B2签证申请人（除12个月内续签外）将不再获发10年全有效期签证，有效期缩短（具体依对等原则和使领馆裁量） |
| **来源** | U.S. Department of State (travel.state.gov)；USCIS；CBP I-94 |

### 4B. 加拿大访客签证（Visitor Visa）转学签/工签

| 路径 | 详情 |
|------|------|
| **访客→学签** | 加拿大**允许**外国访客在境内递交学签申请（IRCC "Study permit from within Canada"）。条件：①持有加拿大有效访客身份 ②获指定学习机构(DLI)录取信 ③提供省级认证信(PAL)（多数申请人需要）④完成预科/前置课程(DLI规定) |
| **访客→工签** | 特定情况下可在境内申请工签：①持LMIA或LMIA豁免的job offer ②配偶/同居伴侣持有效学签/工签 ③已毕业的PGWP申请者。注意：仅持有访客签证者不可直接申请工签，需满足特定豁免条件 |
| **Flagpoling停止** | 2025年加拿大终止了flagpoling（出入边境口岸换签），学签和工签续签须通过IRCC在线递交，不再能在边境当场办理 |
| **访客记录(VR)** | 到期前30天可申请Visitor Record延期在加停留 |
| **来源** | IRCC (Canada.ca/study-permit/apply)；AKM Law "Flagpoling Ended" (2025)；AdmissionHub (2025) |

## 五、【入籍条件】

### 5A. 美国入籍条件

| 维度 | 详情 |
|------|------|
| **居留要求** | 持绿卡**至少5年**（美国公民配偶为**3年**） |
| **实际居住** | 5年内累计在美**至少30个月**（即一半时间） |

## 六、【福利对比】

### 美国 vs 加拿大：永久居民/公民福利一览

| 维度 | 🇺🇸 美国 | 🇨🇦 加拿大 |
|------|------|------|
| **医疗保险** | 无全民医保。65岁以上/残疾：Medicare（须工作10年+纳税）。低收入：Medicaid（各州不同，PR须等5年）。大部分人靠雇主/自购商业保险。**⚠️2026-2027变化：H.R.1预算协调法案收紧移民医保资格，2027年1月起约140万合法移民将失去Medicaid/ACA补贴。PR不受影响（Green Card holders keep eligibility）。** | 全民医保（Medicare/省医保计划）。PR持省医保卡后基本医疗**免费**（看医生/住院/急诊），牙科/处方药有限覆盖。**部分省有3个月等待期**（BC、安省、魁省等）。2024年推出的加拿大牙科保健计划(CDCP)对家庭收入<$90,000的居民提供牙科补贴。 |

### 2E. 美国 O-1A / O-1B 杰出能力非移民签证 + O-3 家属

| 维度 | O-1A 杰出能力 | O-1B 艺术/影视杰出能力 | O-3 家属 |
|------|------|------|------|
| **适用人群** | 科学、教育、商业、体育领域具备 extraordinary ability 的申请人 | 艺术领域 extraordinary ability，或影视行业 extraordinary achievement 的申请人 | O-1/O-2 主申请人的配偶及21岁以下未婚子女 |
| **签证性质** | 非移民工作签证，不直接给绿卡，但可作为EB-1A/NIW前置登陆和履历增强路径 | 非移民工作签证，不直接给绿卡，但可为艺术/影视人才积累美国项目履历 | 陪同身份，不可工作，可在美国学习 |
| **核心要求** | 需美国雇主/美国代理人/美国项目方提交I-129；满足O-1A证据标准（8项中通常满足3项）或一次性重大国际奖项 | 需美国雇主/代理人/项目方提交I-129；艺术类通常6项中满足3项，影视类标准更高，强调行业认可和重大成就 | 与主申请人关系证明；随主申请人身份有效 |
| **常见证据** | 奖项、协会会员、媒体报道、评审经历、原创贡献、学术/行业文章、关键职位、高薪 | 主演/主创/核心职位、媒体报道、重大商业/评论成功、知名机构项目、高薪、专家推荐信 |
| **雇主/赞助** | 必须有美国雇主、美国代理人或项目合同；不允许纯自雇无项目申请 | 同左；艺术/影视可通过agent整合多个项目 | 依附主申请人 |
| **有效期** | 首次最长3年；后续按项目需要1年一续，可无限续签 | 同左 | 与O-1主申请人一致 |
| **转绿卡组合** | 常见组合：O-1A先赴美工作/创业/项目落地 → 同步准备EB-1A或NIW → I-140获批后转I-485/领馆程序 | O-1B先赴美开展艺术/影视项目 → 强化媒体/奖项/商业成功 → 后续EB-1A或NIW（艺术/文化贡献方向） | O-3不可直接转绿卡，但可随主申请人I-485作为家属调整身份 |

| **限制** | 不是绿卡；必须有真实美国工作/项目；身份与项目/雇主绑定 | 影视类标准高于普通艺术类；证据需体现行业认可 | 不可工作 |
| **来源** | USCIS O-1 Visa: Individuals with Extraordinary Ability or Achievement；USCIS Policy Manual；9 FAM 402.13 |

#### EB-1A + NIW + O-1 组合策略

| 客户类型 | 推荐组合 | 逻辑 |
|------|------|------|
| 已满足EB-1A 3项以上且有美国项目 | **O-1A/O-1B快速登陆 + EB-1A同步递交** | O-1解决赴美工作/项目身份，EB-1A解决绿卡；两者证据高度重合 |
| 硕博/STEM/专业人士，EB-1A不足3项 | **O-1A + NIW** | O-1A先以项目/雇主身份赴美，NIW走国家利益绿卡；O-1经历可强化NIW三要素 |
| 企业家/高管/产品负责人 | **O-1A商业杰出能力 + NIW/EB-1A双轨** | 商业成就、媒体、融资、专利、行业影响可同时服务O-1A和EB-1A/NIW |
| 艺术/影视/设计/内容创作者 | **O-1B + EB-1A艺术类** | O-1B适合先进入美国项目体系，后续用项目、媒体、奖项、商业成绩冲EB-1A |
| 家庭随行 | **O-1 + O-3** | 配偶和21岁以下未婚子女用O-3随行，O-3不可工作但可学习 |

> 实务提醒：O-1不是移民签证，但与EB-1A/NIW常见“证据共用、节奏错位”的组合策略。O-1用于先入境和开展项目，EB-1A/NIW用于长期绿卡目标。实际策略需由美国移民律师结合证据包设计。
