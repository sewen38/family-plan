## 1. 葡萄牙 Portugal

### 1.1 投资移民 — 黄金签证（Golden Visa / ARI）

| 维度 | 详情 |
|------|------|
| **政策状态** | ✅ 仍可用，但房产路径2025年4月已关闭 |
| **房产投资** | ❌ 2025年4月起不再接受房产购买作为合格投资 |
| **基金投资** | 投资 **€500,000** 于葡萄牙证监会(CMVM)监管的风险投资基金 |
| **文化捐赠** | 最低 **€200,000** 捐赠于文化艺术项目 |
| **科研投资** | 最低 €500,000 科研活动投资 |
| **创造就业** | 创造至少10个工作岗位 |
| **签证有效期** | 初始居留卡2年，续签3年 |
| **居住要求** | 每年平均居住7天（第一年7天，后续每两年14天） |
| **续签条件** | 维持投资、无犯罪记录、满足居住要求 |
| **转永居** | 合法居住 **5年** 后可申请永久居留 |
| **入籍条件** | ⚠️ 2026年5月3日新法颁布：非EU/CPLP公民从5年→**10年**；EU/CPLP公民**7年**（需A2葡语、入籍考试） |
| **教育** | 子女可就读葡公立学校（免费）；国际学校约€6,000-18,000/年 |
| **医疗** | 缴纳社保后使用公立医疗（SNS），挂号费约€5-15；私立保险约€30-100/月 |
| **养老** | 累计缴纳社保15年以上可领养老金；NHR税惠（2024年已大幅收紧，详见1.6节） |
| **就业** | 黄金签证持有者可自由在葡工作或创业 |
| **社会** | 居住期间享申根区免签通行 |

> 📌 来源：GetGoldenVisa (2026.06)、Mercan Group (2026)、imin-portugal.com (2026)、globalresidenceindex.com (2026)

### 1.2 D7 被动收入签证（Passive Income / Retirement Visa）

| 维度 | 详情 |
|------|------|
| **适用人群** | 非EU/EEA/瑞士公民，有稳定被动收入（养老金、租金、股息、理财收入等） |
| **收入门槛** | 主申：**€920/月**（2026年，=葡萄牙最低工资100%）；配偶+50%（€460/月），每名受养子女+30%（€276/月） |
| **存款要求** | 建议持有至少€11,040/年的存款证明 |
| **居住证明** | 需提供葡境内住宿证明（租赁或购买合同） |
| **签证有效期** | 初始居留许可 **2年**，续签后 **3年** |
| **居住要求** | 每2年内在葡累计居住不少于**16个月**（或连续18个月） |
| **禁止事项** | 不可在葡萄牙从事本地雇佣工作（远程工作可以） |
| **续签条件** | 继续满足收入门槛、保持住宿、无犯罪记录 |
| **转永居** | 合法连续居住 **5年** 后可申请永居 |
| **入籍** | 同上述新法：非EU公民10年/EU及CPLP公民7年（来源：NTL International, 2026.05） |
| **福利** | 教育、医疗、养老等同上；D7持有人可远程工作，但不可本地受雇 |

> 📌 来源：imin-portugal.com (2026)、GetGoldenVisa (2026)、immigrantinvest.com (2026)

### 1.3 D2 企业家签证（Entrepreneur Visa）

| 维度 | 详情 |
|------|------|
| **适用人群** | 非EU/EEA/瑞士公民，欲在葡萄牙创业、开设分公司、投资本地企业或独立执业者 |
| **核心条件** | 提交可行商业计划书、拥有葡NIF税号、葡银行账户、住宿证明、健康保险、无犯罪记录 |
| **最低资本** | 无法定最低资本要求，但须证明有足够资金支持创业及生活（建议≥€5,000-10,000） |
| **签证有效期** | 初始居留许可 **2年**，续签后 **3年** |
| **居住要求** | 每2年内在葡居住累计不少于**16个月**（或连续18个月） |
| **续签条件** | 商业计划执行情况评估、满足税务和社保义务 |
| **转永居** | 合法连续居住 **5年** 后可申请永居 |
| **入籍** | 同上新法 |
| **特点** | 自由职业者、创业者和企业家的首选路径 |

> 📌 来源：thegoldenportugal.com (2026)、imin-portugal.com (2026)、tytle.io (2026)

|------|------|

| **存款要求** | 至少**€11,040**存款证明 |

### 1.5 入籍重大更新（⚠️ 重点关注）

| 要点 | 详情 |
|------|------|
| **法律状态** | 2026年5月3日，总统颁布 Decreto n.º 48/XVII 修订《葡萄牙国籍法》 |
| **新规核心** | 入籍所需居住时间从 **5年延长至10年**（非EU/CPLP公民）；CPLP及EU公民为 **7年** |
| **居住起算点** | 从居留许可**实际签发之日**起算（此前2024年修订为从申请日起算） |
| **生效日期** | 待《共和国日报》(Diário da República) 公布后正式生效 |
| **永居未变** | 永居仍为**5年**，新法仅影响入籍时间线 |
| **语言要求** | A2级葡萄牙语考试（CIPLE）+ 公民知识测试 |
| **黄金签证** | 黄金签证项目本身未受影响；入籍时间延长不改变居留权 |
| **已申请者** | 总统在颁布声明中表示"已提交的程序不应受立法变更影响" |

> 📌 来源：NTL International (2026.05)、Jobbatical (2026)、globalresidenceindex.com (2026)、goldenvisas.com (2026)

## 2. 西班牙 Spain

### 2.1 投资移民 — 黄金签证（已关闭）

| 维度 | 详情 |
|------|------|
| **政策状态** | ❌ **2025年4月3日正式关闭**（《组织法1/2025》） |
| **替代路径1** | **数字游民签证**（详见2.3） |
| **替代路径2** | **非盈利居留(NLV)**（详见2.2） |
| **替代路径3** | **企业家签证**（创业/创新项目，需商业计划审批） |

> 📌 来源：goldenvisas.com (2025)、IMI Daily (2025)、costaluzlawyers.com (2025)

---

### 2.2 非盈利居留（Non-Lucrative Visa / NLV）

| 维度 | 详情 |
|------|------|
| **适用人群** | 非EU公民，有足够被动收入/储蓄，无需在西班牙工作（退休、财务独立人士首选） |
| **收入门槛** | 主申：**400% IPREM = €28,800/年**（2026年IPREM为€600/月）；每名受养人+100% IPREM（€7,200/年） |
| **存款建议** | 按至少1-2年生活费计算，约€28,800-57,600 |
| **健康保险** | 必须是西班牙认可的全面私人健康保险（无自付额） |
| **签证有效期** | 初始 **1年**，续签 **2年+2年** |
| **居住要求** | 每年在西班牙居住至少**183天**（否则影响续签和税务居民身份） |
| **禁止事项** | **严格禁止**在西班牙工作或从事任何专业活动 |
| **续签条件** | 继续满足收入门槛、保持健康保险、满足居住天数 |
| **转永居** | 合法连续居住 **5年** 后可申请永久居留 |
| **入籍** | 合法连续居住 **10年**（Ibero-American国家公民仅需2年），需通过DELE A2西语及CCSE文化考试 |
| **税收** | 年居住超183天即为税务居民，全球收入按累进税率（19%-47%）征税 |

> 📌 来源：immigrantinvest.com (2026)、jurospain.com (2026)、costaluzlawyers.com (2026)

---

### 2.3 数字游民签证（Digital Nomad Visa / DNV）

| 维度 | 详情 |
|------|------|
| **法律基础** | 2023年《创业法》(Startup Act) 引入 |
| **收入门槛** | 主申：**€2,849/月**（2026年=200%西班牙最低工资SMI）（来源：remoteworkeurope.eu 2026） |
| **境外收入** | 至少**80%收入**须来自非西班牙来源 |
| **工作类型** | 可为境外公司雇员、个体经营者/自由职业者（为西公司服务不超过20%收入） |
| **签证有效期** | 境外申请：**1年**；境内转居留许可：**3年**，可续签**2年** |
| **居住要求** | 无严格最低居住天数要求 |
| **续签条件** | 继续满足收入和工作条件 |
| **转永居** | 合法连续居住 **5年** 后可申请永居 |
| **入籍** | 合法连续居住 **10年**（Ibero-American国家公民仅2年） |
| **税收优惠** | 可申请"贝克汉姆法"：西班牙来源收入固定税率**24%**（最高€600,000/年），最长**6年**；境外非劳动收入免税 |

> 📌 来源：remoteworkeurope.eu (2026)、immigrantinvest.com (2026)、getgoldenvisa.com (2026)

---

### 2.4 西班牙入籍通用要求

| 维度 | 详情 |
|------|------|
| **标准入籍** | 合法连续居住 **10年** |
| **快速通道** | Ibero-American国家、菲律宾、赤道几内亚公民仅需**2年**；难民1年；与西公民结婚1年 |
| **语言考试** | DELE A2（西班牙语）+ CCSE（宪法与社会文化知识） |
| **双重国籍** | 西班牙一般不允许双重国籍（Ibero-American国家等有双边协议者除外） |

> 📌 来源：balcellsgroup.com (2026)、immigrationspain.es (2026)

## 3. 希腊 Greece

### 3.1 投资移民 — 黄金签证（Golden Visa）

| 维度 | 详情 |
|------|------|
| **政策状态** | ✅ 欧洲仍开放的最热门黄金签证之一 |
| **投资额分层（2026年）** | |
| └ **Zone A — €800,000** | 雅典(Attica)、塞萨洛尼基、米科诺斯、圣托里尼及人口>3,100的岛屿 |
| └ **Zone B — €400,000** | 希腊其他所有地区 |
| └ **Zone C — €250,000** | 仅限购买需翻修的房产或商业转住宅改造项目 |
| **其他投资方式** | 资本投资€400,000、政府债券€500,000、银行存款€500,000等 |
| **签证有效期** | 初始 **5年** 永居卡 |
| **居住要求** | **无最低居住要求** |
| **续签条件** | 每5年续签，仅需维持投资 |
| **不得短租** | 用于黄金签证的房产**不得**用于短期租赁（如Airbnb） |
| **转永居** | 黄金签证本身即为5年可续永居 |
| **入籍** | 需实际居住在希腊至少**7年**（每年≥183天），通过希腊语B1、历史和文化考试 |
| **教育** | 子女可入读公立学校（希腊语教学）或国际学校 |
| **医疗** | 缴纳社保后可享公立医疗；私人保险约€50-100/月 |
| **就业** | ⚠️ 黄金签证持有者**不可**在希腊工作（仅可担任自己公司的无薪董事） |
| **社会** | 申根区自由通行；家庭可包含配偶、21岁以下子女、双方父母 |

> 📌 来源：getgoldenvisa.com (2026.06)、elxis.com (2026)、buygreece.us (2026)、greecegolden-visa.com (2026)

## 4. 马耳他 Malta

### 4.1 永居计划（MPRP — Malta Permanent Residence Programme）

| 维度 | 详情 |
|------|------|
| **政策状态** | ✅ 2021年启动，2024-2025年多次修订（LN 57/2024, LN 310/2024, LN 146/2025） |
| **两种投资方案** | |
| └ **购房路径** | 购买价值≥**€375,000**的房产（马耳他或戈佐岛） |
| └ **租房路径** | 年租金≥**€14,000**的房产（租期至少5年） |
| **政府行政费** | **€60,000**（不可退还：申请时€15,000 + 获批后€45,000） |
| **政府捐赠** | **€37,000**（购房）或 €...（租房路径具体金额略有不同） |
| **慈善捐赠** | ≥**€2,000**给马耳他注册NGO |
| **资产证明** | 总资产≥**€500,000**（其中≥€150,000为金融资产）；或总资产≥€650,000（其中≥€75,000为金融资产） |
| **成人受养人费用** | 2025年7月起，成人受养人费用降至**€7,500**/人 |
| **签证有效期** | 永居卡 **5年**有效期，可续 |
| **居住要求** | **无最低居住要求** |
| **维持条件** | 投资须维持至少5年；之后须保留马耳他居住地址 |
| **入籍** | 需在马耳他实际居住（具体年数视欧盟/非欧盟身份而定，通常需满足居住要求+语言+背景审查），投资入籍另有MEIN政策（如下） |
| **福利** | 申根区自由通行；可在马耳他工作、学习、居住；非境内收入不征税；LGBTQ友好（全球排名第一） |
| **审批时间** | 约**12-18个月** |

> 📌 来源：getgoldenvisa.com (2026)、visaconnect.com (2026)、malta-citizenship.eu (2026)、PwC Malta (2025)

## 5. 爱尔兰 Ireland

### 5.2 关键技能就业许可（Critical Skills Employment Permit / CSEP）

| 维度 | 详情 |
|------|------|
| **适用人群** | 获得爱尔兰雇主工作offer、职位在关键技能职业清单上或在非不可申请清单上的高薪职位 |
| **薪资门槛（2026年3月起）** | |
| └ **关键技能清单职位** | ≥**€40,904/年**（2026年3月调整） |

## 7. 申根区短期签证规则

### 7.1 90/180天规则（旅游签/短期申根签证）

| 维度 | 详情 |
|------|------|
| **核心规则** | 非EU/EEA公民在**任意180天期间**内，在申根区停留**不得超过90天** |
| **计算方式** | 以任何一天为基准，向前回溯180天，计算期间内累计的申根区停留天数 |
| **适用对象** | 持有申根短期签证（C类）或享有免签入境权利的非EU公民 |
| **签证有效期** | 通常6个月至5年不等（由使领馆视情况签发） |
| **单次最长停留** | **90天/次** |
| **工作限制** | 不得在申根区工作或从事有偿活动 |
| **ETIAS** | 预计2026年下半年启动（针对免签国公民的电子旅行授权，€7，有效期3年） |
| **违规后果** | 超期停留可能导致罚款、驱逐出境、纳入SIS警报系统、未来签证被拒 |

> 📌 来源：欧盟委员会内政总司 (home-affairs.ec.europa.eu)、AXA Schengen (2026)、法国签证官网 (france-visas.gouv.fr)

### 8.5 欧洲综合评估输出原则

欧洲综合问卷不只判断“能不能去欧洲”，必须输出：
1. 可做国家；
2. 可做项目；
3. 推荐优先级；
4. 为什么不推荐其他国家/项目；
5. 是否能工作、是否低居住、是否可转永居/入籍、是否适合客户目标。

优先匹配逻辑：
- 高净值且低居住要求：葡萄牙D9/ARI、希腊黄金签证、马耳他MPRP；
- 被动收入/退休：葡萄牙D7、西班牙NLV、法国Visitor；
- 远程收入：葡萄牙D8、西班牙DNV、希腊DNV；
- 创业：葡萄牙D2、荷兰Startup/Self-employed、法国Talent Passport Founder/Innovative、德国Self-employed；
- 高薪雇主：荷兰HSM/EU Blue Card、法国Talent Passport/EU Blue Card、德国EU Blue Card/Skilled Worker；
- 派遣：荷兰ICT、法国Employee on Assignment/ICT、德国ICT；
- 艺术/科研：法国Talent Passport Artist/Researcher、德国Freelancer/Researcher、荷兰Orientation/Research相关路径；
- 求职：德国Opportunity Card、荷兰Orientation Year（特定毕业生）。
