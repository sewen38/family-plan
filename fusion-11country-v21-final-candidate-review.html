一、新加坡 EP/PIC｜私人投资公司 + Employment Pass 路径

第1章 客户家庭基本信息

本模块适用于希望通过新加坡真实经营主体或私人投资公司（PIC / Private Investment Company）建立区域总部、投资管理或海外业务承接职能，并由该新加坡公司为核心申请人申请 Employment Pass（EP）的家庭。PIC 不是新加坡政府独立移民项目，执行上应拆分为：新加坡公司设立、真实商业/投资活动建立、EP 岗位与薪资设计、COMPASS 评分、EP 获批后家庭准证及后续 PR 路径。

第2章 核心策略

核心策略是“先做真实新加坡经营节点，再申请 EP”，而不是把 PIC 包装为单纯买身份。新加坡 EP 审核重点为合格雇佣、岗位真实性、薪资水平、候选人能力、公司业务实质与 COMPASS 评分。客户应建立一间具有真实董事决策、办公地址、银行账户、会计秘书、商业合同或投资活动的新加坡公司，并由该公司聘任主申请人为区域负责人、投资总监、业务发展总监或管理岗位。

第5章 投资使用建议

第6章 财富架构搭建方案

新加坡节点的财富架构价值在于“真实区域功能 + 透明低税环境 + 家族资产管理”。企业所得税名义税率 17%，但实际税负取决于利润性质、豁免、激励、成本扣除与实质。新加坡通常无股息预提税，但中国向新加坡支付股息、利息、特许权使用费或服务费时需审查预提税、DTA、受益所有人和转让定价。

第7章 税务分析

新加坡 EP/PIC 可解决的税务问题，是将真实境外利润放在新加坡纳税，而不是转移中国利润。若新加坡公司无人员、无办公、无董事决策、无真实合同，仅向中国公司收咨询费或特许权使用费，容易被中国税务机关按受益所有人、转让定价、特别纳税调整或反避税规则否定。

第8章 资金跨境合规方案

资金跨境应分为个人资金、企业资金、境外存量资金三类处理。企业对外投资设立新加坡主体通常需评估 ODI；中国居民个人通过境外 SPV 持股或返程投资需评估 37号文；境外公司回投中国需按 FDI 处理。

第9章 身份路径规划

EP 路径应分三阶段推进：先公司与岗位，后 EP 获批与入境，再续签及 PR 评估。MOM 当前 EP 政府费为申请费 S$105/人，批准后签发费 S$225/人，如需 Multiple Journey Visa 为 S$30/人。薪资门槛需按年龄与行业复算，2025 版常见基准为一般行业 S$5,600/月起、金融服务 S$6,200/月起，并需通过 COMPASS（通常至少 40 分，豁免情形除外）。

第10章 教育规划

新加坡教育资源优质但竞争强。EP 家庭子女可根据年龄申请国际学校、私立学校或参加公立学校 AEIS/S-AEIS 竞争入学；公立学校名额不能作为确定性承诺。教育预算必须与身份时间轴同步，因为国际学校申请通常需要提前 6–12 个月准备。

第11章 福利居住国规划

EP 是工作准证，不等于永久福利身份。客户应按“商业保险 + 私人医疗 + 租房安顿 + 长期 PR 可能性”设计居住计划。EP 持有人在新加坡工作纳税，有助于建立长期记录；但公共住房、部分补贴和公民福利不适用于 EP 阶段。

第14章 财税执行策划案全文

14.1 财税目标

本财税执行策划案的目标，是在不虚构交易、不隐瞒受益所有人、不规避 CRS 的前提下，建立新加坡真实经营和投资管理节点，使境外业务利润、投资收益、家庭资产配置与身份安排形成一致链条。财税方案不以“低税”作为唯一目标，而以可解释、可审计、可开户、可续签、可传承为核心。

14.2 架构图

中国经营主体 -> 合同/成本加成/采购 -> 新加坡运营/PIC
境外客户 -> 合同/收款 -> 新加坡运营/PIC
新加坡运营/PIC -> 薪资/董事费 -> EP主申请人
新加坡运营/PIC -> 分红/投资 -> 家族信托/保险/投资账户
税务控制：转让定价 + DTA + CRS + CFC + 37号文/ODI

14.3 资金双循环架构

经营循环：境外客户收入 -> 新加坡公司 -> 采购/外包/运营成本 -> 新加坡纳税 -> 再投资
家庭循环：完税利润/薪资 -> 家庭账户 -> 教育/生活/保险/投资 -> 税务居民申报

二、香港 ASMTP｜输入内地人才计划

第1章 客户家庭基本信息

香港 ASMTP 适用于中国内地居民已获香港真实雇主聘用，且具备香港所需而本地不易获得的专业知识、技能或经验的申请人。该路径不是投资移民，不要求投资额；成败核心在雇主真实性、职位必要性、薪酬市场性、申请人不可替代性及家庭受养安排。

信息项执行填写口径文件

家庭成员配偶及受养子女关系证明、受养人表格、经济能力证明
目标工作居留、子女教育、香港税务与资产配置预算与时间确认

第2章 核心策略

核心策略是“雇主合规 + 职位合规 + 人才匹配”。申请材料必须证明香港公司有业务需要、岗位职责清晰、薪酬福利与本地市场相若，且申请人拥有相关学历、资格或经验。若客户同时有香港公司或跨境业务，应避免把 ASMTP 做成无真实雇佣的身份包装。

策略层执行动作成功标准
雇主策略完整解释公司业务、财务、招聘与岗位必要性入境处认可岗位真实
人才策略学历、资格、经验与岗位一一对应证明本地不易获得
家庭策略同步准备受养人材料家属随行居留
财税策略利用香港来源地税制与金融账户，但保持实质不做空壳导管
长期策略续签、通常居住、7年永居评估保持雇佣与居住记录

第3章 合规清理详细方案

ASMTP 的材料不能只提交雇佣合同，应同时提交雇主为什么需要该申请人、为什么香港本地市场难以替代、薪酬为什么合理、申请人为什么能胜任。

职位真实性职责、汇报线、招聘记录、市场薪酬职位说明与招聘说明
申请人匹配学历、工作证明、专业成果人才匹配说明
家属依附婚姻/出生、公证翻译、经济能力受养人包
税务与薪俸薪酬支付、MPF/税务安排薪俸税规划

第4章 境外资金归集与投资架构

香港 ASMTP 本身不要求投资，但香港可作为大湾区经营、控股、融资、收款与资产配置节点。客户应区分个人工资、公司利润、投资收益和中国业务收入，避免香港个人账户代收中国公司款项。

第7章 税务分析

香港利得税采用来源地原则，非香港来源利润是否征税需个案判断并保留证据。薪俸税则与在港受雇及工作安排相关。香港无一般股息预提税、无资本利得税，但频繁交易性利润可能被视为应税经营利润。内地向香港支付股息、利息、特许权使用费时，应按内地与香港税收安排及受益所有人规则审查。

香港资金安排必须将个人、公司、家庭资产分账管理。中国公司设香港子公司需评估 ODI；香港公司回投中国需 FDI；中国居民通过境外 SPV 持股需评估 37号文。ASMTP 个人赴港工作不自动解决中国资产跨境问题。

第11章 福利居住国规划

ASMTP 是工作居留安排，受养人身份依附主申请人。香港医疗、住房、教育和长期公共福利与居民身份、通常居住年期及具体资格有关。客户应以商业保险、私宅租赁、学校申请和现金流准备作为初期保障。

第2章 核心策略

EB-1A 的成功关键不是投资额，而是证据质量与美国移民局两步法审查：先满足法规列明的证据类别，再通过 final merits determination 证明申请人已达到领域顶尖层级并将继续在美国从事相关工作。

执行策略分三条线同步推进：

  1. 移民证据线：以 8 CFR 204.5(h)(3) 十项标准为矩阵，建立“奖项、会员、媒体、评审、原创贡献、学术/商业成果、展览、关键职位、高薪、商业成功”证据包。
  2. 登陆税务线：在成为美国税务居民前完成全球资产梳理，重点处理 PFIC、CFC、FBAR、FATCA、Form 5471、Form 8621、Form 3520/3520-A 风险。
  3. 家庭传承线：美国遗产税和赠与税对绿卡/美国税务居民影响重大，高净值家庭必须先设计遗产税暴露、非美资产持有和受益人结构。

    第4章 境外资金归集与投资架构

    美国不是低税资金归集地。赴美前的资金归集应以“已完税、可解释、可申报”为原则，而非把资金临时转入美国账户。

    中国经营主体/个人资产
  -> 完税分红、股权估值、债权债务整理
  -> 香港/新加坡真实经营公司（如有）
  -> 非美基金/PFIC清理或替换
  -> 合规信托/保险/遗产规划
  -> 美国个人税务居民节点
  -> 全球所得申报 + FBAR/FATCA + CFC/PFIC表格

    执行要点：

    • 登陆前确认哪些资产继续个人直接持有，哪些资产进入合规信托/保险，哪些资产应出售或转换。
    • 境外公司不可由美国居民在美国境内实际管理全部决策，否则可能引发美国税务归属和申报复杂化。
    • 美国账户入金必须能解释来源：分红、薪酬、股权转让、房产出售、投资赎回、赠与或贷款均需合同和税务凭证。

    第6章 财富架构搭建方案（有效税务架构）

    美国 EB-1A 的有效税务架构原则：先规划，后登陆；先申报，后投资；先隔离遗产税，后配置美国资产。

    架构层建议做法美国税务风险
    中国经营主体保留真实经营、利润和纳税记录；赴美前确认是否分红或股权重组分红、股权出售、CFC/GILTI、Subpart F
    香港/新加坡公司只有真实人员、合同、客户、董事决策时保留Form 5471、CFC、GILTI、实际管理地
    非美国基金赴美前识别并清理 PFIC；必要时转为美国合规产品Form 8621、超额分配惩罚税、利息 charge
    非美信托成为美国税务居民前设计；明确 grantor / non-grantor 后果Form 3520/3520-A、分配 throwback tax
    美国资产控制美国 situs assets 暴露，配合遗产税规划美国遗产税、赠与税、州税

    不可做事项：登陆后临时把资产转入离岸信托；隐瞒境外账户；用亲属代持境外公司；持有大量 PFIC 不申报；把非美国保险当作免税壳。

    第7章 税务分析

    美国税务居民包括绿卡持有人和满足实质居留测试者。成为美国税务居民后，工资、分红、利息、租金、资本利得、海外公司归属收入、信托分配等均需按美国规则申报。

    重点规则：

    • 全球征税：美国税务居民按全球所得纳税，外国税收抵免只能减少双重征税，不能消灭申报义务。
    • CFC / GILTI / Subpart F：美国股东控制境外公司可能对未分配利润提前纳税。
    • PFIC：非美国基金、部分保险投资产品可能触发高惩罚性税制和 Form 8621。
    • FBAR / FATCA：海外金融账户与金融资产达到门槛需申报；漏报罚款可高于税款。
    • 遗产税：绿卡/美国 domicile 规划不当会将全球资产纳入美国遗产税风险。

    第14章 财税执行策划案全文

    14.1 财税目标

    本项目财税目标不是降低美国税率，而是降低“登陆后无法补救”的合规成本。客户必须在美国税务居民开始日前完成全球资产穿透清单，按“个人资产、公司股权、金融账户、基金、信托、保险、不动产、债权债务、知识产权”分类。

    14.2 迁入前资产盘点
    资产类别必查项目输出文件
    境内公司股权持股比例、未分配利润、历史分红、估值股权结构图、估值报告、完税记录
    境外公司控制权、董事、利润、银行账户、实质CFC/GILTI 初判表
    金融产品基金、ETF、保险投资账户PFIC 清单、Form 8621 风险表
    信托/保险设立人、受托人、受益人、撤销权3520/3520-A 适用意见
    银行证券账户最高余额、共同持有人、受益人FBAR/8938 清单
    不动产所在地、买入成本、贷款、租金租金与资本利得台账
    14.3 资金双循环架构
    经营循环：中国/香港/新加坡真实经营收入 -> 当地完税 -> 合规分红/工资/服务费 -> 美国申报抵免
家庭循环：已完税资产 -> 合规账户/信托/保险 -> 教育/医疗/生活支出 -> 年度税务披露
禁止循环：未完税利润 -> 个人账户 -> 美国消费；空壳离岸公司 -> 隐匿账户 -> 不申报
    14.4 PFIC 处置

    美国税务居民持有非美国共同基金、ETF、部分私募基金、部分投资型保险，可能构成 PFIC。执行步骤：

    1. 登陆前列出所有非美国基金及保险投资底层。
    2. 判断是否出售、转换为美国合规产品、QEF election 或 mark-to-market。
    3. 未处理资产不得在首个美国税年后才被动发现。
      4. 14.5 CFC / GILTI / Subpart F 处置

      客户及家庭成员需按美国 attribution 规则判断是否为美国股东，境外公司是否 CFC。若构成 CFC，需评估 Form 5471、GILTI、Subpart F、分红和外国税收抵免。境外公司董事会、管理地、合同签署和银行授权必须真实，不可由美国住所远程全权管理。

      14.6 FBAR / FATCA 执行

      所有境外金融账户按最高余额和持有人穿透归集。共同账户、公司账户、签字权账户、信托账户、证券账户、基金账户均需判断。FBAR 与 Form 8938 是不同申报体系，不能相互替代。

      14.7 遗产税与赠与税

      登陆前应明确美国 domicile 风险、美国 situs assets、非美国资产持有结构、配偶和子女身份。高净值家庭需专项设计 irrevocable trust、life insurance、pre-immigration gift、股权重组或遗产税保险。任何设计必须在美国税务居民开始日前完成并具有真实法律效力。

      14.8 中国合规衔接

      中国居民个人对外投资、境外 SPV、返程投资、公司分红、股权转让仍需符合中国个税、企业所得税、外汇、37号文、ODI/FDI 规则。美国申报不替代中国完税，中国完税也不替代美国申报。

      财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。

      第14章 财税执行策划案全文

      日本财税策划以“非永住者窗口 + 国外送金管理 + 日本公司真实经营 + 海外公司实际管理地隔离 + 继承赠与税预案”为核心。

      14.1 非永住者/永住税务居民切换

      建立日本居住年限台账，记录入境、出境、住所、家庭、工作、公司经营、银行、学校。每年判断是否仍处于非永住者税务范围，以及国外来源所得是否因汇入日本而纳税。

      14.2 国外送金管理

      每笔汇入日本的资金必须标注来源：工资、分红、贷款、资本金、生活费、投资赎回。保留原始完税凭证、合同和银行流水，避免被认定为未申报国外所得。

      14.3 日本经营管理公司税务

      公司需按日本会计税务规则进行法人税、消费税、源泉所得税、社会保险、雇员工资处理。空壳公司或无真实办公室、无客户、无经营流水，会同时伤害签证和税务合规。

      14.4 海外公司与 CFC

      若客户长期为日本税务居民并控制香港/新加坡/离岸公司，应由日本税理士判断 CFC。境外公司必须保留当地董事、办公室、人员、合同和银行决策，不能由日本住所遥控全部经营。

      14.5 海外资产申报与继承赠与税

      高净值客户需每年判断国外财产调书、财产债务调书义务。继承赠与税需提前设计资产归属、受益人、保险、信托、赠与节奏，并与中国税务和外汇规则衔接。

      日本本地循环：日本公司 -> 日本客户/收入 -> 法人税/消费税/社保 -> 合规薪酬
海外资产循环：中国/香港/新加坡资产 -> 当地完税 -> 是否汇入日本 -> 日本申报判断
      财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。

      新西兰财税方案以“税务居民开始日 + 过渡居民窗口 + AIP资金来源 + FIF/CFC期后管理”为主线。

      14.1 税务居民开始日

      记录首次长期入境、住所、家庭、工作、学校、银行、租房/购房、离境天数，由新西兰税务师确认居民开始日。该日期决定过渡居民窗口和海外资产申报起点。

      14.2 过渡居民窗口

      若符合条件，应在窗口期内完成海外资产盘点、基金替换、公司分红策略、信托审查、未来现金流规划。窗口不是永久免税，不能作为隐瞒资产理由。

      14.3 FIF 管理

      海外股票、基金、ETF、部分非新西兰投资实体需判断 FIF。窗口结束后选择适用计算方法并保留年度报表。迁入前应减少复杂非透明基金和多层产品。

      14.4 CFC 管理

      如客户控制海外公司，需判断 CFC。真实经营公司应保留当地人员、办公室、合同、董事会和税务记录；被动投资公司需重点审查归属收入。

      14.5 AIP 资金来源

      AIP 投资款必须与移民局 KYC、银行 AML 和税务申报一致。资金来源可为经营分红、股权出售、房产出售、投资收益、继承赠与等，但均需形成闭环证据。

      14.6 海外信托

      新西兰对 settlor、trustee、beneficiary 和分配规则敏感。迁入前必须确认信托是否需要重组、是否会造成新西兰披露和纳税义务。

      过渡居民窗口：海外被动资产 -> 合法缓冲期 -> 重组/申报准备
窗口结束后：海外基金(FIF) + 海外公司(CFC) + 信托披露 -> 年度税务申报
AIP资金：完税来源 -> 银行KYC -> 合格投资 -> 期限维护

              第1章 项目定位与适配结论

              土耳其房产入籍适合希望通过可见实物资产完成第二公民身份配置，并接受海外房产交易、持有、出租、税费、维护和退出管理的家庭。该路径核心是购买符合入籍条件的不动产，满足最低投资额与三年禁售要求，并取得主管部门合格证明。

              维度结论
              身份属性土耳其公民身份，可作为旅行、资产配置和 E-2 潜在跳板身份
              投资属性不动产投资，通常最低 US$400,000 等值并登记 3 年不得出售
              税务属性房产所在地土耳其对租金/出售收益有优先征税权；护照不等于税务居民
              适合人群接受海外不动产、希望资产可见、愿意做产权尽调和税务申报的家庭

              第2章 申请条件与法律边界

              3. 土耳其护照不自动改变中国、美国、澳洲等税务居民身份。

                  第14章 财税执行策划案全文

                  本项目财税执行以房产所在地税、居民国税、资金来源和退出税四个维度展开。

                  首先，购房前完成资金来源合法性审查。客户需提供从收入形成、纳税、资金沉淀到跨境支付的完整链条。境内企业主应区分个人资金和公司资金，不能以公司未分配利润直接支付个人房产。境内公司对外投资房产与个人购房路径不同，应分别评估 ODI、外汇和税务后果。

                  其次，交易阶段要保证税务真实性。合同价格、评估价格、银行付款、DAB/外汇证明、产权登记金额应相互匹配。不得为降低税费而低报价格，也不得使用第三方现金补差。所有交易税费、中介费、律师费、评估费、维修基金应保留发票或收据。

                  第三，持有阶段建立租金税务制度。如出租，应签署租赁合同，租金进入可解释账户，按土耳其规则申报。客户仍为中国或其他国家税务居民时，应评估境外租金在居民国的申报和抵免。若房产空置，也应保留物业费、保险、维护记录，避免未来出售时成本无法证明。

                  第四，出售和退出阶段提前规划。三年禁售期届满并不意味着出售无税。出售收益需计算购买成本、交易费用、持有成本、当地税费和居民国资本利得。若未来客户已迁入美国、澳洲、日本、新西兰等高税国家，应在迁入前评估房产成本基础和未来 CGT 风险。

                  第五，继承和家庭治理。个人直接持有房产应同步准备遗嘱、授权、婚姻财产约定和未成年子女安排。若使用公司或信托结构，需综合土耳其产权规则、居民国 CFC、CRS、受益所有人披露和年度维护成本。

                  项目三｜多米尼克 CBI

                  第1章 项目定位与适配结论

                  多米尼克 CBI 是典型护照备份项目，适合希望快速取得第二公民身份、提升通行便利和家庭应急选择，但不以真实迁居或税务居民转换为主要目标的客户。其关键边界是：CBI 公民身份不等于多米尼克税务居民，不等于免除中国、美国、澳洲或其他居民国税务义务。

                  维度结论
                  身份属性公民身份/护照备份
                  投资路径EDF 捐款或 approved real estate
                  税务属性护照≠税务居民；开户仍需真实 CRS/FATCA 自证
                  适合人群重视快速身份、旅行备份、家庭风险隔离的客户
                  不适合人群期待用护照隐藏资产、规避税务居民申报或绕开 KYC 的客户

                  第2章 申请条件与身份边界

                  多米尼克 CBI 必须通过授权代理递交。申请人须通过尽职调查、资金来源审查、背景审查，并完成规定捐款或房产投资。

                  身份边界：

                  1. 多米尼克护照不能单独作为税务居民证明。
                  2. 金融机构开户时会审查出生地、居住地、税务居民、资金来源和受益所有人。
                  3. 美国公民/绿卡/税务居民无论持有多米尼克护照，仍需美国全球申报。
                  4. 中国税务居民取得多米尼克护照后，境外所得和境外资产申报义务仍按中国法判断。

                      第14章 财税执行策划案全文

                      多米尼克 CBI 的财税策划重点是避免客户把“护照”误当作“税务居民”。

                      第一，申请前确认家庭成员真实税务居民。中国税务居民、美国税务居民、澳洲税务居民、日本税务居民或其他居民身份，并不会因取得多米尼克护照自动消失。客户应为每位家庭成员建立税务居民矩阵，列明居住天数、永久住所、经济利益、国籍、绿卡、TIN 和银行自证。

                      第二，资金来源与税务记录同步准备。CBI 支付款项虽然属于身份投资/捐款，但仍需证明资金合法形成。经营利润、工资、房产出售、投资收益、赠与继承均需上溯到纳税或合法取得环节。资金来源解释应与银行流水、税务记录、申请表职业信息一致。

                      第三，CRS/FATCA 口径必须一致。客户不能在 A 银行填中国税务居民、在 B 银行只填多米尼克，也不能用多米尼克护照隐藏美国身份。若客户有美国身份，必须建立 FBAR、FATCA、Form 8938、境外公司/信托申报清单。

                      第四，离岸结构只为真实目的服务。若客户因传承、投资隔离或商业目的设立离岸 SPV、信托或保险，应评估受益所有人披露、CFC、反避税、赠与税、遗产税和居民国申报。不得设空壳公司承接中国利润、虚构服务费或隐藏账户。

                      第五，未来迁居另行规划。若客户未来希望真正改变税务居民，应制定居住天数、住所、家庭搬迁、资产处置、原居民国退出、新居民国登记、DTA tie-breaker 的完整方案。多米尼克护照只是身份工具，不是税务居民切换方案本身。

                      项目四｜马耳他 MPRP

                      第1章 项目定位与适配结论

                      马耳他 MPRP 是欧盟永久居留备份项目，适合希望建立欧洲居住、教育、医疗、资产配置和家庭应急通道，但不立即申请欧盟公民身份的家庭。其关键价值在于欧盟居留、申根便利和潜在税务居民路径设计；关键边界是 MPRP 居留不等于自动成为马耳他税务居民，也不自动适用 remittance basis。

                      维度结论
                      身份属性马耳他永久居留，不是护照
                      投资属性房产租/买 + 政府贡献 + 慈善捐款 + 行政费 + 资产要求
                      税务属性居留≠税务居民；remittance basis 需满足居民但非 domicile 等条件并由税务师确认
                      适合人群需要欧洲备份、子女教育、申根出行、高合规资产环境的家庭
                      不适合人群希望低成本护照、规避 CRS/FATCA、无资金来源证明的客户

                      第2章 申请条件与居留边界

                      MPRP 通常要求申请人通过 licensed agent 递交，满足资产要求、合格房产、政府贡献、慈善捐款、健康保险、无犯罪和尽调要求。2025 更新口径下，常见资产要求为 €500,000 capital，其中至少 €150,000 为 financial assets；房产购买常见 €375,000 起，租赁常见 €14,000/年起；具体以 Residency Malta 最新法规和表格为准。

                      边界：

                      1. MPRP 是永久居留，不是马耳他国籍。
                      2. 居留卡不自动等于马耳他税务居民证明。
                      3. remittance basis 不是“所有海外收入免税”，需区分马耳他来源收入、海外收入汇入、海外资本利得、domicile 和最低税负等规则。
                      4. 欧盟银行 KYC、CRS、AML 审查严格，资金来源必须完整。

                        第4章 Remittance Basis 与税务居民说明

                        马耳他税务体系中，居留、ordinary residence、domicile 和 remittance basis 是不同概念。MPRP 只提供居留权；客户若不实际居住，通常不因持卡自动成为马耳他税务居民。若未来实际迁居并成为马耳他税务居民但非 domicile，可能涉及汇入制逻辑：马耳他来源收入和资本利得通常在马耳他纳税；境外收入在汇入马耳他时可能纳税；境外资本利得即使汇入的处理需由马耳他税务师确认。

                        执行警示：

                        1. 不得把 MPRP 卡片当作税务居民证明向银行提交。
                        2. 不得把海外公司分红随意汇入马耳他后主张免税。
                        3. 不得无实质设马耳他公司套用 DTA。
                        4. 每年应按实际居住、汇入、资产和账户变化更新税务判断。

                            第11章 教育规划

                            第14章 财税执行策划案全文

                            MPRP 的财税执行重点是把欧盟居留身份、马耳他税务居民、domicile、remittance basis 和原居民国义务分层管理。

                            第一，建立居留与税务居民双轨表。客户取得 MPRP 后，只说明拥有马耳他永久居留权；是否成为马耳他税务居民，要看实际居住天数、住所、生活中心、家庭搬迁、税务登记和当地规则。客户仍为中国税务居民时，境外投资收益、海外公司权益和资金汇入仍应按中国规则判断；客户有美国身份时，美国全球申报不受 MPRP 影响。

                            第二，明确 remittance basis 的适用前提。只有在客户成为马耳他税务居民且非 domicile，并满足当地税务规则时，才讨论汇入制。海外收入、海外资本利得、马耳他来源收入和汇入资金的税务处理不同。客户应按年度制定汇入预算：生活费、房租、学费、保险、投资款分别从哪些账户汇入，汇入前是否已在来源地纳税，是否保留银行流水和收入性质证明。

                            第三，资金来源和资产要求要可审计。MPRP 对资产和资金来源审查严格，客户需证明 €500,000 capital 及其中 €150,000 financial assets 的所有权、形成原因和可持续性。企业主应提供公司审计、分红、完税；投资人提供券商记录；房产资产提供产权和估值；赠与继承提供上游来源。

                            第四，避免无实质马耳他公司。若客户没有欧盟业务、董事、办公、雇员和商业合同，不建议设马耳他公司承接中国或亚洲利润。DTA 优惠、受益所有人和经济实质均会被审查。真实业务利润应留在中国、香港、新加坡或其他有实质的经营节点。

                            第五，欧盟银行和 CRS 管理。马耳他和欧盟金融机构会执行 AML、CRS、FATCA 和受益所有人披露。客户所有账户的税务居民自证必须与实际一致。若未来税务居民发生变化，应主动更新银行、保险、券商、信托和基金文件。

                            第六，退出与迁居规划。如果客户未来从 MPRP 备份转为实际迁居马耳他，应在迁居前完成海外公司、信托、保险、投资账户、房产和家庭现金流规划，避免成为税务居民后再处理历史资产导致重复征税或申报风险。

                            固定注释：财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。
