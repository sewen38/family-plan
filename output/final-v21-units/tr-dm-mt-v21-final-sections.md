# 项目一｜土耳其基金入籍 + 美国 E-2 跳板说明

## 第1章 项目定位与适配结论

土耳其基金入籍适合希望以金融资产方式完成第二公民身份配置、避免直接持有海外房产管理负担，并希望未来保留美国 E-2 非移民投资签证跳板可能性的家庭。该路径核心不是税务居民迁移，而是通过合规认购符合条件的土耳其投资基金份额，满足土耳其 exceptional citizenship 投资入籍要求。

执行判断：

| 维度 | 结论 |
|---|---|
| 身份属性 | 土耳其公民身份；可作为美国 E-2 条约国国籍基础之一 |
| 投资属性 | 金融资产配置，通常要求至少 US$500,000 等值基金份额并持有不少于 3 年 |
| 税务属性 | 取得土耳其护照不自动成为土耳其税务居民；仍按实际税务居民国申报全球/来源所得 |
| 适合人群 | 有境外资金或可解释出境资金、接受基金锁定、重视美国 E-2 后续选项的家庭 |
| 不适合人群 | 资金来源链条不完整、希望用护照规避 CRS/FATCA、期待“拿护照免税”的客户 |

## 第2章 申请条件与身份边界

主申请人应满足成年、无重大犯罪记录、通过土耳其及金融机构 KYC/AML 审查、完成合格投资并取得主管部门合格证明。配偶及未成年子女通常可随同申请，具体年龄、依附关系、文件要求以递交日土耳其律师核验为准。

身份边界必须写入客户确认书：

1. 土耳其公民身份是国籍身份，不等于中国税务居民退出、美国税务居民退出或其他国家税务居民切换。
2. 使用土耳其护照申请美国 E-2，只说明申请人具备条约国国籍基础；E-2 仍需真实、实质、活跃、可运营的美国商业投资。
3. 中国籍客户涉及中国出入境、国籍、户籍、税务居民和资产申报时，应按中国法律独立判断，不能以第三国护照替代中国法下身份义务。
4. 银行开户、基金认购、E-2 递交均会询问出生地、现有国籍、税务居民、资金来源和受益所有人。

## 第3章 投资路径与产品筛选

基金路径通常为认购符合土耳其入籍规则的投资基金份额，包括但不限于房地产投资基金、风险投资基金或其他经监管认可的基金类别。执行时不得只看收益率，应按身份合规、监管、赎回、费用、底层资产、基金管理人历史、合格证明出具能力做筛选。

基金筛选表：

| 审查项 | 执行标准 | 风险动作 |
|---|---|---|
| 合格性 | 可用于土耳其入籍并能出具合格投资证明 | 由土耳其律师书面确认 |
| 监管状态 | 基金管理人、托管、审计、信息披露完整 | 获取监管登记和基金文件 |
| 锁定期 | 满足至少 3 年持有承诺 | 确认提前赎回是否影响国籍 |
| 费用 | 管理费、认购费、赎回费、业绩报酬 | 写入费用总表 |
| 收益风险 | 底层资产、流动性、估值机制 | 不得承诺保本收益 |
| KYC | 可接受客户国籍、资金来源和税务居民 | 预审资金来源包 |

## 第4章 美国 E-2 跳板说明

土耳其为美国 E-2 Treaty Investor 签证条约国之一。客户取得土耳其国籍后，可在符合美国 E-2 条件时，以土耳其公民身份申请 E-2。E-2 不是绿卡，不直接通向永久居民；其本质是非移民投资经营签证。

E-2 执行要件：

| 要件 | 客户需满足的实际内容 |
|---|---|
| 条约国国籍 | 递交时持有土耳其国籍，且申请人及企业符合条约国所有权要求 |
| 实质投资 | 对美国真实企业投入有风险资金，金额需与业务类型相匹配 |
| 活跃经营 | 企业不能是被动投资、空壳账户或单纯持有证券/房产 |
| 控制权 | 申请人需拥有或控制企业并有经营管理职责 |
| 非边际企业 | 企业应具备创造收入和就业潜力，不只是维持申请人最低生活 |
| 资金来源 | 美国签证阶段再次审查资金合法来源、路径和商业计划 |

E-2 与税务：进入美国长期居住、满足实质居留测试或取得其他美国身份后，可能成为美国税务居民。美国税务居民需面对全球所得、FATCA、FBAR、PFIC、CFC、Form 8938、Form 5471/8621 等申报。土耳其护照不能降低美国税务居民申报义务。

## 第6章 有效税务架构

土耳其基金项目的有效税务架构不是“用土耳其护照避税”，而是把身份、金融资产、投资收益、未来美国 E-2 经营和原税务居民国申报拆开处理。

结构建议：

```text
已完税家庭资金 / 境外存量资金
  ↓ 资金来源证明 + 合规出境说明
个人直接持有或合规 SPV（需 CFC/37号文/ODI 审查）
  ↓ 基金 KYC / CRS 自证 / FATCA 自证
土耳其合格投资基金份额（3年锁定）
  ↓ 收益、赎回、估值、税务凭证
境外投资账户 / 家庭信托保险 / E-2 美国经营资金池
  ↓
真实税务居民国申报：中国/美国/澳洲/新加坡/香港等
```

税务执行要点：

1. **税务居民判断**：取得土耳其护照或认购基金不自动成为土耳其税务居民；长期居住、永久住所、生活中心才会触发进一步判断。
2. **基金收益申报**：基金分红、赎回、资本利得应在税务居民国按投资收益申报；如有土耳其来源税，可评估 DTA 和外国税收抵免。
3. **SPV 持有审查**：若用香港、新加坡、BVI 等 SPV 持有基金，需判断中国 CFC、37号文、ODI、未来美国 PFIC/CFC 风险。
4. **CRS/FATCA**：开户和基金认购时须填写真实税务居民；美国人或美国税务居民不得以土耳其身份规避 FATCA、FBAR。
5. **E-2 前置税务**：若未来赴美长期居住，应在成为美国税务居民前清理 PFIC 风险、境外公司控制权、信托保险结构、成本基础。
6. **中国资金合规**：个人购汇、境外存量资金、分红、股权转让、薪酬、赠与、继承、ODI/37号文路径不得混用或虚构。

## 第7章 资金来源、KYC 与跨境资金执行

资金来源包应在投资前完成，而不是等银行或基金补件后临时拼接。

资金来源材料：

| 来源类型 | 证明文件 |
|---|---|
| 工资奖金 | 劳动合同、个税记录、工资流水、奖金说明 |
| 公司分红 | 公司章程、审计报表、完税凭证、分红决议、银行流水 |
| 股权转让 | 转让协议、估值报告、完税凭证、收款流水 |
| 房产出售 | 产权证、买卖合同、完税证明、收款流水 |
| 投资收益 | 券商对账单、交易记录、纳税记录 |
| 赠与/继承 | 赠与协议、亲属关系、遗嘱/继承文件、资金来源上溯 |
| 境外存量资金 | 历史开户文件、形成记录、税务居民声明、账户流水 |

跨境资金路径：

1. 境内个人资金：按中国外汇和银行合规要求处理，不使用地下钱庄、虚假贸易或第三方代付。
2. 境内企业资金：若以企业对外投资，应评估 ODI、董事会决议、商务/发改/外汇流程。
3. 境外存量资金：提供形成时间、来源、纳税、账户流水和受益所有人说明。
4. 资金进入基金账户后，保留汇款凭证、认购确认、基金份额确认和合格投资证明。

## 第11章 教育规划

土耳其护照本身不等于美国、英国、欧盟本地生学费身份。客户应把教育规划分为三层：

1. **土耳其本地教育**：如家庭短住伊斯坦布尔，可选择国际学校，年度预算通常 US$8,000-30,000+/孩。
2. **美国 E-2 陪同教育**：E-2 配偶和子女身份取决于主申请 E-2；子女可在美国就读 K-12，但大学学费是否州内价取决于州居住规则，不由 E-2 自动保证。
3. **全球申请身份辅助**：土耳其护照可作为旅行便利，但国际学校、大学录取、奖学金、学费身份仍看居住、签证、税务和学校规则。

执行动作：在 E-2 前 12 个月确定城市、学校、预算、监护、保险；避免先办身份后发现教育路径不匹配。

土耳其护照不自动带来其他国家福利。客户如不实际居住土耳其，不应预期使用当地公共福利。若未来赴美 E-2，美国医疗和福利以商业保险、个人负担和州规则为主。

## 第14章 财税执行策划案全文

本项目财税策划以“身份取得、资产持有、税务居民、美国 E-2”四条线并行，不把护照作为税务结论。

第一步，建立家庭税务居民地图。列明夫妻、子女当前国籍、绿卡、长期签证、每年居住天数、永久住所、主要经济利益、公司董事任职、银行账户税务居民自证。对中国税务居民客户，确认中国个人所得税居民身份、境外所得申报义务、外汇出境路径和历史境外账户。

第二步，建立资金来源审计包。基金认购资金必须能从最终付款账户上溯至收入形成环节。经营所得需有公司财报、完税、分红决议；股权转让需有协议、估值、完税；房产出售需有合同、产权、税票；赠与继承需有赠与人资金来源。任何现金、第三方代付、地下钱庄、虚假贸易均列为禁止路径。

第三步，选择持有方式。个人直接持有最简单，税务穿透清晰，适合以身份为主、投资规模不复杂的家庭。SPV 持有只有在继承、资产隔离、家族治理或后续投资组合确有必要时使用，并必须评估中国 CFC、37号文、ODI、CRS 控制人披露，以及未来美国 E-2/美国税务居民下 PFIC、CFC、Form 5471/8621 风险。

第四步，配置 CRS/FATCA 一致性。所有银行、基金、券商、保险、信托表格中的税务居民、TIN、控制人、受益所有人必须一致。若家庭成员有美国公民、绿卡、实质居留或美国出生地问题，必须先由美国 CPA/律师做 FATCA、FBAR、PFIC 评估。

第五步，规划 E-2 前置架构。取得土耳其护照后，不建议立即用未分类境外资金投入美国企业。应先完成美国投资主体、股权比例、商业计划、资金来源、税务居民触发时间、企业会计和 payroll 设计。E-2 企业应是真实经营，不得只是购买证券、被动房产或空壳 LLC。

第六步，年度税务维护。基金收益、估值变化、赎回、E-2 企业利润、美国工资、境外账户利息股息均按当年真实税务居民国申报。若家庭未来迁入美国、澳洲、日本、新西兰等高税国家，应在迁入前完成资产重组，而不是入境后补救。

固定注释：财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。

# 项目二｜土耳其房产入籍

## 第1章 项目定位与适配结论

土耳其房产入籍适合希望通过可见实物资产完成第二公民身份配置，并接受海外房产交易、持有、出租、税费、维护和退出管理的家庭。该路径核心是购买符合入籍条件的不动产，满足最低投资额与三年禁售要求，并取得主管部门合格证明。

| 维度 | 结论 |
|---|---|
| 身份属性 | 土耳其公民身份，可作为旅行、资产配置和 E-2 潜在跳板身份 |
| 投资属性 | 不动产投资，通常最低 US$400,000 等值并登记 3 年不得出售 |
| 税务属性 | 房产所在地土耳其对租金/出售收益有优先征税权；护照不等于税务居民 |
| 适合人群 | 接受海外不动产、希望资产可见、愿意做产权尽调和税务申报的家庭 |
| 不适合人群 | 不愿管理房产、资金来源不透明、期望低报房价或现金交易的客户 |

## 第2章 申请条件与法律边界

申请人需购买符合条件的不动产，达到规定最低价值，付款路径符合银行和外汇兑换要求，取得估值报告、产权登记、三年禁售注记及合格证明。配偶和未成年子女通常可随同申请。

边界：

1. 购房入籍不是房地产收益承诺，房价、汇率、租金、流动性均有风险。
2. 低报交易价、现金付款、第三方代付、虚假评估可能导致入籍失败和税务风险。
3. 土耳其护照不自动改变中国、美国、澳洲等税务居民身份。
4. 房产持有期间产生租金或出售收益，应按土耳其及税务居民国规则申报。

## 第3章 房产筛选与交易尽调

房产筛选应先身份合规，再投资价值。禁止只因佣金高或开发商承诺回购而忽视产权和估值。

| 尽调项 | 标准 | 文件 |
|---|---|---|
| 入籍资格 | 可用于 CBI，非受限制交易 | 律师确认函 |
| 产权 | 无抵押、查封、争议、重复销售 | Tapu 查册 |
| 估值 | 合规评估足额覆盖 US$400,000 | 官方估值报告 |
| 付款 | 银行转账、DAB/外汇证明完整 | 银行凭证 |
| 卖方 | 开发商/个人卖方资格合规 | 公司/个人 KYC |
| 税费 | 契税、登记、中介、维护明确 | 费用表 |
| 退出 | 三年禁售后可转售机制 | 市场分析 |

## 第4章 投资结构与持有方式

多数家庭以个人名义直接持有，路径清晰、身份合规简单。若考虑公司或 SPV 持有，必须由土耳其律师确认是否影响入籍资格，并评估公司税、会计、CFC、CRS 和继承成本。

推荐结构：

```text
已完税家庭资金
  ↓ 合规出境 / 境外存量资金证明
个人名义土耳其银行账户或律师监管账户
  ↓ 银行付款 + DAB/外汇证明
土耳其合格房产
  ↓ Tapu登记 + 3年禁售注记
入籍申请 + 后续租金/维护/出售税务申报
```

## 第6章 有效税务架构

土耳其房产的税务架构应围绕“不动产所在地优先征税 + 居民国申报抵免 + 继承/退出安排”设计。

架构图：

```text
已完税资金 → 土耳其购房账户 → 土耳其房产
       ↓                ↓
资金来源/KYC       租金收入 → 土耳其申报纳税
       ↓                ↓
CRS/FATCA自证     出售收益 → 土耳其税 + 居民国申报/抵免
       ↓
继承/赠与/婚姻财产安排
```

执行要点：

1. **房产税务**：租金、资本利得、房产相关税费应按土耳其规则处理；税率、免税额和申报规则以当地会计师当年意见为准。
2. **居民国申报**：中国税务居民、美国税务居民或其他国家税务居民，应按居民国规则申报海外租金和出售收益。
3. **DTA 与抵免**：不动产所得通常由所在地优先征税，居民国可能提供外国税收抵免，但需凭证完整。
4. **CRS/FATCA**：房产本身不是金融账户，但土耳其银行账户、公司/SPV、信托保险会产生申报或交换。
5. **继承风险**：个人直接持有需提前规划婚姻、继承、未成年子女、遗嘱和授权。
6. **禁止事项**：不得低报房价、虚假租赁、漏报租金、用现金或第三方代付破坏资金链。

## 第7章 资金来源、KYC 与跨境资金

## 第11章 教育规划

## 第14章 财税执行策划案全文

本项目财税执行以房产所在地税、居民国税、资金来源和退出税四个维度展开。

首先，购房前完成资金来源合法性审查。客户需提供从收入形成、纳税、资金沉淀到跨境支付的完整链条。境内企业主应区分个人资金和公司资金，不能以公司未分配利润直接支付个人房产。境内公司对外投资房产与个人购房路径不同，应分别评估 ODI、外汇和税务后果。

其次，交易阶段要保证税务真实性。合同价格、评估价格、银行付款、DAB/外汇证明、产权登记金额应相互匹配。不得为降低税费而低报价格，也不得使用第三方现金补差。所有交易税费、中介费、律师费、评估费、维修基金应保留发票或收据。

第三，持有阶段建立租金税务制度。如出租，应签署租赁合同，租金进入可解释账户，按土耳其规则申报。客户仍为中国或其他国家税务居民时，应评估境外租金在居民国的申报和抵免。若房产空置，也应保留物业费、保险、维护记录，避免未来出售时成本无法证明。

第四，出售和退出阶段提前规划。三年禁售期届满并不意味着出售无税。出售收益需计算购买成本、交易费用、持有成本、当地税费和居民国资本利得。若未来客户已迁入美国、澳洲、日本、新西兰等高税国家，应在迁入前评估房产成本基础和未来 CGT 风险。

第五，继承和家庭治理。个人直接持有房产应同步准备遗嘱、授权、婚姻财产约定和未成年子女安排。若使用公司或信托结构，需综合土耳其产权规则、居民国 CFC、CRS、受益所有人披露和年度维护成本。

固定注释：财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。

# 项目三｜多米尼克 CBI

## 第1章 项目定位与适配结论

多米尼克 CBI 是典型护照备份项目，适合希望快速取得第二公民身份、提升通行便利和家庭应急选择，但不以真实迁居或税务居民转换为主要目标的客户。其关键边界是：CBI 公民身份不等于多米尼克税务居民，不等于免除中国、美国、澳洲或其他居民国税务义务。

| 维度 | 结论 |
|---|---|
| 身份属性 | 公民身份/护照备份 |
| 投资路径 | EDF 捐款或 approved real estate |
| 税务属性 | 护照≠税务居民；开户仍需真实 CRS/FATCA 自证 |
| 适合人群 | 重视快速身份、旅行备份、家庭风险隔离的客户 |
| 不适合人群 | 期待用护照隐藏资产、规避税务居民申报或绕开 KYC 的客户 |

## 第2章 申请条件与身份边界

多米尼克 CBI 必须通过授权代理递交。申请人须通过尽职调查、资金来源审查、背景审查，并完成规定捐款或房产投资。

身份边界：

1. 多米尼克护照不能单独作为税务居民证明。
2. 金融机构开户时会审查出生地、居住地、税务居民、资金来源和受益所有人。
3. 美国公民/绿卡/税务居民无论持有多米尼克护照，仍需美国全球申报。
4. 中国税务居民取得多米尼克护照后，境外所得和境外资产申报义务仍按中国法判断。

## 第3章 投资路径选择

| 路径 | 金额 | 特点 | 适合客户 |
|---|---:|---|---|
| EDF 捐款 | 主申请 US$200,000 起 | 简单、无资产管理、不可回收 | 追求速度和确定性 |
| Approved Real Estate | US$200,000 起 + 政府费 | 有资产但流动性和退出受限 | 接受房产风险的客户 |

房产路径需特别关注：政府费、开发商资质、转售限制、租金承诺真实性、持有年限、退出市场深度。EDF 路径预算更清晰，但为一次性成本。

## 第4章 KYC/AML 与尽调重点

多米尼克 CBI 的核心不是文件数量，而是尽调通过率。以下事项将显著影响审批：

| 风险项 | 处理方式 |
|---|---|
| 政治公众人物/敏感行业 | 提前披露并准备增强尽调 |
| 资金多层流转 | 做资金路径图和审计说明 |
| 曾被拒签/拒入境 | 如实披露并解释 |
| 税务居民不一致 | 修正银行、税务和申请表口径 |
| 制裁/诉讼/负面新闻 | 提前做数据库检索和律师意见 |

## 第5章 办理流程与时间轴

| 阶段 | 时间 | 动作 |
|---|---:|---|
| 授权代理预审 | 1-2 周 | 家庭、国籍、资金、背景预筛 |
| 文件准备 | 3-8 周 | 公证认证、无犯罪、体检、银行推荐、资金来源 |
| 尽调递交 | 1-3 周 | 表格、KYC、费用支付、递交 CBIU |

| 证书护照 | 2-8 周 | 公民证书、护照签发 |

## 第6章 有效税务架构

多米尼克 CBI 的有效税务架构是“护照备份与税务居民分离”。

```text
原税务居民个人（中国/美国/澳洲等）
  ↓ 已完税资金 + 资金来源证明
多米尼克 CBI 费用/投资
  ↓
多米尼克公民身份（通行/备份）
  ↓ 不自动改变
真实税务居民申报 + CRS/FATCA 自证 + CFC/FBAR/37号文审查
```

执行要点：

1. **税务居民不变**：除非客户真实迁居并满足新税务居民规则，否则原居民国税务义务继续。
2. **护照开户边界**：可用多米尼克护照作为身份证明之一，但 CRS 税务居民必须填真实国家。
3. **离岸公司限制**：如设多米尼克或其他离岸公司，需有商业目的和受益所有人披露；不能用空壳隐藏利润。
4. **CFC 风险**：中国、美国、澳洲等居民控制低税公司，可能被反递延或归属征税。
5. **FATCA/FBAR**：美国身份客户持有任何境外账户、公司、信托均需专项申报。

## 第7章 资金来源与跨境资金

CBI 资金来源需覆盖捐款/投资本金、政府费、尽调费、代理费和家庭生活资产背景。

| 来源 | 文件 |

| 企业利润/分红 | 审计报表、完税、分红决议、银行流水 |
| 工资 | 雇佣合同、个税、工资流水 |
| 房产出售 | 买卖合同、产权、完税、收款流水 |
| 投资收益 | 交易记录、对账单、税务文件 |
| 赠与继承 | 关系证明、赠与/继承文件、上游资金来源 |

CBI 不应使用无法解释的现金、第三方代付、虚假合同或与申请人经济能力不匹配的资金。

## 第11章 教育规划

## 第14章 财税执行策划案全文

多米尼克 CBI 的财税策划重点是避免客户把“护照”误当作“税务居民”。

第一，申请前确认家庭成员真实税务居民。中国税务居民、美国税务居民、澳洲税务居民、日本税务居民或其他居民身份，并不会因取得多米尼克护照自动消失。客户应为每位家庭成员建立税务居民矩阵，列明居住天数、永久住所、经济利益、国籍、绿卡、TIN 和银行自证。

第二，资金来源与税务记录同步准备。CBI 支付款项虽然属于身份投资/捐款，但仍需证明资金合法形成。经营利润、工资、房产出售、投资收益、赠与继承均需上溯到纳税或合法取得环节。资金来源解释应与银行流水、税务记录、申请表职业信息一致。

第三，CRS/FATCA 口径必须一致。客户不能在 A 银行填中国税务居民、在 B 银行只填多米尼克，也不能用多米尼克护照隐藏美国身份。若客户有美国身份，必须建立 FBAR、FATCA、Form 8938、境外公司/信托申报清单。

第四，离岸结构只为真实目的服务。若客户因传承、投资隔离或商业目的设立离岸 SPV、信托或保险，应评估受益所有人披露、CFC、反避税、赠与税、遗产税和居民国申报。不得设空壳公司承接中国利润、虚构服务费或隐藏账户。

第五，未来迁居另行规划。若客户未来希望真正改变税务居民，应制定居住天数、住所、家庭搬迁、资产处置、原居民国退出、新居民国登记、DTA tie-breaker 的完整方案。多米尼克护照只是身份工具，不是税务居民切换方案本身。

固定注释：财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。

# 项目四｜马耳他 MPRP

## 第1章 项目定位与适配结论

马耳他 MPRP 是欧盟永久居留备份项目，适合希望建立欧洲居住、教育、医疗、资产配置和家庭应急通道，但不立即申请欧盟公民身份的家庭。其关键价值在于欧盟居留、申根便利和潜在税务居民路径设计；关键边界是 MPRP 居留不等于自动成为马耳他税务居民，也不自动适用 remittance basis。

| 维度 | 结论 |
|---|---|
| 身份属性 | 马耳他永久居留，不是护照 |
| 投资属性 | 房产租/买 + 政府贡献 + 慈善捐款 + 行政费 + 资产要求 |
| 税务属性 | 居留≠税务居民；remittance basis 需满足居民但非 domicile 等条件并由税务师确认 |
| 适合人群 | 需要欧洲备份、子女教育、申根出行、高合规资产环境的家庭 |
| 不适合人群 | 希望低成本护照、规避 CRS/FATCA、无资金来源证明的客户 |

## 第2章 申请条件与居留边界

MPRP 通常要求申请人通过 licensed agent 递交，满足资产要求、合格房产、政府贡献、慈善捐款、健康保险、无犯罪和尽调要求。2025 更新口径下，常见资产要求为 €500,000 capital，其中至少 €150,000 为 financial assets；房产购买常见 €375,000 起，租赁常见 €14,000/年起；具体以 Residency Malta 最新法规和表格为准。

边界：

1. MPRP 是永久居留，不是马耳他国籍。
2. 居留卡不自动等于马耳他税务居民证明。
3. remittance basis 不是“所有海外收入免税”，需区分马耳他来源收入、海外收入汇入、海外资本利得、domicile 和最低税负等规则。
4. 欧盟银行 KYC、CRS、AML 审查严格，资金来源必须完整。

## 第3章 路径结构与核心条件

| 模块 | 要求 | 执行动作 |
|---|---|---|
| Licensed agent | 需由许可代理递交 | 签署代理协议并完成预审 |
| 资产要求 | €500,000 capital，其中 €150,000 financial assets | 银行、券商、审计、资产证明 |
| 房产 | 购买或租赁合格住宅 | 租约/购房合同，区域和金额核验 |
| 政府贡献 | 常见 €37,000 | 以官方表格为准 |
| 行政/申请费 | 常见 €50,000 起 | 按家庭结构计算 |
| 慈善捐款 | €2,000 | 向合格组织支付 |
| 健康保险 | 覆盖马耳他/欧盟需求 | 购买并续保 |
| 尽调 | 背景、资金、声誉 | 提前做风险筛查 |

## 第4章 Remittance Basis 与税务居民说明

马耳他税务体系中，居留、ordinary residence、domicile 和 remittance basis 是不同概念。MPRP 只提供居留权；客户若不实际居住，通常不因持卡自动成为马耳他税务居民。若未来实际迁居并成为马耳他税务居民但非 domicile，可能涉及汇入制逻辑：马耳他来源收入和资本利得通常在马耳他纳税；境外收入在汇入马耳他时可能纳税；境外资本利得即使汇入的处理需由马耳他税务师确认。

执行警示：

1. 不得把 MPRP 卡片当作税务居民证明向银行提交。
2. 不得把海外公司分红随意汇入马耳他后主张免税。
3. 不得无实质设马耳他公司套用 DTA。
4. 每年应按实际居住、汇入、资产和账户变化更新税务判断。

## 第6章 有效税务架构

MPRP 税务架构应围绕“欧盟居留备份 + 原税务居民延续 + 未来马耳他税务居民/非 domicile 可能性 + 汇入管理”展开。

```text
原税务居民国资产池（中国/香港/新加坡/其他）
  ↓ 已完税资金 + KYC/AML
MPRP 费用/租购房/保险
  ↓
马耳他永久居留节点（不自动税务居民）
  ↓ 若未来实居
马耳他税务居民但非 domicile 评估
  ↓
海外收入汇入规划 / 马耳他来源收入申报 / CRS FATCA 披露
```

执行要点：

1. **居留与税务分离**：持有 MPRP 不等于税务居民；税务居民需看实际居住、住所、生活中心、登记和当地规则。
2. **Remittance basis**：仅在符合条件时考虑，且需年度现金流管理；海外收入是否汇入、以何账户汇入、用途和凭证都要记录。
3. **原居民国义务**：中国、美国、澳洲等原居民国税务义务不会因 MPRP 消失。
4. **欧盟 AML**：马耳他银行、保险、房产和代理均会审查资金来源、受益所有人、制裁和税务居民。
5. **公司架构**：马耳他公司只有在有欧盟业务、董事、办公、会计和实质时才考虑；否则不建议作为空壳 DTA 节点。

## 第11章 教育规划

MPRP 可为家庭建立欧洲教育备选，但不等于自动获得欧盟公民待遇或所有国家本地学费资格。马耳他本地有英语环境和国际学校，适合低龄子女欧洲过渡；若目标为英国、瑞士、法国、德国等教育体系，仍需按当地签证、居住和学校规则单独规划。

预算：马耳他国际学校可按 €8,000-25,000+/孩/年预留，另有注册费、校服、校车、课外活动和保险。若家庭不常住，仅将 MPRP 作为备份，教育预算可列为可选。

## 第14章 财税执行策划案全文

MPRP 的财税执行重点是把欧盟居留身份、马耳他税务居民、domicile、remittance basis 和原居民国义务分层管理。

第一，建立居留与税务居民双轨表。客户取得 MPRP 后，只说明拥有马耳他永久居留权；是否成为马耳他税务居民，要看实际居住天数、住所、生活中心、家庭搬迁、税务登记和当地规则。客户仍为中国税务居民时，境外投资收益、海外公司权益和资金汇入仍应按中国规则判断；客户有美国身份时，美国全球申报不受 MPRP 影响。

第二，明确 remittance basis 的适用前提。只有在客户成为马耳他税务居民且非 domicile，并满足当地税务规则时，才讨论汇入制。海外收入、海外资本利得、马耳他来源收入和汇入资金的税务处理不同。客户应按年度制定汇入预算：生活费、房租、学费、保险、投资款分别从哪些账户汇入，汇入前是否已在来源地纳税，是否保留银行流水和收入性质证明。

第三，资金来源和资产要求要可审计。MPRP 对资产和资金来源审查严格，客户需证明 €500,000 capital 及其中 €150,000 financial assets 的所有权、形成原因和可持续性。企业主应提供公司审计、分红、完税；投资人提供券商记录；房产资产提供产权和估值；赠与继承提供上游来源。

第四，避免无实质马耳他公司。若客户没有欧盟业务、董事、办公、雇员和商业合同，不建议设马耳他公司承接中国或亚洲利润。DTA 优惠、受益所有人和经济实质均会被审查。真实业务利润应留在中国、香港、新加坡或其他有实质的经营节点。

第五，欧盟银行和 CRS 管理。马耳他和欧盟金融机构会执行 AML、CRS、FATCA 和受益所有人披露。客户所有账户的税务居民自证必须与实际一致。若未来税务居民发生变化，应主动更新银行、保险、券商、信托和基金文件。

第六，退出与迁居规划。如果客户未来从 MPRP 备份转为实际迁居马耳他，应在迁居前完成海外公司、信托、保险、投资账户、房产和家庭现金流规划，避免成为税务居民后再处理历史资产导致重复征税或申报风险。

固定注释：财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。
