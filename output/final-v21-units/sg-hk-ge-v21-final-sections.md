# 一、新加坡 EP/PIC｜私人投资公司 + Employment Pass 路径

## 第1章 客户家庭基本信息

本模块适用于希望通过新加坡真实经营主体或私人投资公司（PIC / Private Investment Company）建立区域总部、投资管理或海外业务承接职能，并由该新加坡公司为核心申请人申请 Employment Pass（EP）的家庭。PIC 不是新加坡政府独立移民项目，执行上应拆分为：新加坡公司设立、真实商业/投资活动建立、EP 岗位与薪资设计、COMPASS 评分、EP 获批后家庭准证及后续 PR 路径。

| 信息项 | 执行填写口径 | 文件/核验动作 |
|---|---|---|
| 主申请人 | 股东、董事、高管、投资管理负责人或区域业务负责人 | 护照、简历、学历、行业履历、薪酬与岗位说明 |
| 家庭成员 | 配偶、未成年子女、需规划教育及陪读成员 | 关系证明、出生/婚姻证明、护照 |
| 中国经营背景 | 中国公司、股权、业务、员工、利润来源 | 营业执照、审计报表、纳税记录、合同样本 |
| 境外业务基础 | 东盟/全球客户、供应链、投资资产、品牌/IP | 订单、客户名单、投资计划、银行流水 |
| 核心目标 | EP 身份、子女教育、资产配置、区域利润合规归集 | 目标排序与预算确认 |

## 第2章 核心策略

核心策略是“先做真实新加坡经营节点，再申请 EP”，而不是把 PIC 包装为单纯买身份。新加坡 EP 审核重点为合格雇佣、岗位真实性、薪资水平、候选人能力、公司业务实质与 COMPASS 评分。客户应建立一间具有真实董事决策、办公地址、银行账户、会计秘书、商业合同或投资活动的新加坡公司，并由该公司聘任主申请人为区域负责人、投资总监、业务发展总监或管理岗位。

| 策略层 | 执行动作 | 成功标准 |
|---|---|---|
| 身份策略 | 以新加坡公司聘用主申请人申请 EP | 薪资达到 MOM qualifying salary，COMPASS 可通过或满足豁免 |
| 公司策略 | 设立/收购新加坡私人公司，配置秘书、注册地址、银行账户 | 公司具备真实业务计划、资金、合同、账务 |
| 财税策略 | 中国业务留中国纳税，境外业务由新加坡真实承接 | 转让定价、合同、人员、银行流水闭环 |
| 家庭策略 | EP 获批后安排 DP/LTVP、教育、居住、保险 | 家属材料完整，学校与租房预算落实 |
| 长期路径 | EP 稳定续签后按 ICA PTS 路径评估 PR | IRAS 税单、薪资、居住、社会融入记录持续积累 |

## 第3章 合规清理详细方案

EP/PIC 的最大风险不是政府费用，而是“空壳自雇”和“利润错配”。递交前必须完成四类清理：身份材料真实性、公司商业实质、资金来源合规、税务居民与中国外汇规则匹配。

| 清理事项 | 具体要求 | 责任方 | 输出文件 |
|---|---|---|---|
| 学历与履历 | 学历证书、专业资格、过往职位与现任岗位一致 | 客户/顾问 | 学历验证、简历、推荐信 |
| 新加坡公司实质 | 注册、秘书、注册地址、银行、业务计划、合同 | 公司秘书/律师 | ACRA 文件、章程、董事决议、合同 |
| EP 岗位真实性 | 岗位职责、薪资、汇报线、公司业务需要 | 雇主/顾问 | 雇佣合同、岗位说明、组织架构 |
| 资金来源 | 设立资金、运营资金、薪资资金、投资资金 | 客户/会计师 | 银行流水、分红/工资/资产出售证明 |
| 中国合规 | ODI、37号文、FDI、个人购汇边界 | 中国律师/税务师 | 合规路径意见书 |

## 第4章 境外资金归集与投资架构

建议采用“中国经营主体 + 新加坡运营/PIC + 可选香港收款层 + 家族资产账户”的分层结构。中国公司继续保留中国来源利润并完税；新加坡公司只承接其实际承担的海外销售、区域管理、投资管理或品牌/IP 管理收益。

```text
中国经营主体[生产/研发/中国纳税]
  -> 采购/成本加成服务 -> 新加坡运营公司/PIC[EP雇主/区域总部/投资管理]
境外客户 -> 合同/付款 -> 新加坡公司
新加坡公司 -> 完税后分红/投资 -> 新加坡投资账户/家庭信托/保险/上层SPV
合规标注：ODI、37号文、CRS、转让定价、DTA受益所有人
```

| 资金类型 | 推荐路径 | 禁止动作 |
|---|---|---|
| 公司出资 | 中国公司对外投资需评估 ODI；个人持股需评估 37号文 | 地下钱庄、虚假贸易出资 |
| 海外订单收入 | 新加坡公司真实签约、履约、开票、收款 | 中国团队完成全部业务却由新加坡空壳开票 |
| 薪资资金 | 新加坡公司按 EP 薪资发薪并申报 | 名义高薪、实际不发或循环回流 |
| 投资资金 | 进入公司投资账户或个人合规账户 | 个人账户混收公司收入 |

## 第5章 投资使用建议

PIC 实务资金无官方最低投资额，但建议准备可支撑 12–24 个月运营的资金。资金用途应围绕业务真实需要配置，优先覆盖薪资、办公室、会计秘书、银行合规、审计税务、市场开发与投资活动。

| 用途 | 建议预算 | 执行说明 |
|---|---:|---|
| 公司注册及秘书 | S$315 政府基础费 + S$1,500–5,000/年服务 | ACRA 名称申请 S$15，注册 S$300；服务费另计 |
| EP 薪资池 | 至少 12 个月，按月薪 S$5,600+/金融 S$6,200+ 起算 | 年龄越高要求越高；递交前用 MOM 工具复算 |
| 办公与运营 | S$20,000–80,000/年 | 地址、共享办公室、员工、软件、差旅 |
| 专业服务 | S$8,000–25,000+ | 公司设立、开户、EP 商业叙事、递交 |
| 家庭安顿 | 单人 6个月 S$24,000–60,000；三口 S$60,000–150,000 | 国际学校另计 |

## 第6章 财富架构搭建方案

新加坡节点的财富架构价值在于“真实区域功能 + 透明低税环境 + 家族资产管理”。企业所得税名义税率 17%，但实际税负取决于利润性质、豁免、激励、成本扣除与实质。新加坡通常无股息预提税，但中国向新加坡支付股息、利息、特许权使用费或服务费时需审查预提税、DTA、受益所有人和转让定价。

### 6.1 单项目身份架构图

```text
客户家庭
  -> 主申请人担任新加坡公司高管/投资负责人
  -> 新加坡公司提交 EP
  -> EP 获批后主申请人赴新工作/纳税
  -> 配偶子女申请 DP/LTVP
  -> 稳定经营、纳税、居住后评估 PR(PTS)
```

### 6.2 财富与税务执行结构

| 架构节点 | 功能 | 文件留痕 |
|---|---|---|
| 中国经营主体 | 生产、研发、国内销售、纳税 | 合同、发票、审计、纳税申报 |
| 新加坡运营/PIC | 海外签约、区域管理、投资管理 | 董事会记录、合同、银行流水、员工记录 |
| 可选香港节点 | 大中华区收款或贸易协调 | 避免与新加坡功能重复 |
| 家庭信托/保险 | 传承、隔离、长期投资 | KYC、受益人文件、税务居民声明 |
| 个人税务居民 | EP 居住、薪酬、董事费申报 | IRAS 税单、租约、入境记录 |

## 第7章 税务分析

新加坡 EP/PIC 可解决的税务问题，是将真实境外利润放在新加坡纳税，而不是转移中国利润。若新加坡公司无人员、无办公、无董事决策、无真实合同，仅向中国公司收咨询费或特许权使用费，容易被中国税务机关按受益所有人、转让定价、特别纳税调整或反避税规则否定。

| 税务主题 | 执行结论 | 风险控制 |
|---|---|---|
| 新加坡企业所得税 | 名义税率 17%，实际税负按优惠和利润性质计算 | 建账、审计、税务申报 |
| 个人薪酬税 | EP 持有人按居住天数及薪酬在新申报 | 薪资真实发放，IRAS 记录完整 |
| 中国预提税/DTA | 中国向新加坡支付股息、利息、特许权使用费需审查 | 受益所有人、商业实质、合同与估值 |
| CFC | 中国居民控制低税境外公司且无合理商业目的会被关注 | 真实商业目的、利润与职能匹配 |
| CRS/FATCA | 新加坡账户按税务居民身份交换/申报 | 不隐瞒控制人及美国身份 |

## 第8章 资金跨境合规方案

资金跨境应分为个人资金、企业资金、境外存量资金三类处理。企业对外投资设立新加坡主体通常需评估 ODI；中国居民个人通过境外 SPV 持股或返程投资需评估 37号文；境外公司回投中国需按 FDI 处理。

| 资金来源 | 合规路径 | 递交材料 |
|---|---|---|
| 中国公司利润 | 完税后分红或 ODI 出资 | 审计报告、董事会决议、税单、ODI 文件 |
| 个人合法收入 | 工资、分红、资产出售、境外存量资金 | 个税记录、银行流水、买卖合同 |
| 境外业务收入 | 境外客户付款至新加坡公司 | 合同、发票、物流/交付证明 |
| 家族资产 | 信托、保险、投资账户 | KYC、资金来源、受益所有人声明 |

## 第9章 身份路径规划

EP 路径应分三阶段推进：先公司与岗位，后 EP 获批与入境，再续签及 PR 评估。MOM 当前 EP 政府费为申请费 S$105/人，批准后签发费 S$225/人，如需 Multiple Journey Visa 为 S$30/人。薪资门槛需按年龄与行业复算，2025 版常见基准为一般行业 S$5,600/月起、金融服务 S$6,200/月起，并需通过 COMPASS（通常至少 40 分，豁免情形除外）。

| 阶段 | 时间 | 关键动作 | 成果 |
|---|---|---|---|
| 0 预评估 | 1–2周 | COMPASS、薪资、学历、公司业务预评估 | 是否可递交 |
| 1 公司搭建 | 2–6周 | ACRA 注册、秘书、银行、业务计划 | EP 雇主主体完成 |
| 2 EP 递交 | 3–8周+ | MOM 申请、补件、批准 | IPA/EP 获批 |
| 3 家庭入境 | 1–3个月 | DP/LTVP、租房、学校、保险 | 家庭安顿 |
| 4 续签/PR | 12–36个月+ | 纳税、经营、居住、社会融入 | 续签或 PR 评估 |

## 第10章 教育规划

新加坡教育资源优质但竞争强。EP 家庭子女可根据年龄申请国际学校、私立学校或参加公立学校 AEIS/S-AEIS 竞争入学；公立学校名额不能作为确定性承诺。教育预算必须与身份时间轴同步，因为国际学校申请通常需要提前 6–12 个月准备。

| 教育路径 | 适用家庭 | 费用与执行 |
|---|---|---|
| 国际学校 | 预算充足、希望课程连续 | 年费常 S$25,000–50,000+/孩，另有申请费、校车、押金 |
| 私立学校 | 过渡期或英语适应 | 按学校收费，需核验学生准证/家属身份要求 |
| 公立学校 | 希望融入本地体系 | 需考试及名额，不保证录取 |
| 幼儿园 | 幼龄儿童 | 按园所和地区差异较大 |

## 第11章 福利居住国规划

EP 是工作准证，不等于永久福利身份。客户应按“商业保险 + 私人医疗 + 租房安顿 + 长期 PR 可能性”设计居住计划。EP 持有人在新加坡工作纳税，有助于建立长期记录；但公共住房、部分补贴和公民福利不适用于 EP 阶段。

| 模块 | EP 阶段安排 | PR 后可能变化 |
|---|---|---|
| 医疗 | 商业医疗保险、私人诊所/医院 | 可逐步接入更多本地制度 |
| 住房 | 私宅租赁为主 | PR 后购房选择改善但仍有限制 |
| 家属 | DP/LTVP 依附主申请人 | PR 后家庭稳定性提高 |
| 养老 | EP 外籍人士 CPF 规则有限 | PR/公民后 CPF 义务与权益变化 |

## 第14章 财税执行策划案全文

### 14.1 财税目标

本财税执行策划案的目标，是在不虚构交易、不隐瞒受益所有人、不规避 CRS 的前提下，建立新加坡真实经营和投资管理节点，使境外业务利润、投资收益、家庭资产配置与身份安排形成一致链条。财税方案不以“低税”作为唯一目标，而以可解释、可审计、可开户、可续签、可传承为核心。

### 14.2 架构图

```text
中国经营主体 -> 合同/成本加成/采购 -> 新加坡运营/PIC
境外客户 -> 合同/收款 -> 新加坡运营/PIC
新加坡运营/PIC -> 薪资/董事费 -> EP主申请人
新加坡运营/PIC -> 分红/投资 -> 家族信托/保险/投资账户
税务控制：转让定价 + DTA + CRS + CFC + 37号文/ODI
```

### 14.3 资金双循环架构

```text
经营循环：境外客户收入 -> 新加坡公司 -> 采购/外包/运营成本 -> 新加坡纳税 -> 再投资
家庭循环：完税利润/薪资 -> 家庭账户 -> 教育/生活/保险/投资 -> 税务居民申报
```

| 同期资料 | 功能风险分析、可比利润 | 防止利润错配 |

### 14.7 不可做事项

不得用新加坡空壳公司虚构咨询费；不得由个人账户承接公司收入；不得把中国团队创造的 IP 和利润无估值、无合同、无转让地放入新加坡；不得通过虚假贸易或地下钱庄注资；不得向银行、MOM、IRAS 隐瞒实际控制人和税务居民。

# 二、香港 ASMTP｜输入内地人才计划

## 第1章 客户家庭基本信息

香港 ASMTP 适用于中国内地居民已获香港真实雇主聘用，且具备香港所需而本地不易获得的专业知识、技能或经验的申请人。该路径不是投资移民，不要求投资额；成败核心在雇主真实性、职位必要性、薪酬市场性、申请人不可替代性及家庭受养安排。

| 信息项 | 执行填写口径 | 文件 |
|---|---|---|
| 主申请人 | 内地居民、专业人才、管理人员、技术人员 | ID 990A、护照/旅行证件、身份证、照片 |
| 香港雇主 | 香港真实运营公司 | ID 990B、商业登记证、财务/业务证明 |
| 职位 | 香港本地难以即时提供的人才岗位 | 职位说明、招聘证明、薪酬福利说明 |
| 家庭成员 | 配偶及受养子女 | 关系证明、受养人表格、经济能力证明 |
| 目标 | 工作居留、子女教育、香港税务与资产配置 | 预算与时间确认 |

## 第2章 核心策略

核心策略是“雇主合规 + 职位合规 + 人才匹配”。申请材料必须证明香港公司有业务需要、岗位职责清晰、薪酬福利与本地市场相若，且申请人拥有相关学历、资格或经验。若客户同时有香港公司或跨境业务，应避免把 ASMTP 做成无真实雇佣的身份包装。

| 策略层 | 执行动作 | 成功标准 |
|---|---|---|
| 雇主策略 | 完整解释公司业务、财务、招聘与岗位必要性 | 入境处认可岗位真实 |
| 人才策略 | 学历、资格、经验与岗位一一对应 | 证明本地不易获得 |
| 家庭策略 | 同步准备受养人材料 | 家属随行居留 |
| 财税策略 | 利用香港来源地税制与金融账户，但保持实质 | 不做空壳导管 |
| 长期策略 | 续签、通常居住、7年永居评估 | 保持雇佣与居住记录 |

## 第3章 合规清理详细方案

ASMTP 的材料不能只提交雇佣合同，应同时提交雇主为什么需要该申请人、为什么香港本地市场难以替代、薪酬为什么合理、申请人为什么能胜任。

| 清理事项 | 具体要求 | 输出 |
|---|---|---|
| 雇主商业实质 | 商业登记、财务报表、客户合同、办公室、员工 | 雇主合规包 |
| 职位真实性 | 职责、汇报线、招聘记录、市场薪酬 | 职位说明与招聘说明 |
| 申请人匹配 | 学历、工作证明、专业成果 | 人才匹配说明 |
| 家属依附 | 婚姻/出生、公证翻译、经济能力 | 受养人包 |
| 税务与薪俸 | 薪酬支付、MPF/税务安排 | 薪俸税规划 |

## 第4章 境外资金归集与投资架构

香港 ASMTP 本身不要求投资，但香港可作为大湾区经营、控股、融资、收款与资产配置节点。客户应区分个人工资、公司利润、投资收益和中国业务收入，避免香港个人账户代收中国公司款项。

```text
境外客户 -> 香港运营公司[合同/收款/融资/贸易]
香港公司 -> 采购/委托加工 -> 中国经营主体[出口/生产/中国完税]
香港公司 -> 分红/投资 -> 香港投资账户/家族信托/保险
个人 -> ASMTP雇佣薪酬 -> 香港薪俸税申报
```

| 资金类型 | 合规处理 | 风险提示 |
|---|---|---|
| 香港工资 | 由雇主按合同发放并申报薪俸税 | 不得名义发薪、实际返还 |
| 香港公司利润 | 按来源地判断利得税 | 离岸利得需证据支持 |
| 中国公司利润 | 留在中国纳税或合规分红/ODI | 不得通过香港个人账户转移 |
| 投资收益 | 以真实税务居民身份开户和申报 | CRS/FATCA 需一致 |

## 第5章 投资使用建议

ASMTP 无投资门槛。预算重点应放在香港生活、租房、子女教育、雇主合规材料和专业服务上。

| 用途 | 建议预算 | 说明 |
|---|---:|---|
| 签证签发费 | HK$230/人 | 以入境处收费表核验 |
| 顾问/律师单案 | HK$8,000–30,000 | 标准 ASMTP |
| 含雇主解释与家属 | HK$20,000–60,000 | 复杂雇主、补件、受养人 |
| 单人6个月生活费 | HK$90,000–180,000 | 租房、押金、交通、保险 |
| 三口6个月生活费 | HK$220,000–500,000 | 私校/国际学校另计 |

## 第6章 财富架构搭建方案

香港的价值在于地域来源税制、国际金融结算与大湾区便利。香港公司可承接真实海外销售、融资、控股、采购管理利润；若合同谈判、定价、客户维护、履约管理都在中国完成，利润不能简单放在香港。

### 6.1 单项目身份架构图

```text
香港雇主真实岗位 -> 内地人才获聘 -> ASMTP递交
ASMTP获批 -> 主申请人赴港工作/纳税
配偶子女 -> 受养人签证
持续雇佣/居住 -> 续签 -> 7年通常居住后评估永久居民
```

### 6.2 香港财富管理结构

| 节点 | 功能 | 风险控制 |
|---|---|---|
| 中国经营主体 | 生产、研发、国内业务 | 中国完税、出口合规 |
| 香港运营/控股公司 | 海外订单、融资、结算、控股 | 来源地证据、董事决策 |
| 香港个人账户 | 薪资、生活、投资 | 不代收公司收入 |
| 香港/新加坡信托保险 | 传承与隔离 | CRS、受益人披露 |

## 第7章 税务分析

香港利得税采用来源地原则，非香港来源利润是否征税需个案判断并保留证据。薪俸税则与在港受雇及工作安排相关。香港无一般股息预提税、无资本利得税，但频繁交易性利润可能被视为应税经营利润。内地向香港支付股息、利息、特许权使用费时，应按内地与香港税收安排及受益所有人规则审查。

| 税务主题 | 结论 | 执行控制 |
|---|---|---|
| 薪俸税 | ASMTP 工资通常需在香港申报 | 雇佣合同、发薪、报税 |
| 利得税 | 香港公司利润按来源地及经营事实判断 | 合同、人员、谈判、履约证据 |
| 离岸利得 | 可申请但不可伪造事实 | 保留合同、物流、会议、邮件 |
| DTA/税收安排 | 内地向香港支付款项需受益所有人 | 香港公司实质、非导管 |
| CRS/FATCA | 香港账户信息交换 | 税务居民声明一致 |

## 第8章 资金跨境合规方案

香港资金安排必须将个人、公司、家庭资产分账管理。中国公司设香港子公司需评估 ODI；香港公司回投中国需 FDI；中国居民通过境外 SPV 持股需评估 37号文。ASMTP 个人赴港工作不自动解决中国资产跨境问题。

| 场景 | 合规路径 | 文件 |
|---|---|---|
| 香港雇主发薪 | 香港公司发薪至个人账户 | 雇佣合同、工资单、税表 |
| 中国公司设港司 | ODI/商务/发改/外汇 | 境外投资备案/登记 |
| 港司回投中国 | FDI、外资企业登记 | 投资协议、外汇登记 |
| 个人投资账户 | 以真实税务居民身份开户 | KYC、CRS、自证表 |

## 第9章 身份路径规划

ASMTP 申请通常以入境处 ID 990A（申请人）及 ID 990B（雇主）为核心表格，政府签发费通常 HK$230。获批后申请人按许可来港工作，配偶及未成年子女可申请受养人。后续续签通常围绕雇佣持续性、薪资、在港工作和居住情况；通常居住满 7 年后可评估香港永久居民身份。

| 阶段 | 周期 | 动作 | 成果 |
|---|---|---|---|
| 预评估 | 1–2周 | 雇主、职位、人才匹配 | 是否可递交 |
| 材料准备 | 2–4周 | ID 990A/990B、雇佣与证明 | 递交包 |
| 审理 | 4–12周+ | 入境处审理、补件 | 获批/补件/拒签 |
| 入境工作 | 获批后 | 赴港、发薪、租房 | 身份生效 |
| 续签/永居 | 持续 | 稳定雇佣、通常居住 | 续签及7年后评估 |

## 第10章 教育规划

香港教育路径包括国际学校、私立学校、直资学校和本地学校。受养人子女可在香港就读，但学校录取、学位、语言和学费应独立规划。国际学校通常需提前申请，并准备面试、英文能力证明和押金。

| 学校类型 | 适合家庭 | 预算/执行 |
|---|---|---|
| 国际学校 | 希望衔接 IB/AP/A-Level | 学费、债券、建校费差异大 |
| 私立/直资 | 希望兼顾本地与国际 | 需面试及名额 |
| 官津学校 | 长期融入本地 | 学位和校网需规划 |
| 幼儿园 | 幼龄儿童 | 提前申请、区域选择重要 |

## 第11章 福利居住国规划

ASMTP 是工作居留安排，受养人身份依附主申请人。香港医疗、住房、教育和长期公共福利与居民身份、通常居住年期及具体资格有关。客户应以商业保险、私宅租赁、学校申请和现金流准备作为初期保障。

| 模块 | 初期安排 | 长期规划 |
|---|---|---|
| 医疗 | 商业保险 + 私营/公营服务了解 | 长居后评估公共服务资格 |
| 住房 | 租赁为主，预算压力高 | 根据工作地点和学校选区 |
| 家属 | 受养人签证 | 主申请人身份变动会影响家属 |
| 永居 | 不保证，需通常居住 | 7年后依法评估 |

# 三、格鲁吉亚投资居留｜US$300,000 投资居留路径

## 第1章 客户家庭基本信息

本模块适用于希望通过在格鲁吉亚投资不少于 US$300,000 等值 GEL，建立投资居留、欧亚业务节点、轻资产经营或个人税务居民备选路径的家庭。投资可为商业投资或符合条件的不动产投资；初次投资居留常为 5 年，并可能在持续满足投资/营业额等条件后通向永久居留。递交前必须核验 2025/2026 修法、SDA 费用、投资门槛、短期房产居留门槛及营业额续签要求。

| 信息项 | 执行填写口径 | 文件 |
|---|---|---|
| 主申请人 | 投资人、企业主、高净值家庭成员 | 护照、合法入境证明、照片 |
| 投资类型 | US$300,000 等值 GEL 商业或不动产投资 | 房产登记/估值或公司投资证明 |
| 家庭成员 | 配偶、子女、同行家属 | 关系证明、翻译认证 |
| 业务目标 | 欧亚业务、远程服务、IT、贸易、资产配置 | 商业计划、合同、资金来源 |
| 长期目标 | 居留维护、税务居民备选、永居可能 | 居住计划、税务退出评估 |

## 第2章 核心策略

核心策略是“投资真实性 + 资金来源合规 + 居留维护条件 + 税务居民单独规划”。取得格鲁吉亚投资居留不等于自动成为格鲁吉亚税务居民，也不等于自动免除中国或其他税务居民国纳税义务。若客户希望把格鲁吉亚作为经营节点，必须有真实客户、合同、交付、人员、办公、银行流水和纳税记录。

| 策略层 | 执行动作 | 成功标准 |
|---|---|---|
| 投资策略 | US$300,000 等值 GEL 投资，取得评估与登记 | 投资符合居留路径要求 |
| 居留策略 | 申请投资居留并维护续签条件 | 5年居留及后续永居评估 |
| 财税策略 | 格鲁吉亚公司承接真实欧亚/远程业务 | 利润与职能匹配 |
| 家庭策略 | 安顿、教育、医疗、居住成本准备 | 家庭可实际落地 |
| 风险策略 | 核验 SDA 费用、法规修订、营业额要求 | 避免旧政策误用 |

## 第3章 合规清理详细方案

格鲁吉亚投资居留的核心清理是投资文件、资金来源、产权/公司文件和税务居民边界。房产投资需特别核验产权、估值、登记、抵押、开发商风险；商业投资需核验公司登记、出资、审计、营业额与经营实质。

| 清理事项 | 要求 | 输出 |
|---|---|---|
| 投资真实性 | US$300,000 等值 GEL 投入，估值/登记可验证 | 评估报告、Public Registry extract |
| 资金来源 | 分红、工资、经营所得、资产出售、境外存量资金 | 银行流水、税单、合同 |
| 公司/房产尽调 | 权属、负担、估值、合规用途 | 律师尽调报告 |
| 居留材料 | 护照、合法入境、申请表、照片、关系证明 | 申请包 |
| 税务居民边界 | 判断是否满足格鲁吉亚税务居民及原国退出 | 税务意见 |

## 第4章 境外资金归集与投资架构

建议将格鲁吉亚定位为低成本欧亚业务节点、投资持有节点或个人税务居民备选节点，而非万能避税地。若客户确有欧亚、远程服务、IT、咨询或贸易业务，可由格鲁吉亚公司真实签约、收款和交付；中国团队如提供支持，应按合同收取服务费并在中国纳税。

```text
境外客户 -> 格鲁吉亚公司[合同/交付/收款/本地实质]
格鲁吉亚公司 -> 外包/采购 -> 中国经营主体[成本加成/中国纳税]
格鲁吉亚公司 -> 分红 -> 个人/控股公司/信托
个人 -> 格鲁吉亚税务居民评估 + 原居民国退出评估
```

| 资金类型 | 合规路径 | 禁止动作 |
|---|---|---|
| 投资款 | 已完税资金、境外存量资金、合规汇出 | 地下钱庄、虚假贸易 |
| 公司收入 | 格鲁吉亚公司真实合同和交付 | 仅开票无人员无交付 |
| 中国支持服务 | 中国公司按合同收费并纳税 | 无偿转移中国团队利润 |
| 分红/投资收益 | 按居民国和来源地申报 | 隐瞒 CRS/KYC 信息 |

## 第5章 投资使用建议

本项目以 US$300,000 等值 GEL 投资居留为主。投资可考虑商业公司出资、并购、运营资产或符合条件的不动产。另有短期房产居留路径旧门槛 US$100,000，搜索结果提示 2026-03-01 起提高至 US$150,000，但本模块不以短期房产居留为主。

| 投资用途 | 建议 | 风险控制 |
|---|---|---|
| 商业投资 | 设立/投资格鲁吉亚公司，开展 IT、咨询、贸易、服务 | 需要真实经营、审计、税务记录 |
| 不动产投资 | 购买符合条件房产并取得评估与登记 | 权属尽调、估值足额、流动性风险 |
| 运营资金 | 工资、办公、会计、税务、市场开发 | 与商业计划一致 |
| 家庭安顿 | 单人6个月 US$6,000–15,000；三口 US$15,000–36,000 | 不等同法定投资额 |

## 第6章 财富架构搭建方案

格鲁吉亚架构的价值在于低成本经营、欧亚业务连接和个人税务居民备选。若客户的核心团队和客户管理仍在中国，仅让格鲁吉亚公司开票会形成利润错配。可行结构是：格鲁吉亚公司承担境外客户签约、交付、管理、收款，中国公司承担外包研发或支持并收成本加成。

### 6.1 单项目身份架构图

```text

投资文件/评估/登记 -> SDA投资居留申请

```

## 第7章 税务分析

格鲁吉亚投资居留本身不是税务居民切换工具。客户必须分别判断：是否满足格鲁吉亚税务居民条件；是否仍为中国或其他国家税务居民；境外公司是否触发 CFC；银行账户是否进行 CRS 信息交换；分红、利息、特许权使用费是否有预提税及 DTA 适用。

| 税务主题 | 结论 | 控制 |
|---|---|---|
| 格鲁吉亚公司税 | 按当地税制及利润性质申报 | 会计、审计、纳税记录 |
| 个人税务居民 | 居留不等于税务居民 | 居住天数/高净值规则核验 |
| DTA | 格鲁吉亚与部分国家有 DTA | 逐项查税收协定 |
| CFC | 中国/高税居民控制低税公司需合理商业目的 | 实质经营、董事决策、合同 |
| CRS/FATCA | 银行 KYC 与税务居民披露 | 不隐瞒控制人和美国身份 |
| ODI/37号文 | 中国资金出资格鲁吉亚公司需评估 | 中国合规意见 |

## 第8章 资金跨境合规方案

投资款应以已完税收入、境外存量资金、合法分红、资产出售款或合规企业出资形成。中国公司出资需评估 ODI；中国居民个人以境外 SPV 投资需评估 37号文；境外 SPV 回投中国需按 FDI 处理。所有投资款需形成银行流水闭环、合同闭环和税务解释闭环。

| 来源 | 路径 | 文件 |

| 个人分红/工资 | 完税后汇出或境外存量资金 | 税单、银行流水、分红决议 |
| 资产出售 | 房产/股权/证券出售 | 买卖合同、完税证明、收款流水 |

| 境外收入 | 境外账户直接投资 | 合同、发票、银行流水、税务申报 |

## 第9章 身份路径规划

格鲁吉亚投资居留通常要求投资不少于 US$300,000 等值 GEL，可为商业投资或符合条件的不动产投资；初次投资居留常为 5 年，并可能通向永久居留，条件通常涉及持续投资、营业额或其他维护要求。SDA / Public Service Hall 按处理时限收取服务费，需递交前在官方页面选择处理时限核验。

## 第10章 教育规划

格鲁吉亚教育成本相对低，但国际化选择集中在第比利斯等城市。若家庭以格鲁吉亚作为主要居住地，应提前评估学校语言、课程体系、升学衔接和医疗保险；若仅作为身份备选，则教育规划可作为次级配置。

## 第11章 福利居住国规划

格鲁吉亚投资居留提供居住合法性，但公共福利、医疗、长期保障不能按高福利国家预期设计。客户应配置商业医疗保险、私立医疗预算、租房或购房维护预算，并将居住天数与税务居民评估分开管理。

| 税务居民 | 单独评估 | 原居民国退出/双重居民冲突解决 |

## 第14章 财税执行策划案全文

### 14.1 财税目标

格鲁吉亚财税方案目标，是在投资居留基础上建立可解释的欧亚业务、资产持有或个人税务居民备选结构。方案不能承诺取得居留即自动免税；任何降税效果都必须来自真实业务职能、真实居住事实、合法资金来源、原居民国退出评估和持续申报。

### 14.2 架构图

```text
境外客户 -> 格鲁吉亚公司[合同/交付/收款]
格鲁吉亚公司 -> 本地办公/工资/税费
格鲁吉亚公司 -> 中国公司外包服务[成本加成/中国纳税]
格鲁吉亚公司 -> 完税分红 -> 个人/控股公司/信托
个人 -> 居住天数/高净值规则 -> 格鲁吉亚税务居民评估
控制点：CFC、CRS、DTA、ODI/37号文、资金来源
```

### 14.3 资金双循环架构

```text
投资循环：已完税资金 -> US$300,000投资 -> 房产/公司登记 -> 居留基础 -> 维护/退出
经营循环：境外客户收入 -> 格鲁吉亚公司 -> 成本/税费/外包 -> 利润 -> 分红/再投资
家庭循环：个人资金 -> 居住/教育/医疗/保险 -> 税务居民申报
```

### 14.7 不可做事项

不得只注册格鲁吉亚公司而无办公、人员、客户管理即截留利润；不得用格鲁吉亚账户收中国境内业务收入；不得宣称投资居留自动免中国税；不得忽略原税务居民国 CFC 和 CRS；不得在未核验 2025/2026 修法前沿用旧房产门槛。
