## 一、美国 EB-1A｜全球征税高风险迁入执行策划案

### 第1章 客户家庭基本信息

| 项目 | 执行口径 |
|---|---|
| 目标身份 | EB-1A Extraordinary Ability，美国就业移民第一优先类别，无雇主、无劳工证，可 self-petition |
| 适配人群 | 企业家、科研/技术专家、艺术/体育/商业领域高成就人士、行业奖项或持续声誉较强者 |
| 家庭覆盖 | 主申请人、配偶、未满21周岁未婚子女；以 I-140 + I-485 或领馆程序执行 |
| 核心税务属性 | 取得绿卡或满足实质居留测试后，美国对全球所得征税，海外公司、基金、账户、信托、保险均进入强申报区 |
| 关键前置动作 | 登陆前完成 PFIC 清理、CFC/GILTI 评估、海外账户披露清单、非美信托/保险合规审查、遗产税规划 |

家庭档案必须在立案阶段形成“身份证据档案 + 全球资产税务档案”双档案。身份档案用于证明 extraordinary ability；税务档案用于判断登陆前是否需要分红、出售、赠与、信托、保险或股权重组。

### 第2章 核心策略

EB-1A 的成功关键不是投资额，而是证据质量与美国移民局两步法审查：先满足法规列明的证据类别，再通过 final merits determination 证明申请人已达到领域顶尖层级并将继续在美国从事相关工作。

执行策略分三条线同步推进：

1. **移民证据线**：以 8 CFR 204.5(h)(3) 十项标准为矩阵，建立“奖项、会员、媒体、评审、原创贡献、学术/商业成果、展览、关键职位、高薪、商业成功”证据包。
2. **登陆税务线**：在成为美国税务居民前完成全球资产梳理，重点处理 PFIC、CFC、FBAR、FATCA、Form 5471、Form 8621、Form 3520/3520-A 风险。
3. **家庭传承线**：美国遗产税和赠与税对绿卡/美国税务居民影响重大，高净值家庭必须先设计遗产税暴露、非美资产持有和受益人结构。

### 第3章 合规清理详细方案

| 清理事项 | 执行动作 | 责任方 | 完成节点 |
|---|---|---|---|
| 身份材料真实性 | 核验奖项、媒体、推荐信、论文引用、协会会员、薪酬证明是否可被第三方验证 | 移民律师 + 客户 | I-140 前 |
| 中国税务完税 | 梳理公司分红、股权转让、工资奖金、个人所得税、企业所得税 | 中国税务师 | 登陆前 |
| 境外公司控制 | 识别香港/新加坡/BVI/Cayman/其他境外公司是否构成美国 CFC | 美国 CPA/税务律师 | 登陆前 |
| 非美国基金 | 识别私募基金、ETF、保险投资账户、境外基金是否 PFIC | 美国税务师 | 登陆前 |
| 银行证券账户 | 建立 FBAR / FATCA 账户清单，含最高余额、账户号、持有人、受益人 | 客户 + 私行 | 登陆前 |
| 信托保险 | 审查非美信托、家族信托、大额保单是否触发美国 grantor trust、foreign trust、PFIC 规则 | 美国律师 | 登陆前 |

### 第4章 境外资金归集与投资架构

美国不是低税资金归集地。赴美前的资金归集应以“已完税、可解释、可申报”为原则，而非把资金临时转入美国账户。

```text
中国经营主体/个人资产
  -> 完税分红、股权估值、债权债务整理
  -> 香港/新加坡真实经营公司（如有）
  -> 非美基金/PFIC清理或替换
  -> 合规信托/保险/遗产规划
  -> 美国个人税务居民节点
  -> 全球所得申报 + FBAR/FATCA + CFC/PFIC表格
```

执行要点：
- 登陆前确认哪些资产继续个人直接持有，哪些资产进入合规信托/保险，哪些资产应出售或转换。
- 境外公司不可由美国居民在美国境内实际管理全部决策，否则可能引发美国税务归属和申报复杂化。
- 美国账户入金必须能解释来源：分红、薪酬、股权转让、房产出售、投资赎回、赠与或贷款均需合同和税务凭证。

### 第5章 投资使用建议

EB-1A 无投资门槛。资金安排围绕生活、税务、教育、医疗和资产保护展开。

| 用途 | 建议预算 | 执行说明 |
|---|---:|---|
| 6个月安顿费（单人） | USD 18,000–45,000 | 城市差异极大；纽约、湾区、洛杉矶需更高 |
| 6个月安顿费（三口之家） | USD 45,000–120,000 | 租房押金、购车、保险、学校、搬家另计 |
| 美国税务申报预算 | USD 3,000–30,000+/年 | 视海外公司、信托、PFIC、账户数量决定 |
| 遗产税/信托规划预算 | USD 15,000–100,000+ | 高净值家庭需单独方案 |

### 第6章 财富架构搭建方案（有效税务架构）

美国 EB-1A 的有效税务架构原则：**先规划，后登陆；先申报，后投资；先隔离遗产税，后配置美国资产。**

| 架构层 | 建议做法 | 美国税务风险 |
|---|---|---|
| 中国经营主体 | 保留真实经营、利润和纳税记录；赴美前确认是否分红或股权重组 | 分红、股权出售、CFC/GILTI、Subpart F |
| 香港/新加坡公司 | 只有真实人员、合同、客户、董事决策时保留 | Form 5471、CFC、GILTI、实际管理地 |
| 非美国基金 | 赴美前识别并清理 PFIC；必要时转为美国合规产品 | Form 8621、超额分配惩罚税、利息 charge |
| 非美信托 | 成为美国税务居民前设计；明确 grantor / non-grantor 后果 | Form 3520/3520-A、分配 throwback tax |
| 美国资产 | 控制美国 situs assets 暴露，配合遗产税规划 | 美国遗产税、赠与税、州税 |
| 全球账户 | 建立年度账户清单 | FBAR、Form 8938、罚款风险 |

不可做事项：登陆后临时把资产转入离岸信托；隐瞒境外账户；用亲属代持境外公司；持有大量 PFIC 不申报；把非美国保险当作免税壳。

### 第7章 税务分析

美国税务居民包括绿卡持有人和满足实质居留测试者。成为美国税务居民后，工资、分红、利息、租金、资本利得、海外公司归属收入、信托分配等均需按美国规则申报。

重点规则：
- **全球征税**：美国税务居民按全球所得纳税，外国税收抵免只能减少双重征税，不能消灭申报义务。
- **CFC / GILTI / Subpart F**：美国股东控制境外公司可能对未分配利润提前纳税。
- **PFIC**：非美国基金、部分保险投资产品可能触发高惩罚性税制和 Form 8621。
- **FBAR / FATCA**：海外金融账户与金融资产达到门槛需申报；漏报罚款可高于税款。
- **遗产税**：绿卡/美国 domicile 规划不当会将全球资产纳入美国遗产税风险。

### 第8章 资金跨境合规方案

| 资金类型 | 跨境动作 | 文件要求 | 风险控制 |
|---|---|---|---|
| 中国公司分红 | 中国完税后汇出或留存境外账户 | 董事会决议、审计报表、分红凭证、完税证明 | 避免未完税利润直接出境 |
| 股权转让款 | 估值、协议、税务申报后入境美国 | SPA、估值报告、税单、收款流水 | 防止低价转让被重估 |
| 境外投资赎回 | 清理 PFIC 后再转入美国 | 认购/赎回单、收益明细、税务报告 | 确认美国税务后果 |
| 亲属赠与/贷款 | 建立赠与或贷款协议 | 赠与声明、贷款合同、利率、还款记录 | 注意 Form 3520 或赠与税规则 |

### 第10章 教育规划

美国教育资源强，但税务居民身份不自动带来低成本教育。K-12 公立学校通常与居住地址绑定；私校竞争和费用高；大学阶段涉及州内学费、奖学金、529计划和家庭收入资产披露。

| 阶段 | 规划动作 | 预算提示 |
|---|---|---|
| K-12 公立 | 选州、选学区、租购房地址匹配 | 房租/房产税是隐性教育成本 |
| K-12 私立 | 提前申请、面试、推荐、语言支持 | USD 25,000–70,000+/年/孩 |
| 大学 | 规划州内居民资格、AP/竞赛、529 | 私立大学总成本可 USD 80,000+/年 |
| 税务联动 | 529、赠与、教育信托需美国税务审查 | 避免非美账户影响 FAFSA/CSS 披露 |

### 第11章 福利居住国规划

美国福利高度依赖身份、州、收入和保险。新移民不应以福利为主要规划目标，而应优先建立医疗保险、商业保险、税务申报和信用记录。

| 模块 | 执行安排 |
|---|---|
| 医疗 | 购买雇主保险、Marketplace 保险或私人保险；未保险风险极高 |
| 社保 | 社安福利取决于工作年限和缴税记录 |
| 税务居民 | 州税差异巨大，迁居州选择必须与工作、学校、生活真实一致 |
| 信用 | SSN、银行账户、信用卡、租房记录逐步建立 |
| 生活费用 | 三口之家建议至少准备 6–12 个月现金流，医疗免赔额另备 |

### 第14章 财税执行策划案全文

#### 14.1 财税目标

本项目财税目标不是降低美国税率，而是降低“登陆后无法补救”的合规成本。客户必须在美国税务居民开始日前完成全球资产穿透清单，按“个人资产、公司股权、金融账户、基金、信托、保险、不动产、债权债务、知识产权”分类。

#### 14.2 迁入前资产盘点

| 资产类别 | 必查项目 | 输出文件 |
|---|---|---|
| 境内公司股权 | 持股比例、未分配利润、历史分红、估值 | 股权结构图、估值报告、完税记录 |
| 境外公司 | 控制权、董事、利润、银行账户、实质 | CFC/GILTI 初判表 |
| 金融产品 | 基金、ETF、保险投资账户 | PFIC 清单、Form 8621 风险表 |
| 信托/保险 | 设立人、受托人、受益人、撤销权 | 3520/3520-A 适用意见 |
| 银行证券账户 | 最高余额、共同持有人、受益人 | FBAR/8938 清单 |
| 不动产 | 所在地、买入成本、贷款、租金 | 租金与资本利得台账 |

#### 14.3 资金双循环架构

```text
经营循环：中国/香港/新加坡真实经营收入 -> 当地完税 -> 合规分红/工资/服务费 -> 美国申报抵免
家庭循环：已完税资产 -> 合规账户/信托/保险 -> 教育/医疗/生活支出 -> 年度税务披露
禁止循环：未完税利润 -> 个人账户 -> 美国消费；空壳离岸公司 -> 隐匿账户 -> 不申报
```

#### 14.4 PFIC 处置

美国税务居民持有非美国共同基金、ETF、部分私募基金、部分投资型保险，可能构成 PFIC。执行步骤：
1. 登陆前列出所有非美国基金及保险投资底层。
2. 判断是否出售、转换为美国合规产品、QEF election 或 mark-to-market。
3. 未处理资产不得在首个美国税年后才被动发现。

#### 14.5 CFC / GILTI / Subpart F 处置

客户及家庭成员需按美国 attribution 规则判断是否为美国股东，境外公司是否 CFC。若构成 CFC，需评估 Form 5471、GILTI、Subpart F、分红和外国税收抵免。境外公司董事会、管理地、合同签署和银行授权必须真实，不可由美国住所远程全权管理。

#### 14.6 FBAR / FATCA 执行

所有境外金融账户按最高余额和持有人穿透归集。共同账户、公司账户、签字权账户、信托账户、证券账户、基金账户均需判断。FBAR 与 Form 8938 是不同申报体系，不能相互替代。

#### 14.7 遗产税与赠与税

登陆前应明确美国 domicile 风险、美国 situs assets、非美国资产持有结构、配偶和子女身份。高净值家庭需专项设计 irrevocable trust、life insurance、pre-immigration gift、股权重组或遗产税保险。任何设计必须在美国税务居民开始日前完成并具有真实法律效力。

#### 14.8 中国合规衔接

## 二、澳大利亚 482 SID + 186 TRT 衔接｜工作居留至永居执行策划案

### 第1章 客户家庭基本信息

| 项目 | 执行口径 |
|---|---|
| 目标身份 | Subclass 482 Skills in Demand（SID）工作签证，衔接 Subclass 186 Employer Nomination Scheme TRT 永居 |
| 适配人群 | 具备澳洲雇主 offer、职业在适用清单、薪资达市场工资和收入门槛、满足英语/健康/品行者 |
| 家庭覆盖 | 主申请人、配偶及符合条件子女 |
| 核心税务属性 | 入境工作后较可能成为澳大利亚税务居民，对全球所得、海外公司、海外信托、资本利得和 CFC 进行申报 |
| 核心风险 | 雇主真实性、职位真实性、薪资门槛、LMT、PAYG/Super、CFC、海外资产 CGT 成本基础 |

### 第2章 核心策略

482 SID 不是投资项目，其核心是“真实雇主 + 真实岗位 + 合格职业 + 合规薪酬 + 申请人技能”。执行中必须同时规划 186 TRT：从 482 第一天开始保留工资、岗位、雇主合规、税务和 Super 记录，避免到永居阶段发现证据断层。

### 第3章 合规清理详细方案

| 清理事项 | 执行动作 | 责任方 |
|---|---|---|
| 雇主资格 | 核验 ABN、财务、经营、担保资格、无不良记录 | 澳洲移民律师/雇主 |
| 职位真实性 | 职位描述、组织架构、市场薪资、业务需要 | 雇主/律师 |
| LMT | 如适用，按法规保留招聘广告与结果 | 雇主 |
| 薪资合规 | 满足 Core Skills Income Threshold、市场工资和合同 | 雇主/会计 |
| PAYG/Super | 工资通过 payroll，按澳洲规则预扣和缴 Super | 雇主/会计 |
| 迁入税务 | 海外资产估值、公司控制权、信托、投资账户 | 澳洲税务师 |

### 第4章 境外资金归集与投资架构

```text
赴澳前：中国公司/境外公司/个人资产估值 -> 分红或股权整理 -> 信托/保险审查
赴澳后：澳洲工资 -> PAYG预扣 + Super -> 澳洲个人申报
海外收入：股息/利息/租金/资本利得 -> 澳洲全球所得申报 + 外国税收抵免
境外公司：董事决策与实际管理地隔离 -> CFC/公司居民风险审查
```

### 第5章 投资使用建议

482/186 无投资门槛。资金使用以安顿、职业过渡、税务申报和教育医疗为主。

| 用途 | 建议预算 |

| 三口之家6个月安顿 | AUD 45,000–90,000 |
| 移民律师/代理 | AUD 5,000–20,000+ |
| 税务迁入规划 | AUD 5,000–50,000+ |
| 私校/国际教育 | AUD 20,000–45,000+/年/孩 |

### 第6章 财富架构搭建方案（有效税务架构）

澳洲属于高税迁入地。有效架构不是把利润放到澳洲，而是避免个人迁澳后使境外公司、信托和资产被澳洲规则不必要穿透。

| 架构节点 | 执行建议 | 税务关注 |
|---|---|---|
| 中国经营主体 | 保留中国管理、合同、人员、纳税 | 分红、DTA、外国税收抵免 |
| 香港/新加坡公司 | 仅保留真实境外业务，不由澳洲个人日常管理 | 澳洲公司居民、PE、CFC |
| 家庭信托 | 迁入前审查 settlor、beneficiary、分配规则 | 澳洲 trust attribution |
| 个人投资账户 | 记录迁入日市值作为澳洲 CGT 规划基础 | 资本利得、成本基础 |
| 澳洲工资账户 | 所有工资纳入 PAYG，Super 合规 | 雇主合规、186证据 |

### 第7章 税务分析

澳洲税务居民判断不仅看签证，还看居住、住所、183天、永久居所和生活中心。一旦成为澳洲税务居民，全球所得、海外资本利得、境外公司控制、海外信托分配均需申报。

重点：
- **税务居民**：482 持有人长期在澳工作和家庭居住，通常有较高居民风险。
- **CFC**：澳洲居民控制低税境外公司，可能归属征税。
- **Super**：雇主需按规则为雇员缴纳养老金，记录也支持 186 TRT。
- **PAYG**：工资预扣与税单是永居和税务合规核心证据。
- **CGT**：迁入前资产估值对未来处置至关重要。

### 第14章 财税执行策划案全文

澳洲财税执行策划案以“税务居民开始日 + 资产成本基础 + CFC/信托/公司管理地隔离 + PAYG/Super 合规”为主轴。

#### 14.1 税务居民开始日

记录入境日、租房/购房、家庭到达、雇佣开始、银行开户、学校入学、离境天数。由澳洲税务师判断居民开始日并出具备忘录。

#### 14.2 迁入前估值

股票、基金、境外公司股权、房产、加密资产、债权均需形成迁入日前后估值台账。未来出售时，澳洲 CGT 计算需要可靠成本基础。

#### 14.3 CFC 与公司管理地

#### 14.4 PAYG / Super

482 和 186 的移民合规与税务合规高度绑定。工资必须通过 payroll，按 PAYG 预扣；Super 按法定比例和时间缴纳；payslip、tax return、super statement 同步归档。

#### 14.5 海外收入申报

中国分红、香港账户利息、新加坡公司分红、海外租金、资本利得、信托分配均需进入澳洲年度申报。外国税收抵免需保留完税证明。

#### 14.6 资金双循环

```text
工资循环：澳洲雇主 -> Payroll/PAYG/Super -> 澳洲家庭生活与税务申报
```

## 三、日本 HSP / 经营管理｜非永住窗口与经营实体执行策划案

### 第1章 客户家庭基本信息

| 项目 | 执行口径 |

| 适配人群 | 高学历高收入专业人士、研究/技术/经营人才；或在日本真实设立并经营公司者 |

| 核心风险 | 经营管理空壳、办公室不实、资本/雇佣要求变化、海外资产调书、CFC、国外送金 |

### 第2章 核心策略

日本方案分两种主线：HSP 以积分和专业能力为核心；经营管理以真实公司、办公室、资本、雇员、客户、税务社保为核心。家庭财税策略必须利用非永住者阶段窗口完成海外资产、送金和经营利润隔离，避免长期成为永住税务居民后全球资产申报失控。

### 第7章 税务分析

日本个人税务居民通常分为非永住者和非非永住者（实务常称永住税务居民口径）。非永住者对部分国外来源所得可能在未汇入日本时有不同处理；长期居住后全球所得、海外资产和继承赠与税暴露明显增加。

重点：
- **非永住窗口**：需严格记录居住年限、国外来源收入、汇入日本金额。
- **海外资产申报**：国外财产调书、财产债务调书在高净值家庭中非常关键。
- **CFC**：日本居民控制低税境外公司可能归属征税。
- **继承赠与税**：居住年限、国籍、住所、资产所在地共同影响风险。

### 第14章 财税执行策划案全文

日本财税策划以“非永住者窗口 + 国外送金管理 + 日本公司真实经营 + 海外公司实际管理地隔离 + 继承赠与税预案”为核心。

#### 14.1 非永住者/永住税务居民切换

建立日本居住年限台账，记录入境、出境、住所、家庭、工作、公司经营、银行、学校。每年判断是否仍处于非永住者税务范围，以及国外来源所得是否因汇入日本而纳税。

#### 14.2 国外送金管理

每笔汇入日本的资金必须标注来源：工资、分红、贷款、资本金、生活费、投资赎回。保留原始完税凭证、合同和银行流水，避免被认定为未申报国外所得。

#### 14.3 日本经营管理公司税务

公司需按日本会计税务规则进行法人税、消费税、源泉所得税、社会保险、雇员工资处理。空壳公司或无真实办公室、无客户、无经营流水，会同时伤害签证和税务合规。

#### 14.4 海外公司与 CFC

若客户长期为日本税务居民并控制香港/新加坡/离岸公司，应由日本税理士判断 CFC。境外公司必须保留当地董事、办公室、人员、合同和银行决策，不能由日本住所遥控全部经营。

#### 14.5 海外资产申报与继承赠与税

高净值客户需每年判断国外财产调书、财产债务调书义务。继承赠与税需提前设计资产归属、受益人、保险、信托、赠与节奏，并与中国税务和外汇规则衔接。

```text
日本本地循环：日本公司 -> 日本客户/收入 -> 法人税/消费税/社保 -> 合规薪酬
海外资产循环：中国/香港/新加坡资产 -> 当地完税 -> 是否汇入日本 -> 日本申报判断
```

> **财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。**

## 四、新西兰 SMC / AIP｜过渡居民、FIF/CFC 与投资资金执行策划案

### 第1章 客户家庭基本信息

| 项目 | 执行口径 |
|---|---|
| 目标身份 | Skilled Migrant Category Resident Visa（SMC）或 Active Investor Plus（AIP） |
| 适配人群 | SMC：55岁以下、有认可雇主 skilled job/job offer、6分制；AIP：可投入合格投资并能证明资金来源者 |

| 核心税务属性 | 成为新西兰税务居民后全球所得申报；过渡居民窗口可规划部分海外被动收入；窗口结束后 FIF/CFC 影响重大 |
| 核心风险 | SMC 工作真实性/工资门槛；AIP 合格投资和资金来源；过渡居民资格、FIF/CFC、海外信托 |

### 第2章 核心策略

SMC 与 AIP 应在同一模块下分两条执行线：SMC 以技能、雇主、工资、6分制为核心；AIP 以合格投资、资金来源、投资期限和 KYC 为核心。两者共同的财税核心是：确定新西兰税务居民开始日，争取过渡居民窗口，迁入前完成海外资产重组。

### 第3章 合规清理详细方案

| 清理事项 | SMC | AIP | 共同财税动作 |
|---|---|---|---|
| 身份资格 | 年龄、英语、健康品行、6分、skilled job | 申请人背景、健康品行 | 税务居民开始日 |
| 工作/投资 | 雇主认证、职位、工资、注册 | Growth/Balanced 合格投资 | 资金来源和完税证明 |
| 材料 | IQA、职业注册、雇佣合同 | 资产、投资计划、KYC | 海外资产清单 |
| 风险 | 工资/职位不达标 | 投资不合格或资金来源不清 | FIF/CFC/海外信托 |

### 第6章 财富架构搭建方案（有效税务架构）

新西兰有效架构核心是利用过渡居民窗口并管理 FIF/CFC，而不是把新西兰作为低税利润中心。

| 架构节点 | 执行建议 | 税务关注 |
|---|---|---|
| 新西兰个人税务居民 | 确定居民开始日和过渡居民资格 | 全球所得、过渡居民豁免 |
| AIP 投资账户 | 仅投 acceptable investments，保留 KYC | 投资收益、资金来源 |
| 海外公司 | 保留真实经营地和管理地 | CFC |
| 海外基金/证券 | 迁入前重组，窗口后评估 FIF | FIF 计算方法 |
| 家庭信托 | 迁入前评估 settlor/beneficiary | NZ trust disclosure |

### 第7章 税务分析

新西兰税务居民通常按永久住所、居住天数和生活中心判断。符合条件的新税务居民可能享受过渡居民待遇，通常用于一定期限内豁免部分海外被动收入，但工资、服务收入、新西兰来源收入等不当然豁免。

重点：
- **过渡居民**：必须确认是否符合且是否曾经使用；不可理解为永久免税。
- **FIF**：海外基金、海外证券、部分非新西兰公司投资可能按 FIF 规则计算收入。
- **CFC**：控制海外公司可能被归属征税。
- **海外信托**：settlor 居民身份和披露义务需专项审查。

### 第8章 资金跨境合规方案

| 资金 | 动作 | 文件 |

| SMC 工资 | 新西兰雇主支付并报税 | offer、合同、payslip、税务记录 |
| AIP 投资款 | 合规来源资金进入新西兰投资账户 | 资金来源、完税证明、审计、银行流水 |

### 第14章 财税执行策划案全文

新西兰财税方案以“税务居民开始日 + 过渡居民窗口 + AIP资金来源 + FIF/CFC期后管理”为主线。

#### 14.1 税务居民开始日

记录首次长期入境、住所、家庭、工作、学校、银行、租房/购房、离境天数，由新西兰税务师确认居民开始日。该日期决定过渡居民窗口和海外资产申报起点。

#### 14.2 过渡居民窗口

若符合条件，应在窗口期内完成海外资产盘点、基金替换、公司分红策略、信托审查、未来现金流规划。窗口不是永久免税，不能作为隐瞒资产理由。

#### 14.3 FIF 管理

海外股票、基金、ETF、部分非新西兰投资实体需判断 FIF。窗口结束后选择适用计算方法并保留年度报表。迁入前应减少复杂非透明基金和多层产品。

#### 14.4 CFC 管理

如客户控制海外公司，需判断 CFC。真实经营公司应保留当地人员、办公室、合同、董事会和税务记录；被动投资公司需重点审查归属收入。

#### 14.5 AIP 资金来源

AIP 投资款必须与移民局 KYC、银行 AML 和税务申报一致。资金来源可为经营分红、股权出售、房产出售、投资收益、继承赠与等，但均需形成闭环证据。

#### 14.6 海外信托

新西兰对 settlor、trustee、beneficiary 和分配规则敏感。迁入前必须确认信托是否需要重组、是否会造成新西兰披露和纳税义务。

```text
过渡居民窗口：海外被动资产 -> 合法缓冲期 -> 重组/申报准备
窗口结束后：海外基金(FIF) + 海外公司(CFC) + 信托披露 -> 年度税务申报
AIP资金：完税来源 -> 银行KYC -> 合格投资 -> 期限维护
```

> **财税定制化执行策划案收齐上述材料后另行出具根据难易程度单独收费。**
