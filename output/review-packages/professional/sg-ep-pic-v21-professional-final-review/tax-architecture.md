# 财税架构说明

V21已重画第6章财税优化架构图，并在第14章增加新加坡EP/PIC专项财税架构说明。架构覆盖：

中国经营主体/家庭资产 → ODI/37号文/FDI/银行KYB合规闸口 → 新加坡PIC/控股/运营主体 → 利润归集 → 个人税务居民/非税务居民判断 → 薪酬/股息/投资收益 → 公司税/个人税/GST/预提税/CRS/KYB。

## 财税优化点（>=8）
1. 新加坡公司17%企业所得税承接真实经营利润。
2. 一层税制下税后股息通常不再征新加坡个人税。
3. 无资本利得税用于长期资本性质投资收益规划。
4. 无遗产税用于传承层规划。
5. 合理薪酬与税后股息拆分，兼顾EP/PR记录与个人税负。
6. GST S$1m门槛管理，避免不必要合规成本或漏登风险。
7. 股息通常无预提税；利息/特许权使用费/服务费按性质判断。
8. CRS/KYB一致性降低开户、关户和监管风险。
9. ODI/37号文/FDI闭环提升资金出境、境外持股、返程投资解释力。
