  • 个人所得税与企业所得税需按身份、居住天数、收入来源判断。

核心原则：

  • 先合规，后节税：ODI、37号文、外资返程投资、CRS/FATCA、CFC与转让定价是底线。

5.6 使用税务洼地的共同要求

1. 必须有商业目的：贸易、总部、研发、融资、投资、资产保护，而非单纯避税。

2. 必须有实质：董事、员工、办公室、合同、客户、银行流水、审计。

3. 必须能解释利润归属：利润在哪里创造，就需要匹配人员、资产、风险承担。

4. 必须考虑全球最低税/Pillar Two：大型跨国集团不能只看名义税率。

5. 必须与个人税务居民一致：公司低税不等于个人分红免税。
