第3章 合规清理详细方案

  • 资金来源：每笔投资款必须形成“收入/分红/资产出售/贷款/赠与/继承 → 纳税/完税 → 银行流水 → 出境路径 → 项目收款方”的闭环。

五、税务架构重设计全文

1. 新加坡 EP / PIC：区域总部、IP/贸易利润归集与家族资产管理节点

客户税务痛点

  • 中国公司利润高、股东分红个税高，客户希望把海外订单、采购、品牌授权、投资收益合法沉淀到低税且合规透明的节点。
  • 家庭成员计划通过 EP 常驻新加坡，担心成为新加坡税务居民后，中国/海外投资收益如何申报。
  • 中国居民个人持有境外公司涉及 37号文、资金出境、CRS 交换；企业对外投资涉及 ODI。
  • 若新加坡公司无实质，仅收取中国公司服务费/特许权使用费，容易被中国税务机关按受益所有人、转让定价、反避税否定。

架构节点

  • 中国经营主体：生产、研发、国内销售、员工及发票继续留在中国。
  • 新加坡控股/运营公司：承担东盟/全球销售、采购、品牌管理、投融资、区域管理、PIC 相关创新或生产力升级职能。
  • 香港贸易或收款公司（可选）：用于大中华区客户收款/供应链协调，但需避免与新加坡功能重复。
  • 离岸 SPV（BVI/Cayman 等，可选）：作为家族持股或未来融资层，需披露受益所有人并与 CRS 一致。
  • 家庭信托/新加坡保险：用于家庭财富传承、保单现金价值管理、债务隔离。
  • 个人税务居民：EP 持有人及家庭成员，按实际居住天数、雇佣关系、董事费/薪酬确认新加坡税务居民状态。

资金流

1. 中国公司保留中国来源利润并完税。

2. 境外客户订单由新加坡公司签约、开票、收款；新加坡公司向中国工厂或第三方采购。

3. 新加坡公司按真实职能向中国主体支付采购款、研发服务费或成本加成服务费。

4. 中国公司向新加坡公司支付特许权使用费/服务费时，需合同、受益所有人、转让定价、预提税处理。

5. 新加坡公司完税后，可分红至控股 SPV/信托/个人，或进入新加坡投资账户。

利润归集

  • 合法归集到新加坡的只能是新加坡实际承担的利润：海外销售毛利、区域管理利润、IP 管理收益、投资管理收益。
  • 中国研发、生产、销售团队创造的利润不能通过空壳服务费转走。
  • 应建立转让定价政策：成本加成、再销售价格法、交易净利润法，留存同期资料。

预提税 / DTA

  • 中国向新加坡支付股息、利息、特许权使用费、服务费，可能涉及中国预提所得税及中新 DTA 优惠。
  • 享受 DTA 需证明新加坡公司为受益所有人，具备人员、办公、董事决策、银行账户、风险承担和商业实质。
  • 新加坡通常无股息预提税；企业所得税名义税率 17%，可视具体激励/豁免/初创优惠降低实际税负。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：中国居民个人或企业控制低税境外公司且无合理商业目的，未分配利润可能被反避税关注。
  • CRS：新加坡金融账户会向中国或其他税务居民国交换；信托/公司账户需披露控制人。
  • FATCA/FBAR：如家庭成员同时为美国税务居民/绿卡/公民，美国账户披露极重，不能以新加坡账户规避。
  • 37号文：中国居民个人返程投资或控制境外 SPV，需判断是否办理外汇登记。
  • ODI：中国公司出资设立/并购新加坡主体，应履行发改、商务、外汇流程。
  • FDI：境外 SPV 回投中国时，按外商投资及外汇规则处理。
  • 当地税务居民：EP 人士在新加坡居住达到条件，薪酬/董事费在新加坡纳税；境外来源收入汇入新加坡的具体处理需个案判断。

为什么能合法降税或隔离风险

  • 通过把真实境外业务功能放在新加坡，而不是把中国利润形式上搬走，实现区域利润在新加坡纳税。
  • 低税不是来自隐匿，而是来自业务地域、合同主体、职能风险、董事决策与银行收款真实发生在新加坡。
  • 家庭信托/保险把个人经营风险、婚姻继承风险与经营公司股权分离。

不可做事项

  • 不可让新加坡空壳公司向中国公司虚开发票、虚构咨询服务费。
  • 不可用个人账户承接公司收入。
  • 不可把中国团队完成的研发/IP 强行归属新加坡而无转让、授权和估值文件。
  • 不可为规避 CRS 隐瞒实际控制人。
  • 不可未经 ODI/37号文合规直接以地下钱庄或虚假贸易出资。

2. 香港 ASMTP：大湾区经营、控股、融资与来源地税制节点

客户税务痛点

  • 客户希望利用香港地域来源税制、低企业税和金融便利承接海外业务，但担心被认定为中国实际管理机构或空壳避税。
  • 大湾区家庭常有中国公司、香港账户、海外订单混在一起，利润来源、合同履约、人员地点不清。
  • 通过 ASMTP 或相关人才/投资安排赴港后，个人工资、董事费、股息、投资收益如何区分来源地。

架构节点

  • 中国经营主体：生产、国内业务、研发、供应链执行。
  • 香港运营/控股公司：承担海外销售、融资、结算、品牌授权、股权持有、区域采购。
  • 新加坡或离岸 SPV（可选）：作为上层家族控股，不应与香港形成无实质多层空转。
  • 香港个人税务居民/非永久居民节点：确认薪俸税、利得税、个人投资收益处理。
  • 家庭信托/保险：香港或新加坡保单/信托，用于资产传承。
  • 香港证券/银行投资账户：持有美元资产、基金、债券、股票。

资金流

1. 境外客户由香港公司签约并收款。

2. 香港公司向中国公司采购或委托加工，中国公司按出口/服务收入在中国纳税。

3. 香港公司利润留存、分红至上层 SPV/家族信托，或进入香港投资账户。

4. 如香港向中国支付股息/利息/特许权使用费，按内地与香港税收安排处理预提税及受益所有人审查。

利润归集

  • 香港只归集香港/境外真实执行的贸易、融资、控股、采购管理利润。
  • 若合同谈判、定价、客户维护、履约管理都在中国完成，利润不能简单放在香港。
  • 对离岸利得申请，应证明利润产生地、关键活动地点、合同签署与履约链条。

预提税 / DTA

  • 内地向香港支付股息、利息、特许权使用费可能享受税收安排优惠，但前提是香港公司为受益所有人，且不是导管公司。
  • 香港无股息预提税、无资本利得税（但交易性利润可能为利得税）。
  • 香港利得税两级制适用时，需结合关联实体和实质经营判断。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：中国居民控制香港低税实体且无合理商业目的，未分配利润存在关注。
  • CRS：香港为 CRS 参与地，金融账户会交换至税务居民国。
  • FATCA/FBAR：美国身份客户需披露香港账户、公司权益、信托权益。
  • 37号文：中国居民通过境外 SPV 持有香港公司或返程投资，需评估登记义务。
  • ODI/FDI：中国公司设香港子公司需 ODI；香港公司回投中国需 FDI。
  • 当地税务居民：香港税制重来源地，但个人在港受雇/董事费/经营所得需按香港规则申报。

为什么能合法降税或隔离风险

  • 香港的价值在于来源地税制 + 国际金融结算 + 大湾区运营便利，不是隐藏利润。
  • 将境外订单、融资、控股和投资功能真实落在香港，可减少多国重复征税并提高资金调度效率。
  • 香港控股层能隔离中国经营主体经营风险与家庭长期资产。

不可做事项

  • 不可将中国完成的销售利润全部开到香港。
  • 不可用香港个人账户代收中国公司收入。
  • 不可为了离岸豁免伪造合同签署地点、会议记录、物流资料。
  • 不可把香港公司作为无人员、无办公、无董事决策的导管申请 DTA。

3. 土耳其基金：身份投资与境外金融资产持有节点

客户税务痛点

  • 客户希望通过合规基金投资取得身份，同时把一部分美元/欧元资产放在非中国、非美国节点。

  • 需要区分“身份投资”与“税务居民迁移”：取得土耳其身份不等于自动改变税务居民。

架构节点

  • 中国个人/家庭资产池：已完税收入或合规出境资金。
  • 离岸 SPV 或个人直接持有基金：根据基金 KYC、CRS、继承安排选择。
  • 土耳其合规投资基金账户：满足身份投资条件。
  • 家庭信托/保险（可选）：作为基金份额上层或受益安排，但需看基金是否接受。

资金流

1. 中国境内资金经合规购汇、境外收入、分红、ODI/37号文路径或已在境外资金进入基金认购账户。

2. 基金持有期内收益在基金层或投资者层处理。

3. 到期赎回回到个人/公司/信托账户，再按税务居民国申报资本利得或投资收益。

  • 基金收益属于投资收益，不应包装成经营利润。
  • 若通过 SPV 持有，SPV 的未分配收益需考虑中国 CFC 或未来税务居民国的反递延规则。
  • 家族办公室可记录资金来源、认购、估值、赎回、纳税凭证。

预提税 / DTA

  • 土耳其基金底层资产可能有土耳其本地预提税或基金层税负，取决于基金类型和投资标的。
  • 中国/土耳其 DTA 或投资者税务居民国与土耳其 DTA 可能影响股息、利息、资本利得税收待遇。
  • 若通过香港/新加坡 SPV 持有，需证明 SPV 有实质，否则 DTA 优惠可能被否。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：中国居民控制境外 SPV 持有基金，低税积累收益需关注。
  • CRS：土耳其金融机构一般会收集税务居民信息并交换。
  • FATCA/FBAR：美国税务居民持有海外基金可能触发 PFIC、FBAR、Form 8938 等复杂申报。
  • 37号文：个人以境外 SPV 进行投资需判断是否涉返程或特殊目的公司登记。
  • ODI：中国公司投资土耳其基金需履行 ODI；个人直接投资受外汇规则限制。
  • 当地税务居民：仅持有土耳其身份/基金通常不等于土耳其税务居民；长期居住或中心利益转移才需评估。

为什么能合法降税或隔离风险

  • 合法价值主要是身份备份 + 投资资产地域分散 + 与经营风险隔离。
  • 税负优化来自选择合规持有主体、避免未来迁入高税国后才处理资产、减少遗产/继承/债务混同。
  • 不能把土耳其基金作为逃避中国或美国申报的工具。

不可做事项

  • 不可用未申报资金、地下钱庄或虚假贸易认购基金。
  • 不可向基金或银行隐瞒中国/美国税务居民身份。
  • 不可承诺基金身份投资本身实现免税。
  • 美国税务居民不可忽视 PFIC/FBAR/FATCA 申报。

4. 土耳其房产：不动产持有、租金/资本利得与继承隔离节点

客户税务痛点

  • 客户通过土耳其房产取得身份或资产配置，但忽视租金税、资本利得税、房产税、继承及跨国申报。
  • 中国居民直接持有海外房产，资金来源、租金回流、未来出售收益申报不清。
  • 高税国家迁居前若已持有房产，未来全球征税、遗产税、资产披露风险上升。

架构节点

  • 个人直接持有：简单，但继承、债务隔离、税务居民国申报压力较高。
  • 土耳其本地公司持有（谨慎）：适合多套出租/经营性物业，但会增加公司税、会计和实质成本。
  • 离岸 SPV 持有（需审慎）：有隔离价值，但可能增加土耳其/居民国反避税审查。
  • 家庭信托/保险（上层）：用于传承安排，不一定适合作为直接产权登记主体。
  • 租金收款账户：土耳其本地银行账户或境外账户，需保留租赁合同和纳税凭证。

资金流

1. 已完税资金经合规路径进入土耳其购房律师监管账户/银行账户。

2. 支付购房款、税费、中介费、维护费。

3. 租金进入土耳其账户并在当地申报。

4. 出售后资金回到个人/SPV账户，再按税务居民国申报。

利润归集

  • 租金利润归属于房产所在地土耳其，先在土耳其按本地规则纳税。
  • 出售增值按土耳其规则和居民国规则综合处理，可能有抵免。
  • 如果通过公司持有，租金和增值先归集到公司，再通过分红/清算给股东。

预提税 / DTA

  • 不动产所得通常由不动产所在地优先征税。
  • 土耳其与中国及其他居民国的 DTA 通常允许来源地对不动产收入征税，居民国可能给予税收抵免。
  • 公司分红可能涉及土耳其预提税及 DTA 优惠。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：若用低税 SPV 持有并积累租金/资本收益，中国居民或高税居民国可能适用 CFC。
  • CRS：房产本身不是金融账户，但银行账户、SPV、信托会报告。
  • FATCA/FBAR：美国税务居民需披露海外账户；海外公司/信托另有表格。
  • 37号文/ODI：境内公司或个人通过境外结构投资房产需评估外汇及 ODI 合规。
  • 当地税务居民：购房/身份不必然成为土耳其税务居民；长期居住和生活中心需判断。

为什么能合法降税或隔离风险

  • 土耳其房产主要提供不动产所在地税负确定性、资产地域分散、身份备份。
  • 通过租金合同、当地申报和居民国抵免，减少双重征税。
  • 通过公司或信托上层安排可隔离债务/继承风险，但必须权衡税务和维护成本。

不可做事项

  • 不可低报房价规避契税/税费。
  • 不可用现金或不透明第三方付款破坏资金来源链。
  • 不可漏报租金或出售收益。
  • 不可把土耳其房产身份包装为全球免税身份。

5. 澳大利亚 482：高税工作居留下的迁入前资产隔离与全球征税控制

客户税务痛点

  • 482 签证客户赴澳工作后，可能成为澳大利亚税务居民，被澳洲按全球收入征税。
  • 中国公司股权、境外公司、房产、证券账户、信托在成为澳税务居民后暴露，需要 CGT、CFC、转让定价、资本利得成本基础规划。
  • 澳大利亚与美国/中国账户披露、CRS 数据交叉风险高。

架构节点

  • 中国经营主体：迁居前完成股权、分红、债务、关联交易梳理。
  • 香港/新加坡控股公司：如有真实海外业务，可作为区域控股/利润归集节点。
  • 家庭信托/保险：迁入澳洲前完成，不宜迁入后临时设立规避税。
  • 澳洲个人税务居民节点：482 持有人及家庭成员；确认居住、住所、183天、永久居所等测试。
  • 澳洲雇主薪酬账户：工资 PAYG 纳税。
  • 境外投资账户：进入澳税务居民后按澳洲全球所得申报。

资金流

1. 赴澳前：完成中国公司可分配利润分红、境外资产估值、信托/保险设立、贷款文件整理。

2. 赴澳后：澳洲工资在澳洲纳税；境外股息、利息、租金、资本利得向澳洲申报。

3. 中国公司向澳洲个人分红，需中国预提/个税与澳洲税收抵免匹配。

4. 境外公司分红至澳洲税务居民个人或信托，按澳洲规则申报。

利润归集

  • 482 本身不创造低税利润节点；利润归集仍应在中国/香港/新加坡等真实经营实体。
  • 澳洲个人不应直接控制并指挥境外公司关键决策，否则可能引发公司澳洲税务居民或 PE 风险。
  • 若澳洲个人为境外公司董事，需董事会地点、授权、实质管理文件隔离。

预提税 / DTA

  • 中国向澳洲居民支付股息/利息/特许权使用费，适用中澳 DTA 及中国国内预提税规则。
  • 澳洲对全球收入征税，通常允许外国税收抵免，但抵免不等于免税。
  • 澳洲向非居民支付股息、利息、特许权使用费也有预提税规则。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：澳洲 CFC 规则较强，澳税务居民控制低税境外公司可能被归属征税。
  • CRS：澳洲接收境外金融账户信息，境外账户隐匿风险极高。
  • FATCA/FBAR：若同时有美国身份，澳洲/境外账户仍需向美国披露。
  • 37号文/ODI/FDI：中国居民迁澳前持有境外 SPV，仍需补齐中国外汇和投资合规。
  • 当地税务居民：一旦成为澳洲税务居民，全球所得、资本利得、海外信托分配、境外公司权益均需审查。

为什么能合法降税或隔离风险

  • 合法优化不是把收入藏起来，而是迁入前完成资产定性、成本基础确认、股权层级与信托/保险安排。
  • 避免个人迁澳后直接管理境外公司，使中国/新加坡/香港业务利润不被错误归属澳洲。
  • 利用 DTA 和外国税收抵免减少双重征税。

不可做事项

  • 不可成为澳税务居民后继续不申报中国公司分红、海外账户、海外租金。
  • 不可用亲属代持逃避澳洲 CFC/信托规则。
  • 不可在澳洲远程决策境外公司全部经营却声称公司非澳税务居民。
  • 不可迁入后临时低价转让资产给离岸公司规避 CGT。

6. 日本 HSP / 经营管理：日本居住下的海外资产申报与经营实体隔离

客户税务痛点

  • 日本 HSP/经营管理客户长期居住后可能成为日本税务居民，全球所得、海外资产、国外送金都可能纳税/申报。
  • 日本对高净值人士海外资产申报、国外财产调书、出境税/继承赠与税风险较敏感。
  • 中国公司利润、日本公司经营利润、个人生活资金常混同。

架构节点

  • 中国经营主体：保留中国业务及完税记录。
  • 日本经营管理公司：承担日本本地业务、雇佣、办公室、许可、客户合同。
  • 香港/新加坡控股公司（可选）：用于亚洲区域业务，不应被日本个人实际管理。
  • 家庭信托/保险：日本对信托和保险税务处理复杂，迁日前需专项审查。
  • 日本个人税务居民节点：区分非永久居民、永久居民纳税义务范围。
  • 日本/境外投资账户：海外资产申报重点。

资金流

1. 日本公司获得日本本地收入并在日本缴纳法人税、消费税等。

2. 中国公司与日本公司之间的服务、采购、授权需合同及转让定价。

3. 个人生活费从已申报薪酬/分红/贷款路径进入日本账户。

4. 境外投资收益汇入日本或由日本税务居民取得时，按日本规则申报。

利润归集

  • 日本公司只归集日本本地经营利润。
  • 中国业务利润保留中国；亚洲业务可由香港/新加坡承接，但董事决策和实际管理不能在日本个人居所完成。
  • IP 授权费、管理费需有实质服务和定价依据。

预提税 / DTA

  • 中国向日本居民或日本公司支付股息、利息、特许权使用费适用中日 DTA。
  • 日本向境外股东分红可能有日本预提税及 DTA 优惠。
  • 日本公司向香港/新加坡支付服务费/特许权使用费需判断来源、预提税和转让定价。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：日本有 CFC 规则，长期日本税务居民控制低税境外公司，可能被归属征税。
  • CRS：日本接收海外账户信息；海外资产调书/财产债务调书可能适用。
  • FATCA/FBAR：美国身份叠加时需美国申报。
  • 37号文/ODI/FDI：中国居民出境前境外控股及返程投资需合规。
  • 当地税务居民：日本非永久居民期间与永久居民期间纳税范围不同，居住年限变化是关键。

为什么能合法降税或隔离风险

  • 通过日本本地利润日本纳税、非日本业务不由日本实际管理，避免全球集团利润被日本重构。
  • 迁日前梳理海外资产、受益人、送金路径，降低海外资产申报和继承赠与税冲击。
  • DTA 抵免减少同一收入中日双重征税。

不可做事项

  • 不可用日本经营管理公司空转，只为签证无真实经营。
  • 不可长期在日本遥控香港/新加坡公司却声称管理地在境外。
  • 不可未申报海外账户、海外股息、海外租金。

  • 海外经营利润应留在中国/香港/新加坡等真实经营地。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC/FIF：新西兰有 CFC 和 FIF 规则，海外公司/基金投资可能被按规则归属或计算收入。
  • CRS：新西兰会接收/交换金融账户信息。
  • FATCA/FBAR：美国身份叠加时需美国披露。
  • 37号文/ODI/FDI：中国居民海外控股和投资路径需补齐中国合规。
  • 当地税务居民：居住、永久居所、家庭中心决定税务居民；税务居民开始日影响过渡居民窗口。

为什么能合法降税或隔离风险

  • 利用新西兰过渡居民规则，在合法期间内降低部分海外收入税负并完成资产重组。
  • AIP 投资与个人/家庭资产、经营公司风险隔离，资金来源可审计。
  • 海外经营利润保持在真实经营地，避免新西兰 CFC/FIF 不必要归属。

不可做事项

  • 不可把过渡居民优惠理解为永久免税。
  • 不可迁入后继续隐瞒中国公司、香港账户、海外基金。
  • 不可用 AIP 投资账户承接未解释来源资金。
  • 不可通过空壳海外公司规避 FIF/CFC。

8. 美国 EB-1A：全球征税、遗产税、CFC/PFIC/FBAR 的最高风险节点

客户税务痛点

  • EB-1A 获批及登陆后，如成为美国税务居民/绿卡持有人，美国对全球所得征税，申报强度全球最高。
  • 中国公司股权、香港/新加坡公司、土耳其基金、海外房产、信托、保险、证券账户都可能触发美国复杂申报。
  • 美国遗产税、赠与税、退出税、CFC、PFIC、GILTI、Subpart F、FBAR/FATCA 风险极高。

架构节点

  • 中国经营主体：赴美前处理利润分配、股权架构、估值、董事管理权。
  • 香港/新加坡控股公司：若继续持有，需美国 CFC/GILTI/PFIC 评估。
  • 非美国信托/美国前信托规划：必须在成为美国税务居民前由美国律师设计。
  • 人寿保险/私人配售保险（如适用）：需符合美国税法，不可用非合规保单隐藏投资。
  • 美国个人税务居民节点：绿卡、实质居留测试、公民/居民身份。
  • 全球金融账户：FBAR、Form 8938、5471、8621、3520/3520-A 等可能适用。

资金流

1. 赴美前：完成中国公司分红、股权赠与/出售/信托、成本基础、资产估值和税务意见。

2. 登陆后：全球工资、经营利润分配、股息、利息、租金、资本利得向美国申报。

3. 中国/香港/新加坡公司向美国居民分红，按来源国预提税 + 美国税收抵免处理。

4. 海外基金/保险/信托分配需按美国规则逐项申报。

利润归集

  • EB-1A 个人不能成为境外公司事实管理中心，否则境外公司和利润可能被美国规则穿透/归属。
  • 美国税务居民控制境外公司，CFC/GILTI/Subpart F 可能使未分配利润提前纳税。
  • PFIC 基金收益可能被惩罚性税制处理，因此赴美前应清理非美国基金。

预提税 / DTA

  • 中国向美国居民支付股息、利息、特许权使用费适用中美 DTA 与中国预提税。
  • 美国对税务居民全球征税，外国税收抵免可减少但不能消灭美国税。
  • 美国向非居民支付美国来源 FDAP 收入通常有 30% 预提税，可由 DTA 降低。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC/GILTI/Subpart F：美国居民持有/控制境外公司需重点处理。
  • PFIC：非美国基金、部分保险/投资产品可能触发 Form 8621 和高税负。
  • CRS：美国不是 CRS 参与国，但通过 FATCA 获得全球账户信息。
  • FATCA/FBAR：海外金融账户总额超过门槛需 FBAR；Form 8938 另行适用。
  • 37号文/ODI/FDI：中国侧历史合规仍需处理，尤其未来出售/分红/返程投资。
  • 当地税务居民：美国绿卡/实质居留测试一旦触发，全球税务申报立即升级。

为什么能合法降税或隔离风险

  • 合法优化的关键是成为美国税务居民前完成资产重组：清理 PFIC、设立合规信托、调整公司控制权、实现必要分红或增值。
  • 通过外国税收抵免和 DTA 减少双重征税。
  • 通过非美国资产与美国遗产税规划降低传承税务冲击。

不可做事项

  • 不可登陆后才把资产转入离岸信托以逃避美国税。
  • 不可不申报海外公司、账户、基金、信托。
  • 不可持有大量 PFIC 后忽视 Form 8621。
  • 不可用非美国保险包装投资而不做美国税务测试。
  • 不可让美国个人继续实际控制香港/新加坡公司却不申报 CFC。

9. 多米尼克 CBI：护照备份与税务居民分离节点

客户税务痛点

  • 客户误以为取得多米尼克护照即可成为低税居民或免除中国/美国/澳洲税。
  • CBI 护照适合通行和身份备份，但若无实际税务居民迁移，原税务居民国仍对其征税。
  • 金融机构 KYC 会询问出生地、税务居民、资金来源，不是看护照决定税务归属。

架构节点

  • 原税务居民国个人节点：中国/美国/澳洲/日本等仍是核心纳税地。
  • 多米尼克公民身份节点：通行、备份、部分开户便利。
  • 离岸 SPV/信托/保险：只在有传承或投资隔离目的时使用。
  • 香港/新加坡投资账户：以真实税务居民身份开户。
  • 家庭资产清单：确保护照身份与 CRS 自证一致。

资金流

1. 已完税资金支付 CBI 捐款/投资及费用。

2. 境外投资账户仍按实际税务居民申报 CRS/FATCA。

3. 如未来真正迁移税务居民，应另行满足居住天数、永久住所、生活中心、离境注销或居民身份切换条件。

利润归集

  • 多米尼克 CBI 本身不应作为利润归集中心。
  • 如设多米尼克公司或账户，需有商业目的，否则只是空壳。
  • 投资收益仍归属于个人或持有实体，并在税务居民国申报。

预提税 / DTA

  • 多米尼克与主要国家 DTA 网络有限，不能依赖其降低中国/美国/澳洲来源预提税。
  • 投资收益来源国可能征预提税，居民国再征税或抵免。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：中国/美国/澳洲等税务居民控制多米尼克或其他低税公司，可能被归属征税。
  • CRS：开户时必须填写真实税务居民，不能只填多米尼克护照。
  • FATCA/FBAR：美国人无论持何护照仍需全球申报。
  • 37号文/ODI/FDI：用境外 SPV 持股或返程投资仍看中国外汇规则。
  • 当地税务居民：CBI 公民身份通常不等于多米尼克税务居民。

  • 其价值是身份与旅行风险隔离，不是直接降税。

10. 格鲁吉亚投资居留：低税经营/个人税务居民备选与欧亚业务节点

客户税务痛点

  • 客户希望利用格鲁吉亚相对友好的税制和投资居留作为欧亚业务、轻资产经营或个人税务居民备选。
  • 需要判断是否真的能改变中国/高税国家税务居民，而不是只取得居留。
  • 跨境电商、咨询、IT、贸易收入如果由格鲁吉亚公司承接，需证明实质和客户来源。

架构节点

  • 格鲁吉亚公司/个体经营节点：承接欧亚/远程服务、贸易、IT、咨询等真实业务。
  • 中国经营主体：保留中国团队和中国来源利润。
  • 香港/新加坡控股或收款节点（可选）：如客户/银行需求更适合，可与格鲁吉亚分工。
  • 个人税务居民节点：是否满足格鲁吉亚税务居民条件及是否脱离原居民国。
  • 家庭信托/保险/投资账户：上层传承与资产配置。

资金流

1. 境外客户与格鲁吉亚公司签约、付款。

2. 格鲁吉亚公司支付本地工资、办公、外包和税费。

3. 中国公司如提供服务，应按合同收取服务费并在中国纳税。

4. 格鲁吉亚公司完税后分红至个人/控股公司/信托。

利润归集

  • 可归集格鲁吉亚公司实际承担的销售、交付、技术、管理利润。
  • 如核心团队仍在中国，仅由格鲁吉亚公司开票，会被视为利润错配。
  • 可设计为：格鲁吉亚承担客户签约与交付，中国公司承担外包研发/支持并收成本加成。

预提税 / DTA

  • 格鲁吉亚与部分国家有 DTA，但网络和适用效果需逐项核查。
  • 向中国支付服务费、股息、利息、特许权使用费需看来源地和 DTA。
  • 分红、利息、特许权使用费可能有格鲁吉亚本地预提税。

CFC / CRS / FATCA / FBAR / 37号文 / ODI / FDI / 当地税务居民影响

  • CFC：中国或高税国居民控制格鲁吉亚低税公司，需合理商业目的和实质。
  • CRS：格鲁吉亚金融账户合规程度需按银行要求填写税务居民信息；不能隐瞒。
  • FATCA/FBAR：美国身份客户仍需披露。
  • 37号文/ODI/FDI：中国个人/公司出资格鲁吉亚公司需评估外汇及 ODI。
  • 当地税务居民：成为格鲁吉亚税务居民需满足居住或高净值等条件，同时还需处理原居民国退出/双重居民冲突。

为什么能合法降税或隔离风险

  • 若客户确有欧亚、远程服务、IT、贸易业务，把职能、合同、银行、雇员、管理放在格鲁吉亚，可以形成低成本经营节点。
  • 与中国主体通过转让定价分工，减少中国利润被无依据外移的风险。
  • 格鲁吉亚居留作为备选迁移地，有助于高税国家迁入前的税务居民路径设计。

不可做事项

  • 不可只注册格鲁吉亚公司而无办公、人员、客户管理即截留利润。
  • 不可用格鲁吉亚账户收中国境内业务收入。
  • 不可宣称取得投资居留就自动免中国税。
  • 不可忽略原税务居民国 CFC 和 CRS。

11. 马耳他 MPRP（Malta Permanent Residence Programme）

官方/政府费

  • 依据 MPRP Regulations S.L. 217.26（2025 updated PDF）与 Residency Malta Agency。
  • 常见核心费用结构：
  • 行政/申请费：主申请及家庭按法规缴付，2025 更新后市场/法规摘要常见 €50,000 起，家属另有加收；
  • 政府贡献：常见 €37,000；
  • 慈善捐款：€2,000；
  • 居留卡/生物识别/代理递交等按官方表格。
  • 房产：购买合格住宅通常 €375,000 起；租赁通常 €14,000/年 起（需按法规区域/更新版本核验）。
  • 资产要求：搜索到官方 FAQ 摘要显示申请人需有 €500,000 capital，其中至少 €150,000 为 financial assets。

律师/顾问费/服务费

  • MPRP 必须通过 licensed agent / accredited agent 递交。专业费约 €20,000–60,000+；复杂家族、税务、房产、KYC 可更高。

投资/资金门槛

  • 不是纯投资入籍，而是永久居留。需同时满足：合格房产（买或租）、政府贡献、慈善捐款、资产要求、健康保险、尽调与合规。

生活/安顿费

  • 马耳他单人 6 个月：约 €12,000–30,000。
  • 三口之家 6 个月：约 €30,000–72,000；国际学校/保险另计。

材料清单

  • MPRP 申请表、护照、出生/婚姻/离婚证明、无犯罪、资金来源、资产证明、银行流水、税务证明、健康保险、租赁/购房合同、宣誓声明、家属依附证明、KYC 表格。

第三方材料清单

  • Licensed agent KYC、Veriff/身份验证、无犯罪证明、翻译认证、银行推荐/资信、资产评估、税务居民证明、保险证明、房产合同/租赁注册。

12. 11国组合下的推荐分层架构：按税务功能而非护照数量搭建

A. 低税经营与利润归集层

  • 首选：新加坡 EP/PIC 用作东盟/全球区域总部、IP/贸易/投融资节点。
  • 辅助：香港 ASMTP 用作大湾区贸易、融资、控股、收款节点。
  • 机会型：格鲁吉亚投资居留 用作欧亚远程服务、IT、轻资产经营节点。

关键规则：三者不能重复承担同一利润功能。每个节点必须有人员、合同、银行、董事决策、客户、成本和风险。

B. 高税迁入保护层

  • 美国 EB-1A：最高优先级，登陆前必须完成 PFIC/CFC/信托/遗产税规划。
  • 澳大利亚 482：迁入前做资产估值、信托/保险、境外公司管理地隔离。
  • 日本 HSP/经营管理：关注非永久居民窗口、海外资产申报、继承赠与税。
  • 新西兰 SMC/AIP：利用过渡居民窗口和 FIF/CFC 规则进行投资重组。

关键规则：高税国家身份不是税务优化工具，而是风险触发器。必须先规划，再登陆/长期居住。

C. 投资与身份备份层

  • 土耳其基金/房产：身份投资、不动产/基金资产分散，但不改变税务居民。
  • 多米尼克 CBI：护照与通行备份，不是税务居民切换。
  • 马耳他 MPRP：欧盟居留备份，可规划税务居民/非 domicile 路径，但需实居和汇入规则判断。

关键规则：身份文件、居留卡、护照不是纳税地。纳税地由居住、住所、生活中心、国籍/绿卡、实际管理地和 DTA 决定。

第15章：最终落地检查清单

14. 最终落地检查清单

1. 先判税务居民：每位家庭成员当前及未来 3 年税务居民国、居住天数、绿卡/国籍、永久住所。

2. 再判利润来源：中国、香港、新加坡、格鲁吉亚、日本/澳洲/新西兰/美国各自承担哪些职能风险。

3. 资金来源闭环：工资、分红、股权转让、企业利润、贷款、赠与、继承均需凭证。

4. 出境路径合规：个人购汇、境外存量资金、ODI、37号文、FDI 不混用。

5. 转让定价文档：关联交易合同、定价政策、同期资料、董事会记录、交付证据。

6. DTA 证据包：税务居民证明、受益所有人说明、办公人员、董事会、银行流水、合同发票。

7. CRS/FATCA 一致性：所有银行、券商、基金、保险、信托自证与真实税务居民一致。

8. 高税国登陆前规划：美国、澳洲、日本、新西兰必须在迁入前完成资产重组和申报策略。

9. 信托/保险专项审查：不要用通用离岸信托模板套美国/澳洲/日本/新西兰家庭。

10. 不可做清单入合同：禁止虚假服务费、代持隐匿、地下钱庄、空壳 DTA、护照冒充税务居民。
