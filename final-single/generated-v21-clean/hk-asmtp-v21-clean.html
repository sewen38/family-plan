二、香港 ASMTP｜输入内地人才计划

第1章 客户家庭基本信息

香港 ASMTP 适用于中国内地居民已获香港真实雇主聘用，且具备香港所需而本地不易获得的专业知识、技能或经验的申请人。该路径不是投资移民，不要求投资额；成败核心在雇主真实性、职位必要性、薪酬市场性、申请人不可替代性及家庭受养安排。

信息项执行填写口径文件
主申请人内地居民、专业人才、管理人员、技术人员ID 990A、护照/旅行证件、身份证、照片
香港雇主香港真实运营公司ID 990B、商业登记证、财务/业务证明
职位香港本地难以即时提供的人才岗位职位说明、招聘证明、薪酬福利说明
家庭成员配偶及受养子女关系证明、受养人表格、经济能力证明
目标工作居留、子女教育、香港税务与资产配置预算与时间确认

第2章 核心策略

核心策略是“雇主合规 + 职位合规 + 人才匹配”。申请材料必须证明香港公司有业务需要、岗位职责清晰、薪酬福利与本地市场相若，且申请人拥有相关学历、资格或经验。若客户同时有香港公司或跨境业务，应避免把 ASMTP 做成无真实雇佣的身份包装。

策略层执行动作成功标准
雇主策略完整解释公司业务、财务、招聘与岗位必要性入境处认可岗位真实
人才策略学历、资格、经验与岗位一一对应证明本地不易获得
家庭策略同步准备受养人材料家属随行居留
财税策略利用香港来源地税制与金融账户，但保持实质不做空壳导管
长期策略续签、通常居住、7年永居评估保持雇佣与居住记录

第3章 合规清理详细方案

ASMTP 的材料不能只提交雇佣合同，应同时提交雇主为什么需要该申请人、为什么香港本地市场难以替代、薪酬为什么合理、申请人为什么能胜任。

清理事项具体要求输出
雇主商业实质商业登记、财务报表、客户合同、办公室、员工雇主合规包
职位真实性职责、汇报线、招聘记录、市场薪酬职位说明与招聘说明
申请人匹配学历、工作证明、专业成果人才匹配说明
家属依附婚姻/出生、公证翻译、经济能力受养人包
税务与薪俸薪酬支付、MPF/税务安排薪俸税规划

第4章 境外资金归集与投资架构

香港 ASMTP 本身不要求投资，但香港可作为大湾区经营、控股、融资、收款与资产配置节点。客户应区分个人工资、公司利润、投资收益和中国业务收入，避免香港个人账户代收中国公司款项。

境外客户 -> 香港运营公司[合同/收款/融资/贸易]
香港公司 -> 采购/委托加工 -> 中国经营主体[出口/生产/中国完税]
香港公司 -> 分红/投资 -> 香港投资账户/家族信托/保险
个人 -> ASMTP雇佣薪酬 -> 香港薪俸税申报
资金类型合规处理风险提示
香港工资由雇主按合同发放并申报薪俸税不得名义发薪、实际返还
香港公司利润按来源地判断利得税离岸利得需证据支持
中国公司利润留在中国纳税或合规分红/ODI不得通过香港个人账户转移
投资收益以真实税务居民身份开户和申报CRS/FATCA 需一致

第5章 投资使用建议

ASMTP 无投资门槛。预算重点应放在香港生活、租房、子女教育、雇主合规材料和专业服务上。

用途建议预算说明
签证签发费HK$230/人以入境处收费表核验
顾问/律师单案HK$8,000–30,000标准 ASMTP
含雇主解释与家属HK$20,000–60,000复杂雇主、补件、受养人
单人6个月生活费HK$90,000–180,000租房、押金、交通、保险
三口6个月生活费HK$220,000–500,000私校/国际学校另计

第6章 财富架构搭建方案

香港的价值在于地域来源税制、国际金融结算与大湾区便利。香港公司可承接真实海外销售、融资、控股、采购管理利润；若合同谈判、定价、客户维护、履约管理都在中国完成，利润不能简单放在香港。

6.1 单项目身份架构图

香港雇主真实岗位 -> 内地人才获聘 -> ASMTP递交
ASMTP获批 -> 主申请人赴港工作/纳税
配偶子女 -> 受养人签证
持续雇佣/居住 -> 续签 -> 7年通常居住后评估永久居民

6.2 香港财富管理结构

节点功能风险控制
中国经营主体生产、研发、国内业务中国完税、出口合规
香港运营/控股公司海外订单、融资、结算、控股来源地证据、董事决策
香港个人账户薪资、生活、投资不代收公司收入
香港/新加坡信托保险传承与隔离CRS、受益人披露

第7章 税务分析

香港利得税采用来源地原则，非香港来源利润是否征税需个案判断并保留证据。薪俸税则与在港受雇及工作安排相关。香港无一般股息预提税、无资本利得税，但频繁交易性利润可能被视为应税经营利润。内地向香港支付股息、利息、特许权使用费时，应按内地与香港税收安排及受益所有人规则审查。

税务主题结论执行控制
薪俸税ASMTP 工资通常需在香港申报雇佣合同、发薪、报税
利得税香港公司利润按来源地及经营事实判断合同、人员、谈判、履约证据
离岸利得可申请但不可伪造事实保留合同、物流、会议、邮件
DTA/税收安排内地向香港支付款项需受益所有人香港公司实质、非导管
CRS/FATCA香港账户信息交换税务居民声明一致

第8章 资金跨境合规方案

香港资金安排必须将个人、公司、家庭资产分账管理。中国公司设香港子公司需评估 ODI；香港公司回投中国需 FDI；中国居民通过境外 SPV 持股需评估 37号文。ASMTP 个人赴港工作不自动解决中国资产跨境问题。

场景合规路径文件
香港雇主发薪香港公司发薪至个人账户雇佣合同、工资单、税表
中国公司设港司ODI/商务/发改/外汇境外投资备案/登记
港司回投中国FDI、外资企业登记投资协议、外汇登记
个人投资账户以真实税务居民身份开户KYC、CRS、自证表

第9章 身份路径规划

ASMTP 申请通常以入境处 ID 990A（申请人）及 ID 990B（雇主）为核心表格，政府签发费通常 HK$230。获批后申请人按许可来港工作，配偶及未成年子女可申请受养人。后续续签通常围绕雇佣持续性、薪资、在港工作和居住情况；通常居住满 7 年后可评估香港永久居民身份。

阶段周期动作成果
预评估1–2周雇主、职位、人才匹配是否可递交
材料准备2–4周ID 990A/990B、雇佣与证明递交包
审理4–12周+入境处审理、补件获批/补件/拒签
入境工作获批后赴港、发薪、租房身份生效
续签/永居持续稳定雇佣、通常居住续签及7年后评估

第10章 教育规划

香港教育路径包括国际学校、私立学校、直资学校和本地学校。受养人子女可在香港就读，但学校录取、学位、语言和学费应独立规划。国际学校通常需提前申请，并准备面试、英文能力证明和押金。

学校类型适合家庭预算/执行
国际学校希望衔接 IB/AP/A-Level学费、债券、建校费差异大
私立/直资希望兼顾本地与国际需面试及名额
官津学校长期融入本地学位和校网需规划
幼儿园幼龄儿童提前申请、区域选择重要

第11章 福利居住国规划

ASMTP 是工作居留安排，受养人身份依附主申请人。香港医疗、住房、教育和长期公共福利与居民身份、通常居住年期及具体资格有关。客户应以商业保险、私宅租赁、学校申请和现金流准备作为初期保障。

模块初期安排长期规划
医疗商业保险 + 私营/公营服务了解长居后评估公共服务资格
住房租赁为主，预算压力高根据工作地点和学校选区
家属受养人签证主申请人身份变动会影响家属
永居不保证，需通常居住7年后依法评估

第14章 财税执行策划案全文

14.1 财税目标

香港 ASMTP 财税方案目标，是让客户在香港真实就业、真实发薪、真实申报的基础上，利用香港金融与来源地税制进行跨境经营和资产配置。方案不得以香港个人账户代收中国收入，不得将中国完成的利润虚假转入香港公司，也不得以无实质香港导管申请税收安排优惠。

14.2 架构图

中国经营主体[生产/研发/中国完税]
  -> 采购/委托加工 -> 香港运营公司[海外订单/融资/收款]
境外客户 -> 合同/付款 -> 香港公司
香港公司 -> 薪资 -> ASMTP申请人[薪俸税]
香港公司 -> 分红/投资 -> 香港账户/信托/保险
控制点：来源地利得税、内地香港税收安排、CRS、37号文/ODI/FDI

14.3 资金双循环架构

经营循环：境外收入 -> 香港公司 -> 中国采购/外包 -> 香港利润判断 -> 留存/分红
家庭循环：香港薪资 -> 个人账户 -> 租房教育保险 -> 投资账户 -> 税务居民申报

14.4 执行包 A：薪俸税与身份包

表面问题：客户想办理香港专才身份。

阶段教育体系身份影响预算提醒

10.2 公立 / 私立 / 国际学校选择

学校类型优点缺点适合家庭
公立学校本地融入强、费用相对低竞争、语言与学位限制计划长期在港居住家庭。

PR/公民子女教育权益与持有临时签证阶段不同。香港永居可提升长期稳定性，但教育结果仍取决于学校、语言、成绩与家庭居住安排。身份阶段、租房区域、入学季和续签节奏需要一起倒排。

项目预算口径风险

十一、福利居住国规划

香港福利居住国规划属于国家层内容，永久保留。此处同时保留医疗、养老、居住成本、居住权稳定性、税务与公共福利、长期身份维护，并以 Medicare、Super 作为跨国福利制度比较词，说明香港并非澳洲式 Medicare/Super 福利结构。

维度香港专才阶段永居阶段执行提醒
医疗可使用公私营医疗资源，具体费用和资格以香港规则为准权益稳定性更强建议配置商业医疗保险。
养老/MPF受雇通常涉及MPF供款持续在港工作形成退休金记录雇主/雇员供款要合规。
居住成本租金高，港岛/九龙核心区压力大购房与租房均需预算首年预留押金、中介费、家具和搬迁费。
居住权稳定性依赖签证与续签永居后稳定性显著提高7年通常居住记录从第一年开始。
税务与公共福利薪俸税/利得税与公共服务联动税务与福利稳定性提升不要把低税等同于无申报。
长期身份维护签证、工作、公司、居住四线维护永居和特区护照需另行判断国籍法边界必须说明。
